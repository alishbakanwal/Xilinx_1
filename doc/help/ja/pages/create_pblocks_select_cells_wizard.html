[Select Cells] ページ

Create Pblocks ウィザードのこのページでは、現在のプロジェクトの階層セルを指定して、Pblock に関連付けます。

  • [Add] () : [Select Cell] ダイアログ ボックスが開き、任意のセルを選択できます。別の Pblock が選択した各セルごとに作成されます。Pblock が作成されたら、追加のセルを Pblock に割り当てることができます。
    ヒント: [Select Cell] ダイアログ ボックスには、現在のデザインの [Netlist] ウィンドウからのセルのみがリストされます。セルは [Netlist] ウィンドウで直接選択することもできます。 [Floorplanning] → [New Pblock] をクリックして新しい Pblock を作成するか、[Assign to Pblock] をクリックして既存 Pblock に追加します。
  • [Remove] () : Pblock に関連付けられたセルのリストから選択したセルが削除されます。

[Next] をクリックして Create Pblocks ウィザードの [Format Names] ページに進みます。
