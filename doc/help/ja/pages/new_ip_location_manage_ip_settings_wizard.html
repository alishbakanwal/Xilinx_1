[Manage IP Settings] ページ

  • [Part] : Manage IP プロジェクトのターゲット パーツまたはボードを選択します。パーツ名またはボード名を直接入力するか、または参照ボタンをクリックして [Select Device] ダイアログ ボックスから選択できます。
  • [Target Language] : RTL プロジェクトで使用されるターゲット言語を選択します。デザインに追加された IP はターゲット言語でサポートされる場合に追加されます。
  • [Target Simulator] : Vivado® シミュレータおよびサードパーティ シミュレータを含むリストからシミュレータを選択します。
  • [Simulator Language] : デザインの検証に使用するシミュレータでサポートされる HDL 言語を指定します。このオプションでは、シミュレーション用に IP 出力ファイルを生成するために使用する言語を指定します。
  • [IP Location] : Manage IP プロジェクトを作成するディレクトリを指定します。Manage IP プロジェクトおよび管理される IP カスタマイズの両方がここで指定されたディレクトリに書き出されます。
    ヒント: [IP Location] で指定したディレクトリが見つからない場合は、Vivado Design Suite で作成されます。

[Finish] をクリックして Manage IP プロジェクトを作成します。
