[Primary Clock Constraints] ダイアログ ボックス

このダイアログ ボックスでは、現在選択しているプライマリ クロックまたはクロックの属性を定義できます。クロックの属性は、create_clock コマンドのオプションと関連しています。

  • [Frequency] : クロックの演算周波数を MHz で指定します。
  • [Period] : クロック周期をナノ秒 (ns) で指定します。
    ヒント: [Frequency] または [Period] のいずれかが必ず必要です。指定のない属性は、指定した属性から自動的に派生されます。
  • [Rise at] : 波形を定義するため、クロックの立ち上がりエッジの時間を指定します。
  • [Fall at] : 波形を定義するため、クロックの立ち下がりエッジの時間を指定します。
    ヒント: クロックの波形は、自動的に周期の 50% と定義されます。デフォルトの波形を変更するには、[Rise at]/[Fall at] 値を使用してください。
  • [Jitter] : 選択したプライマリ クロックに関連する入力ジッターを指定します (オプション)。ジッター値は set_input_jitter コマンドを使用してクロックと関連付けます。

[OK] をクリックすると、現在選択しているクロックまたはタイミング制約ウィザードのクロックへのクロックの属性が追加されます。
