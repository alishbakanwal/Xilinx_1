[Add Netlist Sources] ページ

[Add Netlist Sources] ページでは、合成後のネットリストを現在のプロジェクトに追加できます。Vivado® Design Suite では、EDIF、NGC、構造 System Verilog、または構造 Verilog 形式のネットリストと Vivado デザイン チェックポイント (DCP) ファイルをインポートできます。デザイン ファイルはこの順序でデザインに読み込まれて、コンパイルされます。コンパイルの順序および [Hierarch Update] コマンドの詳細は、『Vivado Design Suite ユーザー ガイド : Vivado IDE の使用』 (UG893) のこのセクションを参照してください。

重要: EDIF および NGC ファイルを使用する場合は、最上位のセル名がセルが含まれているファイル名と一致している必要があります。
  • [Add Files] : ファイル参照ダイアログ ボックスが開き、プロジェクトに含めるネットリスト ソース ファイルを任意の数だけ選択できます。複数のファイルを選択すると、ファイル リストに指定した順に追加されます。
    ヒント: をクリックして [Add Files] をクリックしても同じ操作を実行できます。
  • [Add Directories] : ファイル参照ダイアログ ボックスが開き、デザイン ソース ファイルを検索するディレクトリを任意の数だけ選択できます。指定したディレクトリ内でネットリスト ソース ファイルが検索されて現在のプロジェクトに追加されます。
    ヒント: をクリックして [Add Directories] をクリックしても同じ操作を実行できます。
  • [Remove] () : 現在のプロジェクトから選択したファイルが削除されます。ファイルは現在のプロジェクトから削除されますが、ハード ドライブからは削除されません。
  • [Move Up] : ファイル リストでファイルが選択されている場合、上矢印ボタン () が使用できるようになり、リスト内でファイルを上に移動できます。
  • [Move Down] : ファイル リストでファイルが選択されている場合、下矢印ボタン () が使用できるようになり、リスト内でファイルを下に移動できます。
  • [Copy sources into project] : ローカル プロジェクト ディレクトリの sources フォルダーに指定したデザイン ソース ファイルをコピーできます。ローカルおよびリモート ソースの使用に関する詳細は、『Vivado® Design Suite ユーザー ガイド : システム レベル デザイン入力』 (UG895) のこのセクションを参照してください。
  • [Add sources from subdirectories] : [Add Directories] で指定したディレクトリのサブディレクトリに含まれるソース ファイルがすべて追加されます。

[Next] をクリックして [Add Constraints] ページに進みます。
