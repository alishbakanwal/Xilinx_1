[Report Clock Networks] ダイアログ ボックス

[Report Clock Networks] ダイアログ ボックスを使用すると、Vivado IDE で [Clock Networks] ウィンドウを生成して開くことができます。クロック ネットワーク レポートには、開いている合成後またはインプリメント後のデザインのクロック ツリーのクロック ビューが表示されます。

  • [Results name] : Vivado® IDE でクロック ネットワーク レポートに付ける名前を指定します。
  • [Open in a new tab] : Vivado IDE の新しいタブにクロック ネットワーク レポートが開きます。オフの場合は、新しいレポートで最後に開いたレポートが上書きされます。

[OK] をクリックすると、クロック ネットワーク レポートが生成されます。
