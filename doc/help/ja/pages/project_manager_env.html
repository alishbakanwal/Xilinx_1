Project Manager 環境

Project Manager 環境では、インメモリ デザインを開かずに現在のプロジェクトを実行できます。この環境には、Flow Navigator の [Project Manager] をクリックするといつでもアクセスできます。

この環境では、次を実行できます。

  • デザイン ソース、制約、およびシミュレーション ソースを追加
  • Vivado® IP カタログにアクセスし、IP をコンフィギュレーションし、ユーザー リポジトリを追加
  • Vivado 言語テンプレートにアクセスし、新しい RTL モジュールを定義
  • [Project Summary] を確認して、リソース使用率およびタイミングを含むデザインのステートを決定
  • ターゲット パーツの変更、合成、インプリメンテーション、シミュレーション、ビットストリーム オプションのコンフィギュレーションなど、[Project Settings] ダイアログ ボックスを使用して現在のプロジェクトをコンフィギュレーション
