[Security Register Setup] ページ (オプション)

Program eFUSE Registers ウィザードのこのページでは、RSA 認証を使用したり、BBRAM 暗号化を無効にしたりといった UltraScale® アーキテクチャの主な機能をオン/オフにする FUSE_SEC レジスタを管理できます。

  • [Enable security register programming] : チェック ボックスでさまざまな eFUSE 制御へのアクセスを個別にオンまたはオフにできます。このページには各セキュリティ レジスタの簡単な説明が記述されます。特定の設定の詳細は、次のマニュアルを参照してください。
