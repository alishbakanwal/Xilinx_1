[ILA Status] ウィンドウ

このウィンドウは、[Hardware ILA] ダッシュボードに表示されるデフォルトの 5 つのウィンドウの 1 つで、ILA デバッグ コアのステートの視覚的なサマリがリアルタイムに表示されます。このウィンドウには、[Core status] および [Capture status] がグラフィックで表示され、コアのトリガー イベントおよびデータ キャプチャ サイクルを確認できるようになっています。

[ILA Status] ウィンドウのツールバーには、次のコマンドが含まれます。

Toggle Auto-Retrigger 自動再トリガーが有効になり、問題のないトリガー、キャプチャ、表示が完了した後に、ILA コアのトリガーが実行し直されます。
Run Trigger コアのベーシックまたはアドバンス トリガー設定で定義されたトリガー イベントが選択した ILA コアで検出されるようになります。
Run Immediate 選択した ILA コアがトリガー設定に関係なく即座にトリガーされるようになります。
Stop Trigger 選択した ILA コアのトリガーが停止されます。
Compile Trigger State Machine ILA トリガーを実行せずにトリガー ステート マシンがコンパイルされます。
ヒント: トリガー ステート マシンは、トリガーが実行されると自動的にコンパイルされます。
