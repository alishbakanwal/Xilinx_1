[Peripheral Details] ページ

  • [Name] : 新しい AXI4 ペリフェラルの名前を入力します。
  • [Version] : <major#.minor#> バージョン表示方法に従ったバージョン番号を入力します。
  • [Display Name] : Vivado® IP カタログにペリフェラルをリストする際に使用する表示名を指定します。
  • [Description] : ペリフェラルの説明を入力します。
  • [IP Location] : 新しい AXI IP を作成するディレクトリを指定します。
  • [Overwrite existing] : 同じ名前のディレクトリが存在する場合に上書きするときにオンにします。
AXI ペリフェラル IP の名前とバージョンが組み合わせられて指定ディレクトリに IP のディレクトリ名が作成されます。
ヒント: [Overwrite existing] がオフのときに指定した名前とバージョンの組み合わせが既に存在している場合は、エラー メッセージが表示されます。
