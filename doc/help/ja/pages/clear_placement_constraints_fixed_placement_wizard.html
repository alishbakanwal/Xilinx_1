[Fixed Placement] ページ

このページの選択肢は、ここまでの Clear Placement ウィザードでの選択に固定したセルがあるか、ポートがあるかによって異なります。
ヒント: 固定されたセルはユーザーが割り当てた配置で、固定されていないセルはツールで自動的に割り当てられた配置です。詳細は、『Vivado® Design Suite ユーザー ガイド : 制約の使用』 (UG903) のこのセクションを参照してください。
  • [Fixed Cells] : 未配置になるセルに固定位置がある場合にのみ表示されます。
    • [Keep <#> fixed cells] : [Clear Placement] コマンドが適用される可能性のある固定セルの配置は削除されません。 <#> は、固定セルの数になります。
    • [Unplace all <#> placed cells] : [Clear Placement] コマンドが適用される可能性のあるすべてのセルの配置が削除されます。 <#> は、影響のあるセルの数になります。
  • [Fixed Ports] : 未配置になるポートに固定位置がある場合にのみ表示されます。
    • [Do not unplace any fixed ports] : [Clear Placement] コマンドが適用される可能性のある固定ポートの配置は削除されません。
    • [Keep <#> fixed ports] : 選択した固定ポートの配置は削除されません。 <#> は選択した固定ポートの数になります。
    • [Unplace all <#> placed ports] : [Clear Placement] コマンドが適用される可能性のあるすべてのポートの配置が削除されます。 <#> は、影響のあるポートの数になります。

[Next] をクリックして [Clear Placement Summary] ページに進みます。
