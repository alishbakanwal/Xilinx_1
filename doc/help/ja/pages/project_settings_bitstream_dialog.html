[Project Settings] ダイアログ ボックス : [Bitstream] ページ

[Project Settings] ダイアログ ボックスでは、デザイン フローを通して使用されるさまざまな設定に対して値を指定できます。設定ページは、合成、シミュレーション、インプリメンテーションなどのデザイン フローの手順別に分類されています。

[Bitstream] ページでは、ビットストリーム生成に影響するオプションを定義および変更できます。

  • [Configure additional bitstream settings] : [Edit Device Properties] ダイアログ ボックスが開き、[Enable Bitstream Compression] および [Enable Bitstream Encryption] などのビットストリームのその他のプロパティを設定できます。デバイス コンフィギュレーション設定の詳細は、『Vivado® Design Suite ユーザー ガイド : プログラムおよびデバッグ』 (UG908) のこのセクションを参照してください。
    ヒント: この選択肢は、インプリメント済みデザインが開いている場合にのみ使用できます。インプリメント済みデザインが開いていない場合は、インプリメント済みデザインを開くと、その他のビットストリーム設定ができるようになることを示すメッセージが表示されます。
  • [Write Bitstream Options] : ビットストリームを書き込む際に使用可能な特定のオプションが表形式で表示されます。表のオプションを選択すると、そのオプションの詳細が表示されます。
