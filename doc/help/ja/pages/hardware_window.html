[Hardware] ウィンドウ

[Hardware] ウィンドウには、Vivado のハードウェア マネージャーで開いている現在のハードウェア サーバー、ハードウェア ターゲット、およびハードウェア デバイスのツリー ビューが表示され、そのハードウェアのステータスが表示されます。[Hardware] ウィンドウでデバイス、ターゲット、またはハードウェア サーバーを選択すると、[Properties] ウィンドウで選択したオブジェクトのプロパティが表示されます。

[Hardware] ウィンドウの右クリック メニューからは、選択したハードウェア サーバーやターゲットを閉じたり、更新したりできます。デバイスを選択して右クリック メニューには、デザインのビットストリームをデバイスにプログラムしたり、デザインにインプリメントされているシステム モニターとデバッグ コアを管理したりするコマンドが含まれています。

ローカル ツールバーには、次のコマンドが含まれます。

Search 検索パターンを入力する検索フィールドが開きます。
Collapse all 現在のデバイス階層の展開を閉じて、システム モニターとデバッグ コアが表示されないようにします。
Expand all 現在のデバイス階層を展開して、システム モニターとデバッグ コアを表示します。
Scroll 別のウィンドウでオブジェクトを選択すると、[Hardware] ウィンドウでその選択したオブジェクトまで自動的にスクロールされるようになります。
Auto Connect ローカル ホストで実行されるハードウェア サーバーとターゲットに自動的に接続されます。
ヒント: このコマンドは、現在のターゲットが閉じている場合にのみ使用できます。
Run Trigger コアのベーシックまたはアドバンス トリガー設定で定義されたトリガー イベントが選択した ILA コアで検出されるようになります。
Run Immediate 選択した ILA コアがトリガー設定に関係なく即座にトリガーされるようになります。
Stop Trigger 選択した ILA コアのトリガーが停止されます。
