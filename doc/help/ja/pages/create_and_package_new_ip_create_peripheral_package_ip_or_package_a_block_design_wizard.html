[Create Peripheral, Package IP, or Package a Block Design] ページ

  • [Package your current project] : 現在開いているプロジェクトを Vivado® IP カタログに追加する IP コアとしてパッケージします。
  • [Package a block design from the current project] : 現在のプロジェクトのブロック デザインを IP コアとしてパッケージします。
    ヒント: このオプションは、現在のプロジェクトにブロック デザインが含まれているときのみに使用できます。
    • [Select a block design] : 現在のプロジェクトに含まれているブロック デザインのリストからブロック デザインを選択します。
  • [Package a specified directory] : IP パッケージのために分類されたファイルのディレクトリを指定します。Vivado ツールでは、ディレクトリの内容を読み込む際に IP のパッケージをしやすくするための推論ルールが使用されます。
  • [Create a new AXI4 peripheral] : HDL、ドライバー、テスト アプリケーション、BFM (Bus Functional Model)、および BFM サンプル テンプレート例を含む AXI4 ペリフェラルが作成されます。
    重要: BFM モデルを使用する場合は、BFM ライセンスを取得する必要があります。
