Open New Hardware Target ウィザード

Vivado® Design Suite ハードウェア マネージャーを開くと、デザイン ビットストリームをターゲットにダウンロードでき、ハードウェア マネージャーの Vivado ロジック アナライザー機能と Vivado シリアル I/O アナライザー機能を使用できます。新しいハードウェア ターゲットを開くには、まずローカル ホストかリモート システムのいずれかで実行されるハードウェア サーバーに接続する必要があります。ハードウェア サーバーでは、Vivado Design Suite または Vivado Lab Edition から物理的なプログラミング ターゲットまでの接続が管理されます。これは、ハードウェア プログラマまたはテスト ボードに接続されているマシンで実行する必要があります。

hw_server コマンドを別のアプリケーションとして起動する必要があります。このコマンドは、Vivado Design Suite インストール ディレクトリの /bin フォルダーに含まれます。詳細は、『Vivado® Design Suite ユーザー ガイド : プログラムおよびデバッグ』 (UG908) のこのセクションを参照してください。

ハードウェア ターゲットを開くには、ウィザードの次のページに進む必要があります。

  • [Hardware Server Settings] ページ
  • [Select Hardware Target] ページ
  • [Open Hardware Target Summary] ページ

[Next] をクリックして [Hardware Server Settings] ページに進みます。
