[Create HDL Wrapper] ダイアログ ボックス

[Create HDL Wrapper] ダイアログ ボックスでは、現在のブロック デザインに対して新規 HDL ラッパーを定義できます。ラッパーでは、ブロック デザインが最上位デザインとして定義されるので、ブロック デザインを合成、インプリメンテーションして、ビットストリームの生成ができます。

  • [Copy generated wrapper to allow user edits] : HDL ラッパーのコピーが作成され、それがプロジェクト ソース ファイルに追加されるので、必要に応じて編集できます。
    重要: ブロック デザインの外部インターフェイスを変更した場合は、ラッパーを手動でアップデートする必要があります。ラッパーは、Vivado® ツールでは自動的にアップデートされません。
  • [Let Vivado manage wrapper and auto-update] : Vivado で自動的に新規ラッパーが管理され、ブロック デザインが変更される際に必要に応じてアップデートされるようになります。

[OK] をクリックすると、HDL ラッパーが作成され、最上位デザインに追加されます。
