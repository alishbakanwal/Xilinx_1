[Unplace Ports] ページ

Clear Placement ウィザードのこのページには、選択した (または選択解除した) ポートに関連する情報が表示されます。
ヒント: このページは、[Clear Placement] コマンドを実行する前にポートを選択した場合にのみ表示されます。
  • [Ports to Unplace] :
    • [Unplace <#> selected ports] : 現在選択されているポートの配置制約が削除されます。 <#> は選択したポートの数になります。
    • [Unplace all except for <#> selected ports] : 現在選択されているポート以外のポートすべての配置制約が削除されます。
    • [Unplace all ports] : デザイン内のポートすべての配置が削除されます。
    • [Do not unplace any ports] : 選択した (または選択解除した) ポートの配置は削除されません。

[Next] をクリックして [Unplace Cells] ページに進みます。
