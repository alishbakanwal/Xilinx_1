[Create Interface Definition] ダイアログ ボックス

インターフェイス定義を使用すると、論理信号を共通のインターフェイスにまとめて、IP インテグレーターのブロック図内の IP 同士を接続できます。このダイアログ ボックスでは、新しい空のインターフェイスを定義するか、既存のインターフェイス定義を使用して新しいインターフェイスを作成できます。

  • [Create a new interface definition] : 新しいバンク インターフェイスの定義を作成します。
  • [Extend from] : 既存のインターフェイス定義を開始点として使用して、新しいインターフェイスを作成します。[Browse] ボタン () をクリックして [Interface Definition Chooser] ダイアログ ボックスを開き、開始点として使用するインターフェイスを選択します。
  • [Identification] : [Vendor:Library:Name:Version] (VLNV) 設定を組み合わせて、新規インターフェイス定義に対して固有識別子を作成します。
  • [Location] : 新規インターフェイス定義を書き込む IP リポジトリを定義します。指定したリポジトリは、[Project Settings] ダイアログ ボックスの [IP] ページを使用して Vivado® IP カタログ に追加して、Vivado ツールでインターフェイス定義が正しく使用されるようにする必要があります。

[OK] をクリックすると、[Interface Definition] ウィンドウが開き、インターフェイスの定義を完了できます。
