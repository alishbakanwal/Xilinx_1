ブロック デザインの [Design Hierarchy] ウィンドウ

Vivado® IP インテグレーターの [Design Hierarchy] ウィンドウには、ブロック デザイン階層のツリー ビューがインターフェイス接続とネット別に表示されます。このウィンドウには、階層セルと Vivado IP カタログまたは IP リポジトリから追加された IP の両方のブロック デザイン セルが表示されます。

[Design Hierarchy] ウィンドウのツールバーには、次のコマンドが含まれます。

Search 検索パターンを入力する検索フィールドが開きます。
Collapse all すべてのオブジェクト階層の展開が閉じられます。
Scroll オブジェクトをデザイン キャンバスなどの別のウィンドウで選択した場合に、[Design Hierarchy] ウィンドウでその選択したオブジェクトがスクロールして表示されます。
Diagram 最上位の [Design Canvas] ウィンドウに選択したオブジェクトが表示されるか、階層セルが選択されている場合は新しいデザイン キャンバス ウィンドウを開いてその階層セルの内容が表示されます。
