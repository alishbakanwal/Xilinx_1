[Add Interfaces] ページ

このページでは、スレーブ/マスター インターフェイス ピンおよび割り込みピンを AXI4 ペリフェラルに追加して設定できます。

  • [Enable Interrupt Support] : IP に割り込みピンを追加する場合にオンにします。生成された IP では、エッジまたはレベル割り込みがサポートされます。
  • [Add Interface] : 追加するインターフェイスがリストされている中央のウィンドウ上部にある緑色のプラス記号 () をクリックして新しいインターフェイス ピンをペリフェラル IP に追加します。
  • [Remove Interface] : ペリフェラル IP からインターフェイス ピンを削除するには、赤色のマイナス記号 () をクリックします。
  • [Name] : 中央ウィンドウで選択されているインターフェイスの名前を指定します。
  • [Interface Type] : AXI インターフェイスのインターフェイス タイプを指定します。サポートされているインターフェイス タイプは AXI4、AXI4-Lite、および AXI4-Stream です。詳細は、「AXI4 の詳細」を参照してください。
  • [Interface Mode] : 選択したインターフェイスのモードを [Slave] または [Master] に設定します。
  • [Data Width (Bits)] : AXI4 インターフェイスの読み出し/書き込みデータ幅を定義します。データ幅の単位はビットで、選択した AXI4 インターフェイスによって異なります。使用可能な値から適切なデータ幅を選択します。
  • [Memory Size (Bytes)] : メモリ サイズ (MEMORY_SIZE) の単位はバイトで、AXI4 スレーブ インターフェイスでのみサポートされます。使用可能な値から適切なメモリ サイズを選択します。
  • [Numb er of Registers] : レジスタ (NUM_REG) 数の指定は、AXI4-Lite スレーブ インターフェイスでのみサポートされています。

AXI4 ペリフェラル IP にインターフェイス ピンを追加し、コンフィギュレーションが完了したら、[Next] をクリックして [Create Peripheral] ページに進みます。
