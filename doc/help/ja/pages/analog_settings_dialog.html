[Analog Settings] ダイアログ ボックス

このダイアログ ボックスでは、[Waveform] ウィンドウのアナログ信号の波形特性を指定できます。タイムラインを右クリックして [Waveform Style] をクリックすると、信号をデジタルかアナログかのいずれかに定義できます。[Analog Settings] ダイアログ ボックスでは、その信号の波形特性を定義できます。

  • [Row Height] : [Waveform] ウィンドウのアナログ信号波形に使用するピクセル数を指定します。
  • [Y Range] : アナログ信号波形の Y 軸の値の範囲を定義します。
    • [Auto] : シミュレーション中の信号の値に基づいて、[Waveform] ウィンドウで Y の値の範囲が自動的に定義されるようになります。
    • [Fixed] : アナログ信号波形の最小および最大値を指定します。波形は、指定した値の範囲内で描かれます。
      • [Min] : 波形の一番下に表示される最小値を指定します。
      • [Max] : 波形の一番上に表示される最大値を指定します。
  • [Interpolation style] : データがまばらな場合に、信号波形を補間する方法を指定します。現在の波形カーブの線形延長を実行する場合は [Linear] を、次の値まで最後の値を維持する場合は [Hold] を指定します。
  • [Off scale] : 値が指定した [Y Range] の値を超える場合に信号を処理する方法を指定します。範囲を超える波形を非表示にする場合は[Hide] を、最小値または最大値で信号を切って指定した範囲内に収まるようにするには [Clip] を、上下の近い信号波形を重ねて表示させるには [Overlap] を指定します。
  • [Horizontal Line] : アナログ波形の中心線に指定した [Y Value]で水平方向の線を引きます。
    • [Y Value] : 水平方向の線を引くように指定した場合の、アナログ波形の値を指定します。
