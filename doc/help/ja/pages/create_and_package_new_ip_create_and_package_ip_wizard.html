Create and Package New IP ウィザード

Create and Package New IP ウィザードでは、現在または指定のディレクトリ、あるいは現在のプロジェクトに含まれるブロック デザインから新しい IP コアを定義できます。AXI4 ペリフェラルを作成することもできます。

  • ペリフェラルの作成、IP のパッケージ、またはブロック デザインのパッケージ
  • 現在のプロジェクトのパッケージ
  • IP の新規作成
  • ブロック デザインのパッケージ
  • 指定したディレクトリのパッケージ
  • IP パッケージャー プロジェクト名の編集
  • ペリフェラルの詳細
  • インターフェイスの追加
  • ペリフェラルの作成
