[Default Part] ページ

[Default Part] ページでは、テンプレート デザインで使用可能な使用可能なターゲット パーツまたはボードが表形式で表示されます。テンプレート デザインには、特定のパーツまたはボードのセットをターゲットにしたピン割り当ておよびタイミング制約が含まれています。この表には、デバイス名、I/O ピン数、ブロック RAM 数、および DSP 数が示されています。

デザインのターゲット パーツまたはボードを選択したら [Next] をクリックして、[New Project Summary] ページに進みます。
ヒント: [Configurable Microblaze Design] テンプレートを使用している場合は、[Configure Project] ページが次に開きます。
