[Open Hardware Target Summary] ページ

[Open Hardware Target Summary] ページには、ハードウェア サーバーへの接続とハードウェア ターゲットに関するレポートが表示されます。このサマリで定義した内容が正しいかどうかを確認します。

  • [Back] をクリックすると、前のページに戻って必要に応じて選択を変更できます。
  • [Finish] をクリックすると、指定どおりに hw_server とターゲットに接続されます。
  • [Cancel] をクリックすると、hw_server に接続せずにウィザードが閉じます。

サーバーへの接続が構築され、hw_server が開いたら、ハードウェア マネージャーにより Program and Debug 環境が開きます。[Hardware] ウィンドウには、現在のターゲットが表示されます。
