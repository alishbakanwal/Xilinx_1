[Create User Defined Probe] ダイアログ ボックス

このダイアログ ボックスでは、カスタムのユーザー定義のプローブが作成できます。新しいプローブでは、既存プローブの特定のビット値をまとめて、波形ビューアーで表示されるデータを単純または明確にできます。このダイアログ ボックスの [Probe Bits] の表に物理的な ILA プローブ ポートおよび定数値を追加すると、新しいカスタム プローブを定義できます。

ユーザー定義のプローブでは、ILA コアからの単一の物理的プローブからビット値をマップしたり、複数の物理的プローブからのビット値を単一のユーザー定義のプローブにまとめたりできます。単一のプローブからのビットをマップするプローブは、トリガーを作成したり、データを表示するために使用できます。複数の物理的プローブからのビットをまとめたプローブは、[ILA Waveform] ウィンドウにデータを表示したり、ILA データ ファイルにエクスポートするためだけに使用できます。

  • [ILA Core] : 現在のデバイスにプログラムされた ILA デバッグ コアのリストから選択します。
  • [New Probe Name] : ユーザー定義のプローブの名前を指定します。プローブ名はツールで自動的に生成されますが、必要に応じて変更できます。
    重要: バス プローブの場合は、名前にビットの範囲を含めておく必要があります。
  • [Add] () :
    • [Add Probe] : [Add Probes] ダイアログ ボックスが開き、追加するプローブを現在の ILA デバッグ コアで定義されている使用可能なプローブから選択できます。選択したプローブは、現在の位置のユーザー定義プローブに追加されます。
    • [Add Constant] : 定数値付きの単一ビットが現在の位置のユーザー定義プローブに挿入されます。定数値のデフォルトは 1 ですが、必要に応じて変更できます。
  • [Remove] () : 現在選択しているプローブまたは定数が削除されます。
    ヒント: バス プローブの個別ビットを選択して削除することもできます。
  • [Move Up] : [Probe Bits] の表で 1 つまたは複数のプローブが選択されている場合、青色の上方向矢印ボタン () が使用できるようになり、リスト内でファイルを上方向に移動できます。
  • [Move Down] : [Probe Bits] の表で 1 つまたは複数のプローブが選択されている場合、青色の下方向矢印ボタン () が使用できるようになり、リスト内でファイルを下方向に移動できます。

[OK] をクリックすると、ユーザー定義のプローブが作成され、ダイアログ ボックスが閉じます。
