[Project Name] ページ

  • [Project name] : 作成するサンプル プロジェクトの名前を入力します。
  • [Project location] : サンプル プロジェクトを作成するディスクまたはネットワーク ディレクトリを指定します。ファイル参照ダイアログ ボックスを開いてプロジェクト ディレクトリまでナビゲートすることもできます。
  • [Create project subdirectory] : 指定したプロジェクト ディレクトリのプロジェクトと同じ名前でサブディレクトリが作成され、プロジェクト ファイルおよびフォルダーが保存されます。オフの場合は、プロジェクト ファイルおよびフォルダーは指定したプロジェクト ディレクトリに直接書き込まれます。
