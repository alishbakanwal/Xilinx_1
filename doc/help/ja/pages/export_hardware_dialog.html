[Export Hardware] ダイアログ ボックス

このダイアログ ボックスを使用すると、Vivado Design Suite から Xilinx® Software Development Kit (SDK) までのデザインの移行が単純化され、ソフトウェア アプリケーションが作成されて、ELF (Executable Linkable File) ファイルとハードウェアが関連付けられます。MicroBlaze、Zynq®-7000 All Programmable SoC、または Zynq UltraScale+™ MPSoC などエンベデッド プロセッサと一緒に、IP インテグレーターからのブロック デザインを含む最上位デザインをエクスポートするために使用します。

  • [Include bitstream] : エクスポートされるハードウェア仕様またはハードウェア記述ファイルにビットストリーム ファイルが含まれるようになります。デザインがインプリメントされ、ビットストリームが生成されたら、ビットストリームを SDK にエクスポートしてソフトウェア アプリケーションを開発できます。
    ヒント: プログラミング ロジック (PL) ブロックに何のロジックも含まれないような珍しい状況で、合成後にデザインをエクスポートできます。
  • [Export to] : SDK でハードウェア プロジェクトを作成するため、ハードウェア仕様ファイル (.hdf) へのパスを指定できます。デフォルトでは、ローカルのプロジェクト ディレクトリ構造にファイルが出力されます。
