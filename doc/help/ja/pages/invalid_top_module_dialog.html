[Invalid Top Module] ダイアログ ボックス

このダイアログ ボックスは指定した新しいリコンフィギャラブル モジュール (RM) のトップ モジュールがモジュールを定義するファイル内に見つからなかった場合に表示され、どう処理するかをユーザーが指定できます。

  • [Specify a new top module] : 新しいトップ モジュールの名前を指定するテキスト文字列を入力するか、参照ボタン () をクリックして候補のリストを表示させます。
  • [Ignore and continue with invalid top module] : 問題はこの時点では無視して、後で処理することにします。
  • [Automatically pick new top module] : ツールにより、使用可能なソース ファイルに基づいて最適なトップ モジュールが自動的に検出されます。

選択したら、[OK] をクリックします。
