New IP Location ウィザード

New IP Location ウィザードでは、新しい Manage IP プロジェクトを特定のディレクトリに指定できます。IP カタログからの IP をカスタマイズでき、別のデザイン プロジェクトでそのカスタマイズ IP をサポートするために必要な出力ファイルの生成もできます。[Next] をクリックして [Manage IP Settings] ページに進みます。
