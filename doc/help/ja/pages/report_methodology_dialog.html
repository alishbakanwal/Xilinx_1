[Report Methodology] ダイアログ ボックス

[Report Methodology] ダイアログ ボックスでは、設計手法チェックを実行して、現在のデザインのエラーや問題を検出できます。手法解析は、特に設計手法に準拠しているかどうかをチェックするデザイン ルール チェックの特殊な形式で、そのプロセスを実行するとよく発生するエラーが検出されるようになっています。

  • [Results name] : Vivado® IDE の [Methodology Results] ウィンドウのレポートに適用する名前を指定します。
  • [Export to file] : ノイズ レポートの結果が指定ファイルに書き込まれます。ファイル形式は CSV、HTML、またはテキストのいずれかとして指定できます。
  • [Open in a new tab] : Vivado IDE の [Methodology Results] ウィンドウに新しいタブでレポートが開きます。オフの場合は、新しいレポートで最後に開いたレポートが上書きされます。
