[Import Ports] ページ (オプション)

[Import Ports] ページでは、I/O プランニング プロジェクトで使用するポート定義を含む CSV (Comma Separated Value) または XDC ファイルをインポートできます。

  • [Import CSV] : オンにすると、インポートする CSV ファイルを指定できます。CSV は、メイン メニューから [File] → [Export I/O Ports] をクリックすると作成できます。
  • [Import XDC] : オンにして XDC ファイルを指定します。XDC ファイルでは、create_port などの Tcl コマンドを使用してデザインのポート リストが定義されます。
  • [Do not import I/O ports at this time] : I/O ポート リストのインポートを飛ばします。この時点で飛ばしても、後でポートを定義またはインポートする必要があります。

[Next] をクリックして [Default Part] ページに進みます。
