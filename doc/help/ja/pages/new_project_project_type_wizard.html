[Project Type] ページ

  • [RTL Project] : VHDL、Verilog、または SystemVerilog デザイン ソースおよび前に合成済みのデザイン モジュールの EDIF ネットリストの使用をサポートする、新しい RTL プロジェクトを作成します。[Next] をクリックして[Add Sources] ページに進みます。
    • [Do not specify sources at this time] : 空の新しいプロジェクトを作成して、デザイン ソースは後で指定します。オンにした状態で [Next] をクリックすると、途中のページを飛ばして [Default Part] ページに進むことができます。
  • [Post-synthesis Project] : Vivado® 合成またはその他の合成ツールにより生成した EDIF ネットリストを使用して合成後のネットリスト プロジェクトを作成します。合成後のプロジェクトは、配置配線および前に生成済みのネットリスト デザインのビットストリームを生成するのに使用できます。[Next] をクリックして [Add Netlist Sources] ページに進みます。
    • [Do not specify sources at this time] : 空の新しいプロジェクトを作成して、デザイン ソースは後で指定します。オンにした状態で [Next] をクリックすると、途中のページを飛ばして [Default Part] ページに進むことができます。
  • [I/O Planning Project] : 新規プロジェクトを作成し、ポートを初期段階で定義し、またパッケージ ピン ロケーションを割り当てます。詳細は、『Vivado Design Suite ユーザー ガイド : I/O およびクロック プランニング』 (UG899) を参照してください。[Next] をクリックして [Import Ports] ページに進みます。
  • [Imported Project] : 新しいプロジェクトを作成して ISE® Design Suite の Project Navgator、XST、または Synopsys Synplify の RTL プロジェクト データをインポートし、既存プロジェクトを Vivado Design Suite に移行します。プロジェクト ソース ファイルおよびコンパイル順はインポートされますが、インプリメンテーション結果およびプロジェクト設定はインポートされません。[Next] をクリックして[Import Project] ページに進みます。
  • [Example Project] : 使用可能なデザイン テンプレートを使用して新規サンプル プロジェクトを作成します。[Next] をクリックして[Select Project Template] ページに進みます。
