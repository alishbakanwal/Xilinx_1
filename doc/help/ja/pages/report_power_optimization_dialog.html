[Report Power Optimization] ダイアログ ボックス

このダイアログ ボックスを使用すると、現在のデザインの Vivado® Design Suite で実行される消費電力最適化がレポートできます。消費電力最適化は、インテリジェント クロック ゲーティングを使用してデザイン内の動的電力を削減するためのインプリメンテーション中に実行可能なオプションの手順です。ロジック最適化後または配置後に実行すると、デザインで必要な消費電力を削減できます。消費電力最適化では、デザインのクロックまたはロジックは変更されません。

  • [Results name] : Vivado IDE の [Power Opt Results] ウィンドウのレポートに適用する名前を指定します。
  • [Export to file] : 消費電力最適化の結果が指定ファイルに書き込まれます。ファイル形式はテキストまたは XML のいずれかとして指定できます。
  • [Open in a new tab] : Vivado IDE の [Power Opt Results] ウィンドウに新しいタブで消費電力最適化レポートが開きます。オフの場合は、新しいレポートで最後に開いたレポートが上書きされます。
