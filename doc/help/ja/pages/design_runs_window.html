[Design Runs] ウィンドウ

[Design Runs] ウィンドウには、合成およびインプリメンテーションの run を管理、起動、リセットするコマンドがあります。合成およびインプリメンテーション run のステータスおよび統計も表形式で表示され、run の実行時間、完了レート、リソース使用率、およびタイミング結果などの情報が各列に表示されます。

ヒント: [Design Runs] ウィンドウの列をカスタマイズするには、列ヘッダーを右クリックしてさまざまなメニュー コマンドにアクセスします。

ローカル ツールバーには、次のコマンドが含まれます。

Search 検索パターンを入力する検索フィールドが開きます。[I/O Ports] の表から指定した値が検索されます。
Collapse リストされた run の展開が閉じます。
Expand リストされた run が展開されます。
Launch Runs 選択した run を開始します。使用するプロセッサの数を指定し、リモート ホストで開始します。
Reset Run 終了または部分的に終了した run をリセットして、オプションでその run に関連付けられたデータを削除できます。
Launch Next Step ネットリスト最適化、配置、物理的最適化、および配線などのインクリメンタル ステップでインプリメンテーション run が開始されます。
Reset to Previous Step インプリメンテーション run を前の段階に戻して、編集して実行し直せるようにします。
Create Runs 新しい合成およびインプリメンテーション run がプロジェクトに追加されます。使用する run のターゲット パーツおよびストラテジを指定できます。
Open Run 選択した run が合成済みデザインまたはインプリメント済みデザインとして開かれます。
Show Percentages リソース使用率が使用可能なリソースのパーセントで表示されます。

複数の合成およびインプリメンテーション run をサポートするプロジェクトには、アクティブ run または現在 Vivado® Design Suite のメモリにある run も含まれます。アクティブ run の結果が [Messages]、[Compilation]、[Reports]、[Project Summary] ウィンドウに表示されます。非アクティブ run を選択して、右クリックで [Make Active] コマンドを使用すると、アクティブ run を変更できます。

右クリック メニューのその他のコマンドを使用すると、デザイン run のプロパティおよび設定を変更したり、レポートおよびログを表示したり、run ディレクトリを開いてディスクの結果ファイルおよびログを確認したりできます。
