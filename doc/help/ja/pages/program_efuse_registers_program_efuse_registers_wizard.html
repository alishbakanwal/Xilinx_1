Program eFUSE Registers ウィザード

ヒント: ワンタイム プログラマブル eFUSE レジスタを使用して、デバイス リソースとデバイスを暗号化スキームに組み込む前に、[Program BBR Key] コマンドを使用してデザインの暗号化スキームを定義およびテストすることをお勧めします。
このウィザードでは、指定したハードウェア デバイスの eFUSE レジスタを順を追って設定できます。 eFUSE レジスタは、次を含む 7 シリーズ、UltraScale®、および UltraScale+® デバイスのワンタイム プログラマブル ビットです。
  • FUSE_KEY : ビットストリーム暗号化用の AES キーを格納
  • FUSE_RSA : UltraScale アーキテクチャの RSA 認証キーを格納
  • FUSE_USER および FUSE_USER128 : デザイン リビジョンまたはデバイスのその他のデータを識別するのに使用可能なユーザー定義の HEX コードを格納
  • FUSE_CNTL : eFUSE レジスタへの読み出し/書き込みアクセスを制限するレジスタを制御
  • FUSE_SEC : UltraScale アーキテクチャの特別なセキュリティ設定を制御
注意: eFUSE はハードウェア デバイスのワンタイム プログラマブル セルで、ファクトリでプログラムされる Device DNA、AES-GCM/RSA 暗号キー、およびユーザー指定の値を格納するために使用されます。eFUSE レジスタの詳細は、『UltraScale アーキテクチャ コンフィギュレーション ユーザー ガイド』 (UG570) または『7 シリーズ FPGA コンフィギュレーション ユーザー ガイド』 (UG470) を参照してください。

Program eFUSE Registers ウィザードのページは、UltraScale アーキテクチャおよび 7 シリーズ デバイスによって異なります。UltraScale アーキテクチャの eFUSE レジスタを設定する場合は、次のページが表示されます。

  • [Cryptographic Key Setup] ページ (オプション)
  • [User Register Setup (Optional)] ページ
  • [Control Register Setup] ページ (オプション)
  • [Security Register Setup] ページ (オプション)
  • [Program eFUSE Registers Summary] ページ

7 シリーズ デバイスの eFUSE レジスタを設定する場合は、次のページが表示されます。

  • [AES Key Setup] ページ (オプション)
  • [Control Register Setup] ページ (オプション)
  • [Program eFUSE Registers Summary] ページ

デバイスの暗号キー (FUSE_KEY) およびユーザー コード (FUSE_USER) をプログラムするには、まずこのウィザードを使用することをお勧めします。この後、2 度目にウィザードを実行して、制御レジスタ (FUSE_CNTL) を無効にし、これらの eFUSE ビットへの読み出し/書き込みアクセスがされないようにします。最後にもう 1 度ウィザードを実行し、UltraScale アーキテクチャのセキュリティ レジスタ (FUSE_SEC) をプログラムします。
