[Create Block Design] ダイアログ ボックス

[Create Block Design] ダイアログ ボックスでは現在のプロジェクトに追加する新しいブロック デザイン (BD) を定義します。

  • [Design name] : 作成するブロック デザインの名前を指定します。
  • [Directory] : ローカル プロジェクト ディレクトリ、または現在のプロジェクト ディレクトリ外のディレクトリのいずれかに、ブロック デザイン ファイルを作成する箇所を指定できます。
  • [Specify source set] : デザイン ソース ファイルセット、デフォルトのシミュレーション ファイルセット、または新しいシミュレーション ファイルセットに新しいブロック デザインが追加されます。
    ヒント: 通常は、ブロック デザインはデザイン ソース ファイルセットに作成することをお勧めします。ファイルセットをシミュレーション ファイルセットに設定すると、シミュレーション専用のブロック デザインを作成できます。

[OK] をクリックすると、デザインが Vivado IDE の IP インテグレーターで開きます。
