[Synthesis Design Run Settings] ダイアログ ボックス

[Synthesis Design Run Settings] ダイアログ ボックスでは、選択した合成 run に使用するストラテジとオプションを定義できます。選択したストラテジのデフォルト オプションは表形式で表示され、カスタマイズするストラテジのオプションや値が変更できるようになっています。

  • [Strategy] : 定義済み Vivado® Design Suite ストラテジを指定するか、選択したストラテジのオプションを変更して独自のストラテジを作成できます。
    ヒント: 選択したストラテジのこのオプションを変更した場合は (ストラテジ名にアスタリスク (*) で表示)、[Save Strategy As] ボタン () をクリックすると、それらのオプションを新しいユーザー定義のストラテジとして保存できます。
  • [Description] : 選択したストラテジの目的のサマリを簡単に記述しておきます。この部分は、ユーザー定義のストラテジを保存する際に定義できます。
  • [Options] : 選択した合成ストラテジで使用可能なオプションが表形式で表示されます。表のオプションを選択すると、そのオプションの詳細が表示されます。[Options] の表の一番下の [More Options] フィールドを使用すると、synth_design コマンドにその他のオプションを追加できます。
