[Export Constraints] ダイアログ ボックス

このダイアログ ボックスでは、インメモリ デザイン制約を新しいザイリンクス デザイン制約ファイル (XDC) にエクスポートできます。

  • [Output file name] : 出力される XDC ファイルのパスとファイル名を指定します。
  • [Include unconstrained placement locations] : デフォルトでは、固定されたロケーション (LOC) のみが XDC ファイルに書き込まれます。配置が固定されているかどうかに関わらず、すべての配置ロケーションを制約ファイルにエクスポートできます。
    ヒント: 固定された LOC はユーザーの割り当てた配置で、固定されていない LOC はツールで自動的に割り当てられた配置です。詳細は、『Vivado® Design Suite ユーザー ガイド : 制約の使用』 (UG903) のこのセクションを参照してください。
