[Import Project] ページ

ISE® Design Suite、XST、または Synplify プロジェクト ファイルからデザイン ソースおよびプロジェクト設定をインポートできます。Vivado® IDE では指定したプロジェクト内のソース ファイルが検出されて自動的に新規プロジェクトにこれらのファイルが追加されます。また、最上位モジュール、ターゲット デバイス、および VHDL ライブラリの割り当てなどの設定も既存プロジェクトからインポートされます。

  • [ISE] : このボタンをオンにして ISE Design Suite の既存プロジェクト (.xise) を指定します。
  • [Synplify] : このボタンをオンにして Synopsys Synplify の既存プロジェクト (.prj) を指定します。
  • [XST] : このボタンをオンにして XST の既存プロジェクト (.xst) を指定します。
  • [Copy sources into project] : 指定したデザイン ソース ファイルがローカル プロジェクト ディレクトリの sources フォルダーにコピーされます。ローカルおよびリモート ソースの使用に関する詳細は、『Vivado® Design Suite ユーザー ガイド : システム レベル デザイン入力』 (UG895) のこのセクションを参照してください。
