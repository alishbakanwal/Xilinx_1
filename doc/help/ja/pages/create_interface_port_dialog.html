[Create Interface Port] ダイアログ ボックス

[Create Interface Port] ダイアログ ボックスでは、現在のブロック デザインに追加する新しい外部インターフェイス ポートを定義します。

  • [Interface name] : 作成するインターフェイス ポートの名前を指定します。
  • [VLNV] : 現在のブロック デザインで使用可能な定義済みインターフェイスの VLNV (Vendor:Library:Name:Version) がリストされます。
  • [Mode] : インターフェイスに [Master]、[Slave]、[Monitor] のいずれかを定義します。
  • [Connect to selected interface] : 新しいインターフェイス ポートが選択した互換性のあるポートまたはピンに接続されます。
    ヒント: このチェック ボックスは、[Create Interface Port] コマンドを実行するときに、デザインの インターフェイス ピンが選択されていると、オンになります。

[OK] をクリックすると、新しいインターフェイス ポートが作成され、オプションでそれに接続されます。
