[Report DRC] ダイアログ ボックス

[Report DRC] ダイアログ ボックスでは、デザイン ルール チェックの選択を確認でき、指定したレポートが [DRC Results] ウィンドウまたはテキスト ファイルに出力されるようになります。

  • [Results name] : Vivado® IDE で [DRC Results] ウィンドウに適用する名前を指定します。
  • [Output File] : DRC レポートの結果を指定ファイルに書き込みます。
  • [Rule Decks] : 実行するように選択できるさまざまなルール デックがリストされます。DRC ルール デックとは、デザイン フローの異なる段階で実行可能なデザイン ルール チェックのコレクションです。Vivado Design Suite には、定義済みルール チェックのセットが含まれます。 ユーザー定義のルール デックを作成することもできます。デフォルトのルール チェックには、デザインをチェックするのによく必要とされるルールが含まれています。
    ヒント: ルール デックを選択すると、そのコレクション内で定義されるルール チェックすべても選択されます。各チェックは [Rules] フィールドで有効または無効にできます。
  • [Rules] : [Report DRC] コマンド中に実行するチェックを個別に選択します。チェックをカテゴリ別にまとめて、個別チェックの表示を展開または非展開にできます。カテゴリで一部のチェックまたはすべてのチェックを必要に応じて選択できます。
  • [Open in a new tab] : Vivado IDE の [Results] ウィンドウに新しいタブで DRC 結果を開きます。オフの場合は、新しいレポートで最後に開いたレポートが上書きされます。
