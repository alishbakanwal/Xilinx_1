[Save Constraints As] ダイアログ ボックス

このダイアログ ボックスでは、現在のデザインのアクティブな制約セットのコピーを新しい名前で作成して保存できます。

  • [New Constraint set name] : 作成する制約セットを指定します。制約セットに関連する制約ファイルすべては、新しい制約セットの下にある新しいファイルに同じ名前でコピーされます。
  • [Make Active] : 選択した制約セットをプロジェクトのアクティブ制約セットにします。アクティブな制約セットには、現在のデザインに適用される制約すべてが含まれ、新しい制約が保存されます。
