[Create Source File] ダイアログ ボックス

このダイアログ ボックスでは、現在のプロジェクトに追加する Verilog または VHDL などの新しいデザイン ソース ファイルを定義します。

  • [File Type] : VHDL (.vhd)、Verilog (.v)、Verilog ヘッダー (.vh)、SystemVerilog (.sv)、およびメモリ ファイル (.mem) などのファイル形式がサポートされます。
  • [File Name] : デザイン ソース ファイルの名前を指定します。
  • [File Location] : 新しいソース ファイルのディレクトリは、ローカル プロジェクト ディレクトリ、または現在のプロジェクト ディレクトリ内にでも外にでも指定できます。
[OK] をクリックすると、[Sources] ウィンドウに表示されるソース リストに新しいファイルのプレースホルダーが追加されます。このソース ファイルは、Add Sources ウィザードで [Finish] をクリックするまで作成されません。
ヒント: [Finish] をクリックすると、[Define Module] ダイアログ ボックスが開き、Verilog、SystemVerilog、および VHDL ファイルのモジュールを定義できます。Verilog ヘッダーおよびメモリ ファイルは手動で作成する必要があります。
