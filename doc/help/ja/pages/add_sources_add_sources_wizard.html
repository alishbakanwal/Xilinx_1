Add Sources ウィザード

Add Sources ウィザードでは、さまざまなファイル タイプを現在のプロジェクトに追加できます。

  • 制約の追加または作成
  • デザイン ソースの追加または作成
  • シミュレーション ソースの追加または作成
  • DSP ソースの追加または作成
  • 既存ブロック デザインの追加
  • 既存 IP の追加
