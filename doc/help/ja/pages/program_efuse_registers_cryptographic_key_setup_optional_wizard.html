[Cryptographic Key Setup] ページ (オプション)

UltraScale® または UltraScale+® デバイスの場合、Program eFUSE Registers ウィザードのこのページでは、現在のデバイスの FUSE_KEY および FUSE__RSA レジスタをプログラムできます。AES (Advanced Encryption Standard) または難読化 AES 暗号キー、および RSA 暗号キーが指定したキー ファイルから読み出され、256 ビット レジスタに書き込まれ、デバイスをプログラムする際に解読が実行されるようになります。

インプリメント済みデザインには、write_bitstream コマンドを使用してビットストリームを生成する前に、暗号化 (ENCRYPTION) プロパティを適用しておく必要があります。ENCRYPTION プロパティは、Vivado IDE で [Tools] → [Edit Device Properties] をクリックし、[Edit Device Properties] ダイアログ ボックスの [Encryption] ページから追加できます。

  • [Enable AES key programming] :
    • [AES key file (.nky)] : 暗号キー ファイルのファイル名とパスを指定します。NKY ファイルには、AES キー、難読化 AES キー、および RSA キーを含めることができます。
      ヒント: デフォルトでは、このファイルは暗号化されたビットストリームの生成時にインプリメンテーション run ディレクトリに書き込まれます。
    • [AES key] : 上記で指定したファイルから読み出された AES 暗号キー値または難読化暗号キー値が表示されます。チェック ボックスをオンにすると、デバイスの AES 暗号化が実行されます。
    • [RSA key] : 上記で指定したファイルから読み出された RSA 暗号キー値が表示されます。チェック ボックスをオンにすると、デバイスの RSA 暗号化が実行されます。
暗号化されたビットストリームを使用してデバイスをプログラムするには、まず暗号キーを BBR または eFUSE レジスタにプログラムしてから、その暗号化されたビットストリームをデバイスにプログラムするという 2 段階のプロセスが使用されます。
program_hw_devices -key {efuse} [current_hw_device]
program_hw_device [current_hw_device]
