[Interface Definition] ウィンドウ

[Interface Definition] ウィンドウでは、新規インターフェイス定義を定義および編集できます。このウィンドウの一番上には、その新しいインターフェイス定義の [Vendor:Library:Name:Version] (VLNV) 情報が表示されます。また、そのインターフェイスの [Description] および [Display Name] も入力できます。ウィンドウの残りの部分は表形式で、そのインターフェイス定義に含まれる定義済みポートがリストされます。ポートに関連する情報の列は必要に応じて表示または非表示に設定できます。

[Interface Definition] ウィンドウのツールバーには、次のコマンドが含まれます。

Search ポートの表に適用するための検索パターンを入力する検索フィールドを開きます。検索パターンと一致するフィールドを持つポートが表示されます。
Add Interface Port インターフェイス定義の新規ポートを定義できる [Add Port] ダイアログ ボックスが開きます。
Remove Interface Port 選択したポートをインターフェイス定義から削除できます。
ヒント: このコマンドは、1 つまたは複数ポートが選択されている場合にのみアクティブになります。
Copy Ports [Interface Definition Chooser] ダイアログ ボックスが開き、ポート定義をコピーするインターフェイスを選択できます。
重要: このコマンドを実行すると、現在のポート定義すべてがコピーしたインターフェイスのポート定義に置換されます。インターフェイスのポートを追加または編集する前にこのオプションを使用すると、変更が失われます。

[Interface Definition] ウィンドウには [Save] ボタンと [Dicard] ボタンがあり、インターフェイス定義に変更があるとアクティブになります。[Save] ボタンをクリックすると、インターフェイス定義がそれが保存されている IP リポジトリに書き込まれ、[Discard] ボタンをクリックすると、変更がキャンセルされて、そのインターフェイスが最後に保存されたバージョンが読み込み直されます。
