[Package] ウィンドウ

[Package] ウィンドウには、ターゲット デバイスの外部パッケージがグラフィカルに表示されます。コンフィギュレーション ピン、クロック兼用 (CC) ピン、VCC および GND ピン、GT ピンなどさまざまなピン タイプがグラフィカルに表示されるので、使用可能なピンをすばやく区別できます。[Show Differential IO Pairs] ツールバー ボタン () をクリックして差動ピン ペアの表示をオンにすると、関連するピンを表示することもできます。

I/O ポートをインタラクティブにパッケージ ピンへ割り当てるには、[I/O Ports] ウィンドウでポートを選択して、それらを [Package] ウィンドウのピンにドラッグ アンド ドロップします。ツールバーにはオブジェクトの配置を制御するための次のコマンドが含まれます。

Place Ports 選択したポートを I/O バンクに配置するか、エリアに配置するか、シーケンシャルに配置するかを定義できます。I/O ポートの配置に関する詳細は、『Vivado Design Suite ユーザー ガイド : I/O およびクロック プランニング』 (UG899) のこのセクションを参照してください。
Autocheck I/O Placement インタラクティブな配置中に I/O 配置のデザイン ルール チェックを自動的に実行します。これらの DRC チェックをオンにしておくと、デザイン ルール違反を回避でき、ポートを最適に配置できます。
ヒント: オンにしておくと、デザイン ルール違反を回避できますが、複数のエラーがある場合は、これをオフにしてポート割り当てを修正する必要があることもあります。

デフォルトでは、[Package] ウィンドウにはパッケージの最上位がトップダウンで表示され、ピンは順番どおりに配列されますが、[Show Bottom View] ツールバー ボタン () をクリックすると、パッケージをひっくり返して表示できます。

拡大および縮小して、特定パッケージ ピンを確認するのに必要な詳細レベルで表示できます。ツールバー メニューには、何を表示するかを制御するための次のコマンドが含まれます。

Previous 拡大レベルおよび表示位置など、[Package] ウィンドウの前の表示が復元されます。
Next [Previous] で表示していた前の表示が元の表示に戻されます。
ヒント: このコマンドは、[Previous] コマンドを使用した後にのみ使用可能です。
Zoom In パッケージ ピンが拡大されます。
Zoom Out パッケージのエリアがより多く表示されます。
Zoom Fit パッケージ全体が表示されます。
Select Area [Package] ウィンドウで長方形を描画してオブジェクトを選択します。
Fit Selection 現在選択しているオブジェクトが [Package] ウィンドウの中央に拡大されて表示されます。
Autofit Selection 別のウィンドウで選択したオブジェクトが [Package] ウィンドウの中央に自動的に拡大されます。
[Package] ウィンドウでは、クリックによる表示の拡大/縮小機能がサポートされています。クリックしてドラッグしながら線やボックスを描画すると、パッケージを拡大または縮小表示できます。[Package] ウィンドウの右側または下側のスクロールバーを使用すると、画面をスクロールして別のエリアを表示できます。
ヒント: スクロールバーは、表示エリアがパッケージで拡大された場合にのみ表示されます。

[Package] ウィンドウは Vivado® IDE のワークスペースに表示されます。ワークスペースの詳細は、『Vivado Design Suite ユーザー ガイド : Vivado IDE の使用』 (UG893) のこのセクションを参照してください。
