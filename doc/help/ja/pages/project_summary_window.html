[Project Summary]

[Project Summary] には、ターゲット パーツとボード、合成およびインプリメンテーションのステータス、タイミング サマリとリソース使用率などのデザイン プロジェクトとデザイン情報が表示されます。サマリ情報の中には、あくまで見積もりにすぎないものもあるほか、合成およびインプリメンテーションが実行されないと使用できないものもあります。合成およびインプリメンテーションが完了したら、DRC 違反、タイミング値、使用率 (%)、消費電力の見積もりなどがサマリに追加されます。[Project Summary] には、[Message]、[Log]、[Reports] などのウィンドウおよび [Project Settings] ダイアログ ボックスなど、デザインの詳細な情報へのリンクも含まれます。

合成およびインプリメンテーション run が複数あるプロジェクトの場合、[Project Summary] ウィンドウに表示されるデータは現在のデザインのアクティブ run によって異なります。[Project Summary] には、アクティブな合成およびインプリメンテーション run が表示されます。アクティブ run は [Design Runs] ウィンドウで変更できるほか、[Tcl Console] ウィンドウ に Tcl コマンドの current_run を入力しても変更できます。

[Project Summary] は Vivado® IDE のワークスペースに表示されます。ワークスペースの詳細は、『Vivado Design Suite ユーザー ガイド : Vivado IDE の使用』 (UG893) のこのセクションを参照してください。
