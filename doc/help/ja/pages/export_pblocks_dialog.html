[Export Pblocks] ダイアログ ボックス

このダイアログ ボックスでは、選択した Pblock に割り当てられたデザイン エレメントの EDIF ネットリストがエクスポートできます。これにより、Pblock に割り当てられたロジックのネットリストを別のデザインでデザイン ソースとして使用できるようになります。最上位デザインの EDIF ネットリストもエクスポートできます。

  • [Directory name] : 選択した Pblock からエクスポートされる出力ファイルのディレクトリを指定します。top.edn ファイルは、指定したディレクトリと、選択した各 Pblock のディレクトリとネットリストに書き込まれます。
  • [File types to generate] : 選択した Pblock から出力する出力ファイル形式を指定します。
    重要: 現時点では、EDIF ネットリスト形式のみが使用でき、必ず選択する必要があります。これらのファイルはデザイン ソース ファイルとしてネットリスト プロジェクトに追加できますが、update_design を使用してデザインに読み込むことはできません。
  • [Add] () : [Select Pblock] ダイアログ ボックスが開き、任意の Pblock を選択できます。別のネットリストが選択した各 Pblock ごとに作成されます。
  • [Remove] () : エクスポートする Pblock のリストから選択した Pblock が削除されます。

[OK] をクリックすると、選択した Pblock からネットリストがエクスポートされます。
