Create Pblocks ウィザード

Create Pblocks ウィザードでは、現在のデザインのフロアプランを作成するために Pblock を作成するプロセスを順を追って指定できます。

フロアプランは、デザインが満たされない場合やタイミングが満たされたことがない場合に実行することをお勧めします。フロアプランをすることで、任意のデザイン階層レベルまたはその他の関連ロジックを Pblock というデバイスの予約領域に配置できるようになります。この Pblock を使用すると、領域に割り当てられたセルの高度な配置が指定できますが、その領域内の詳細な配置は配置ツールで実行されます。

Pblock を作成するには、Pblock の名前を定義し、デザインから Pblock へセルを割り当て、デバイスのエリアを Pblock に割り当てて、割り当てられたセルを配置にデバイス リソースを予約する必要があります。ウィザードには、次のページがあります。

  • [Select Cells] ページ
  • [Format Names] ページ
  • [Create Pblocks Summary] ページ

[Next] をクリックして Create Pblocks ウィザードの [Select Cells] ページに進みます。
