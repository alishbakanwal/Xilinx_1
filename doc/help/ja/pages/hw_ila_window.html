[Hardware ILA] ウィンドウ

[Hardware ILA] ウィンドウは、現在のデバイスにプログラムされた ILA (Integrated Logic Analyzer) デバッグ コアに関連付けられた設定、ステータス、トリガー入力と出力、キャプチャされたデータ サンプル、および波形動作などのコレクションとして動作するハードウェア マネージャーのダッシュボードです。ILA コアを使用すると、デザインの特定の信号のプローブ、プログラムされたハードウェア イベントでのトリガー、ザイリンクス FPGA デバイスからのリアルタイムでのデータ キャプチャを実行できます。

[Hardware ILA] ウィンドウに関連付けられるデフォルト ウィンドウは、次のとおりです。

  • [ILA Settings] ウィンドウ : ILA デバッグ コアのさまざまな設定にアクセスできます。
  • [ILA Status] ウィンドウ : ILA デバッグ コアのステートの視覚的なサマリがリアルタイムに表示されます。
  • [ILA Trigger Setup] ウィンドウ : ILA デバッグ コアに割り当てられたプローブにアクセスして、これらのプローブのトリガーを管理できるようになります。
  • [ILA Capture Setup] ウィンドウ : ILA デバッグ コアのキャプチャ モードが [BASIC] に設定される場合に、デバッグ プローブに関連するキャプチャ式を定義および管理できます。
  • [ILA Wave] ウィンドウ : ILA デバッグ コアのプローブに関連する波形動作が表示されます。

[Hardware ILA] ウィンドウ左側のスライドアウト ボタンをクリックすると、ダッシュボードに含まれる [Dashboard Options] 機能を使用することもできます。[Dashboard Options] を開くと、現在のデバイス、デバイスにプログラムされたデバッグ コア、各コアに関連付けられたウィンドウがリストされます。このリストのチェック ボックスを使用すると、ダッシュボードにウィンドウを追加または削除できます。

[Hardware ILA] ウィンドウは Vivado® IDE のワークスペースに表示されます。ワークスペースの詳細は、『Vivado Design Suite ユーザー ガイド : Vivado IDE の使用』 (UG893) のこのセクションを参照してください。
