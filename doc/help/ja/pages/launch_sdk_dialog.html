[Launch SDK] ダイアログ ボックス

ザイリンクス ソフトウェア開発キット (SDK) が起動されます。 SDK では、MicroBlaze、Zynq®-7000 All Programmable SoC、または Zynq UltraScale+™ MPSoC などのエンベデッド プロセッサを使用する IP インテグレーターからのブロック デザインを含む最上位デザイン用のソフトウェアが設計できます。このダイアログ ボックスでは、SDK で必要とされるファイルのディレクトリを指定できます。

  • [Exported location] : [Export Hardware] ダイアログ ボックスで定義したハードウェア仕様ファイルのディレクトリを指定します。
    ヒント: これは、通常ローカルのプロジェクト ディレクトリ構造に書き込まれますが、別のディレクトリに書き込まれている可能性もあります。
  • [Workspace] : 開く SDK プロジェクト ワークスペースを指定します。指定したディレクトリが存在しない場合は、それが作成されます。
