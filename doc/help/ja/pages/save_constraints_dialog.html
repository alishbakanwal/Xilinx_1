[Save Constraints] ダイアログ ボックス

このダイアログ ボックスでは、新しい制約を書き込むターゲット制約ファイルを定義できます。このダイアログ ボックスが表示されるのは、現在の制約セットで定義されるターゲット制約ファイルがない場合のみです。

制約は読み出され、リストされるファイルの順にデザインに適用されます。リストの最初で定義される制約は、後で定義される制約で上書きされる可能性があります。制約の優先順位の詳細は、『Vivado Design Suite ユーザー ガイド : 制約の使用』 (UG903) のこのセクションを参照してください。
重要: 新しい制約のみが指定したファイルに保存されます。既存のデザイン ファイルは、元の制約ファイルに保存されます。制約の入力および保存の詳細は、『Vivado® Design Suite ユーザー ガイド : 制約の使用』 (UG903) のこのセクションを参照してください。
  • [Create a new file] : 新しいターゲット制約ファイルの作成を指定して、現在の制約セットに追加されるようにします。
    • [File Type] : 現時点では、XDC ファイルのみがサポートされています。
    • [File Name] : 制約ファイルの名前を指定します。
    • [File Location] : 新しい制約ファイルのディレクトリを指定します。新しい制約ファイルは、ローカル プロジェクト ディレクトリまたはそれ以外のディレクトリで作成できます。
  • [Select an existing file] : 現在のデザインに関連付けられた制約ファイルで定義されている制約ファイルがドロップダウン メニューのリストに表示されます。
