[Older Project Version] ダイアログ ボックス

このダイアログ ボックスは、以前のバージョンの Vivado® Design Suite で作成または最後にアップデートしたプロジェクトを開くと表示されます。内部データ構造の変更により、プロジェクトは最新バージョンにアップデートする必要があります。 または、表示目的にプロジェクトを読み取り専用モードで開く必要があります。

ヒント: 最新バージョンにアップデートする前に、元のプロジェクトをアーカイブするか、プロジェクトのコピーを作成しておきます。ザイリンクスでは、互換性を持たせるために古いプロジェクトを使用できるよう、古いリリースの Vivado Design Suite のプロジェクトをアーカイブしておくこともお勧めしています。
  • [Automatically upgrade to the current version] : 内部プロジェクト データ構造が最新の Vivado Design Suite バージョンにアップグレードされます。プロジェクトをアップグレードした後、IP またはデザインで使用されるブロック デザインも別々にアップグレードしておく必要があります。詳細は、『Vivado Design Suite ユーザー ガイド : IP を使用した設計』 (UG896) のこのセクションまたは『Vivado Design Suite ユーザー ガイド : IP インテグレーターを使用した IP サブシステムの設計』 (UG994) のこのセクションを参照してください。
    重要: このプロセスはやり直すことができません。
  • [Open project in read-only mode] : 内部プロジェクト データはアップグレードされませんが、プロジェクトを開いて、表示またはプロジェクトのコピーを保存できます。
