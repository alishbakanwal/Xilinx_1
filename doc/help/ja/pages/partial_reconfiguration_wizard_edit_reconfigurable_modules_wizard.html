[Edit Reconfigurable Modules] ページ

Setup Partial Reconfiguration (PR) ウィザードのこのページでは、PR プロジェクトのリコンフィギャラブル モジュール (RM) を作成、編集、削除できます。パーティション定義 (partitionDef) はまず [Create Partition Definition] ダイアログ ボックスまたは Tcl コマンドの create_partition_def を使用して作成する必要があります。

このページには、現在のプロジェクトの既存のパーティション定義に関連付けられたリコンフィギャラブル モジュールが表で表示されます。この表には、現在のプロジェクトに含まれる既存の RM とそれに関連付けられた partitionDefs が表示されます。また、RM を管理するためのツールバーも含まれます。

  • [Add] () : 現在のプロジェクトに新しい RM が追加されます。クリックすると、[Add Reconfigurable Module] ダイアログ ボックスが開きます。新しく追加した RM が表に表示されます。
  • [Remove] () : 現在のプロジェクトから選択した RM が削除されます。
  • [Edit] () : 選択した RM の属性を編集できます。クリックすると、[Edit Reconfigurable Module] ダイアログ ボックスが開きます。
[Next] をクリックして Partial Reconfiguration ウィザードの [Edit Configurations] ページに進みます。
注: Partial Reconfiguration ウィザードのコマンドは、各ページの後には実行されませんが、すべてのページを確認して設定全体が終了するまで保存されます。[Skip to Finish] をクリックして [Partial Reconfiguration Summary] ページに進むと、すべてのページを確認しなくても設定を終了できます。
