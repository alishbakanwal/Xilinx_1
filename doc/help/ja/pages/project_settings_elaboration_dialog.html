[Project Settings] ダイアログ ボックス : [Elaboration] ページ

[Project Settings] ダイアログ ボックスでは、デザイン フローを通して使用されるさまざまな設定に対して値を指定できます。設定ページは、合成、シミュレーション、インプリメンテーションなどのデザイン フローの手順別に分類されています。

RTL デザイン (アウト オブ コンテキスト (OOC) の階層モジュール、IP、およびブロック デザインを含む) の場合、[Elaboration] ページで、早く開くことができる小さいデザインか、詳細にエラボレートされたデザインを選択できます。

  • [Link IP Module Options] フィールド
    • [Blackbox model] : デザインに OOC デザイン モジュール、IP コア、ブロック デザインが含まれる場合に、階層的な詳細を追加しないで、エラボレートされたデザインの上位部分を表示するために使用します。
      ヒント: トップダウン フローやチーム ベースのデザイン フローで生成されて使用可能な IP がない場合は、このモデルを使用します。
    • [Netlist model] : アウト オブ コンテキストの IP やモジュールの合成済みネットリストを含め、デザイン階層のエラボレーションすべてを表示するには、このモデルを使用します。
  • [Load constraints] : エラボレートされたデザインに制約を読み込んでプレビューします。
    重要: すべてのデザイン制約がエラボレートされたデザインで有効なわけではありません。無効な制約は無視されます。
