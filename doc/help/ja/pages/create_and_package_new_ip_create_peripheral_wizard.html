[Create Peripheral] ページ

[Create Peripheral] ページには、[Finish] をクリックしたときに実行される処理のサマリが示されるほか、次に実行するタスクを選択できます。

  • [Add IP to the repository] : AXI4 ペリフェラル IP を作成し、IP ディレクトリをユーザー リポジトリとして Vivado IP カタログに追加します。
  • [Edit IP] : AXI4 ペリフェラル IP を作成して Vivado® IP カタログに追加し、Edit IP プロジェクトで IP を開いてパッケージ IP をさらに編集する場合にオンにします。Edit IP プロジェクトの名前には、新しく作成したペリフェラル IP の名前が含められ、プロジェクトは指定の IP ディレクトリに作成されます。
  • [Verify peripheral IP using AXI4 BFM Simulation interface] : AXI4 ペリフェラル IP を作成し、Vivado IP カタログに追加します。Vivado ツールではこのペリフェラル IP を使用して Vivado IP インテグレーター ブロック デザインが作成され、AXI4 BFM シミュレーション インターフェイスを使用して Vivado シミュレータが起動されます。
    重要: AXI4 BFM シミュレーションにはライセンスが必要です。詳細は、『Vivado Design Suite ユーザー ガイド : ロジック シミュレーション』 (UG900) のこのセクションを参照してください。
  • [Verify peripheral IP using JTAG interface] : AXI4 ペリフェラル IP を作成し、Vivado IP カタログに追加します。Vivado ツールでは、Vivado IP インテグレーター ブロック デザインが作成され、JTAG-to-AXI Master デバッグ コアを使用してハードウェアでペリフェラル IP をデバッグできるようになります。JTAG-to-AXI デバッグ コアの使用に関する詳細は、『Vivado Design Suite ユーザー ガイド : IP を使用した設計』 (UG908) のこのセクションを参照してください。

[Finish] をクリックしてペリフェラル IP を作成し、選択したオプションに進みます。
