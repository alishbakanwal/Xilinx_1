[Device] ウィンドウ

[Device] ウィンドウには、ターゲット デバイスのダイがグラフィカルに表示されます。[Device] ウィンドウには、I/O バッファー、ブロック RAM、DSP、コンフィギャラブル ロジック ブロック (CLB)、クロック バッファー、クロック領域などのデバイス リソースが表示されます。特定のデバイス リソースを拡大したり、ネットリストのインタラクティブな自動配置を実行したり、配線を試したり、インタラクティブな配線を実行したりできます。

インタラクティブな配置には、[Netlist] または [I/O Ports] ウィンドウでオブジェクトを選択し、[Device] ウィンドウのデバイス リソースにそれらをドラッグ アンド ドロップします。ツールバーにはオブジェクトの配置を制御するための次のコマンドが含まれます。

Cell Drag and Drop Mode セルがドロップされた Pblock 領域、LOC プロパティの付いた SITE リソース、または BEL プロパティの付いた BEL リソースに割り当てられるかを制御できます。詳細は、『Vivado Design Suite ユーザー ガイド : インプリメンテーション』 (UG904) のこのセクションを参照してください。
Draw Pblock セルを Pblock に配置するには、まずこのこのコマンドを使用して 1 つを定義する必要があります。Pblock を使用すると、デザインの特定のエレメントを配置するデバイスの領域を定義できます。この手法は「フロアプラン」と呼ばれ、使用すると、課題の多いデザインでタイミング クロージャを達成しやすくなります。詳細は、『Vivado Design Suite ユーザー ガイド : デザイン解析およびクロージャ テクニック』 (UG906) のこのセクションを参照してください。
[Place Port Modes] : 選択したポートを I/O バンクに配置するか、エリアに配置するか、シーケンシャルに配置するかを定義できます。I/O ポートの配置に関する詳細は、『Vivado Design Suite ユーザー ガイド : I/O およびクロック プランニング』 (UG899) のこのセクションを参照してください。
Autocheck I/O Placement インタラクティブな配置中に I/O 配置のデザイン ルール チェックを自動的に実行して、エラーが発生しないようにできます。
Routing Resources デザインのネットリストの配線に使用するさまざまな配線リソースを含め、デバイスがかなり詳細に表示されます。このビューでは、ネットを選択し、デバイス全体の配線をトレースできます。デザインのクリティカル ネットのインタラクティブな配線を実行することもできます。詳細は、『Vivado Design Suite ユーザー ガイド : インプリメンテーション』 (UG904) のこのセクションを参照してください。

拡大および縮小して、特定デバイス リソースを確認するのに必要な詳細レベルで表示できます。ツールバー メニューには、何を表示するかを制御するための次のコマンドが含まれます。

Previous 拡大レベルおよび表示位置など、[Device] ウィンドウの前の表示を復元します。
Next [Previous] で表示していた前の表示から元の表示に戻します。
ヒント: このコマンドは、[Previous] コマンドを使用した後にのみ使用可能です。
Zoom In デバイス リソースの詳細が拡大されて表示されます。
Zoom Out デバイスのエリアが縮小され、より多く表示されます。
Zoom Fit デバイス全体が表示されます。
Select Area [Device] ウィンドウで長方形を描画してオブジェクトを選択します。
Fit Selection 現在選択しているオブジェクトが [Device] ウィンドウの中央に拡大されて表示されます。
Autofit Selection 別のウィンドウで選択したオブジェクトが [Device] ウィンドウの中央に自動的に拡大されます。
[Device] ウィンドウでは、クリックによる表示の拡大/縮小機能がサポートされています。クリックしてドラッグしながら線やボックスを描くと、デバイスを拡大または縮小表示できます。[Device] ウィンドウの右側または下側のスクロールバーを使用すると、画面をスクロールして別の新しいエリアを表示できます。
ヒント: スクロールバーは、表示エリアがデバイスで拡大された場合にのみ表示されます。

[Device] ウィンドウは Vivado® IDE のワークスペースに表示されます。ワークスペースの詳細は、『Vivado Design Suite ユーザー ガイド : Vivado IDE の使用』 (UG893) のこのセクションを参照してください。
