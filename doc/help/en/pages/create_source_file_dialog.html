Create Source File

This dialog box lets you define a new design source file, such as a Verilog or VHDL file, to add to the current project.

  • File Type: Specifies the file format, including VHDL (.vhd), Verilog (.v), Verilog header (.vh), SystemVerilog (.sv), and memory file (.mem).
  • File Name: Specifies the name of the design source file.
  • File Location: Specifies the location for the new source file whether in the local project directory or in a location outside of the current project.
When you click OK, a placeholder for the new file is added to the list of sources displayed in the Sources window. The source file is not created until you click Finish in the Add Sources wizard.
Tip: After you click Finish, the Define Module dialog box opens to let you define the module for Verilog, SystemVerilog, and VHDL files. You must manually create Verilog header and memory files.
