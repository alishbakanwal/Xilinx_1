Create Block Design Port

The Create Block Design Port dialog box lets you define a new external port to add to the current block design.

  • Port name: Specifies the name of the port to create.
  • Direction: Defines the port as an input, output, or bidirectional port.
  • Type: Specifies the type of signal carried by the port. The type can be defined as clock, reset, interrupt, data, clock enable, or other.
  • Create vector: Specifies the port as a bus port and defines the width of the port.
  • Frequency (MHz): This option is available when an input is configured as a clock port. It specifies the frequency of the clock port in MegaHertz.
  • Polarity: This option is available when an input port is configured as a reset. A reset port can be specified as an active-high or active-low reset.
  • Interrupt type: This option is available when an input port is configured as an interrupt. This defines an interrupt port as an edge or level type of interrupt.
  • Sensitivity: This option is available when an input port is configured as an interrupt. This defines the sensitivity of a level-type interrupt as either high or low; or defines the sensitivity as rising edge or falling edge for an edge-type interrupt.
  • Connect to matching selected ports: Connects the new port to the selected ports or pins that are compatible with the new port.
    Tip: This checkbox is available when other pins or ports in the design are selected when you run the Create Port command.

Click OK to create the new port and optionally, connect it to selected ports.
