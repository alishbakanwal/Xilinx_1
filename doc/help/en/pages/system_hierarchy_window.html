Block Design Hierarchy Window

The Vivado® IP integrator Design Hierarchy window displays a tree view of the block design hierarchy with Interface Connections and Nets sorted separately. The window shows block design cells for both hierarchical cells and IP added from the Vivado IP catalog, or from a custom IP repository..

The Design Hierarchy window toolbar includes the following commands:

Search Opens a search field to enter a search pattern.
Collapse all Collapses all object hierarchies.
Scroll Scrolls to the selected object in the Design Hierarchy window when an object is selected in another window, such as the Design Canvas window.
Diagram Displays the selected object in the top-level Design Canvas window or opens a new Design Canvas to display the contents of a hierarchical cell if one is selected.
