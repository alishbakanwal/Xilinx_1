Create Pblocks - Select Cells

This page of the Create Pblocks wizard lets you specify the hierarchical cells in the current design to associate with Pblocks.

  • Add (): Opens the Select Cell dialog to let you select one or more cells in the design. A separate Pblock will be created for each cell selected. You can assign additional cells to a Pblock after it has been created.
    Tip: The Select Cell dialog box lists just the cells from the Netlist window for the current design. You can also select cells directly in the Netlist window and use the Floorplanning > New Pblock command to create a new Pblock, or the Assign to Pblock to add it to an existing Pblock.
  • Remove (): Removes selected cells from the list of cells to associate with Pblocks.

Click Next to proceed to the Format Names page of the Create Pblocks wizard.
