Add or Create Simulation Sources

The Add or Create Simulation Sources page lets you specify an existing simulation fileset or create a new simulation fileset to add simulation sources to, such as a test bench for the design or a module of the design. Supported simulation source files include Verilog, Verilog header, SystemVerilog, VHDL, and memory file formats.

  • Specify simulation set: The currently active simulation fileset is the default fileset to add simulation files to. However, you can also select another existing sim_set or create a new simulation fileset for the project. Selecting Create Simulation Set will open the Create Simulation Set dialog box letting you specify the name for the new simulation fileset.
    • Make Active: When a new simulation fileset is created, you have the option of selecting this checkbox to make it the active sim_set for the project.
  • Add Files: Opens a file browser letting you select one or more simulation source files. Selecting multiple files will add them in sorted order to the Add or Create Simulation Sources page.
    Tip: Alternatively, you can click and select Add Files.
  • Add Directories: Opens a file browser letting you select one or more directories to search for simulation source files. Specified directories will be searched and any simulation source files found will be added to the current project.
    Tip: Alternatively, you can click and select Add Directories.
  • Create File: Opens the Create Source File dialog box in which you can specify the file type, file name, and file location to write the file to. An empty source file will be created and added to the project. When the Add Sources wizard is finished, a module editor will open for the newly created source files. See the Define Module dialog box for more information, or refer to this link in the Vivado® Design Suite User Guide: System-Level Design Entry (UG895).
    Tip: Alternatively, you can click and select Create Files.
  • Remove (): Removes selected files from the current project. The files are removed from the project but are not removed from the hard drive.
  • Move Up: When one or more files have been selected in the file list display area, the blue Up arrow button () is activated, letting you move the selected file or files up in the listed order. Simulation source files are read into the design and evaluated in the order listed. Refer to this link in the Vivado Design Suite User Guide: Using the Vivado IDE (UG893) for more information on compile order and the Hierarchy Update command.
  • Move Down: When one or more files have been selected in the file list display area, the blue Down arrow button () is activated on the left hand menu, letting you move the selected file or files down in the listed order. Simulation source files are read into the design and evaluated in the order listed.
  • Scan and add RTL include files into project: This checkbox scans the added RTL files and adds any referenced Verilog 'include files into the local project directory structure as well.
  • Copy sources into project: This checkbox lets you copy the specified design source files under the sources folder of the local project directory. Refer to this link in Vivado® Design Suite User Guide: System-Level Design Entry (UG895) for more information on using local and remote sources.
  • Add sources from subdirectories: This checkbox adds source files from the subdirectories under the directories specified using the Add Directories option.
  • Include all design sources for simulation: This checkbox enables the inclusion of all design source files into the simulation fileset to support exporting the design for use with third-party simulators.
