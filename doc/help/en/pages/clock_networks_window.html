Clock Networks Window

The Clock Networks window displays the results of the Report Clock Networks command. Each clock tree shows the clock network from source to endpoint with the endpoints sorted by type. You can traverse and explore the whole clock tree using Filter commands from the toolbar, displaying and hiding objects on the clock tree such as the input port, cells and input pins, clock and non-clock endpoints.
Tip: The full clock tree is only available in the Clock Networks window. The text version of the Report Clock Networks command shows only the name of the clock roots.

The Clock Networks window toolbar includes the following commands:

Collapse Collapses all clock trees displayed in the Clock Networks window.
Expand Selected Clock Trees Expands the selected clock trees.
Schematic Opens a schematic view containing the selected clock.
Filter Ports Shows input ports for clocks.
Filter Net Shows the clock nets between loads on the clock tree.
Filter Cell Shows netlist cell objects on the clock tree, such as MMCMs and clock buffers.
Filter Input Pins Shows the input pins on cells.
Filter Output Pins Shows the output pins on cells.
Filter Clock Endpoints Shows the clock path endpoints on the clock tree.
Filter Non-Clock Endpoints Shows non-clock path endpoints on the clock tree.
Rerun Regenerates the Clock Network report.
