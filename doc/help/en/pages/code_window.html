Text Editor

The Text Editor provides an editing environment for HDL source files, XDC constraint files, Tcl scripts, and other text based files. The Text Editor recognizes the language syntax of specific files format (Verilog, SystemVerilog, VHDL, SDC, XDC, and Tcl) and highlights keywords and identifiers of the language while editing the file. For recognized file formats, it also offers real-time syntax checking, and help identifying and fixing errors.
Tip: In cases where the language syntax is not recognized, the Text Editor treats the file as a simple text file.

The Text Editor offers features in the right-click menu to let you configure how tabs and white spaces are managed, and offers code folding for hierarchical language syntax, where Verilog modules, VHDL entities, and Tcl procs can be collapsed and expanded as needed. For files that have been modified, you can compare the file in memory to the one saved on disk, to see what changes have been prior to saving the file.

The Text Editor toolbar offers some traditional commands for editing a file, such as Copy (), Cut (), Paste (), and Delete (). The Save () command lets you save changes to an edited text file, while Undo () and Redo () let you back out or restore changes when needed.

In addition, the Text Editor toolbar has commands for editing code files, such as Verilog, VHDL, Tcl, and XDC files:

Toggle Line Comments Quickly change a line of code into a comment, or a single line comment into a line of text. The Text Editor will insert or remove the appropriate character string to create or remove a single line comment, based on the language syntax of the file being edited:
  • // for Verilog and SystemVerilog
  • -- for VHDL
  • # for Tcl and XDC
Toggle Column Selection Mode Enable selecting characters in a grid based on columns and rows rather than selecting characters per line.
Find Display the Find field at the bottom of the Text Editor to enable searching through the current text file.
Language Templates Open the Language Template dialog box to let you browse and select from a collection of predefined Verilog, SystemVerilog, VHDL, and XDC design templates.
Insert Template Add the currently selected template from the Language Template dialog box into the text file at the location of the cursor.
Note: The Language Template dialog box does not need to be open to insert the selected template.

The Text Editor appears in the Vivado® IDE workspace. For more information on workspace windows refer to this link in the Vivado Design Suite User Guide: Using the Vivado IDE (UG893).
