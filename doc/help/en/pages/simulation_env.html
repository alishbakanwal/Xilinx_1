Simulation

The Simulation environment is opened when Run Simulation is selected from the Flow Navigator, or the launch_simulation command is run from the Tcl Console. To create the Simulation environment the Vivado tool compiles the RTL source files and test bench for the design and launches the simulation snapshot. Refer to this link in the Vivado Design Suite User Guide: Logic Simulation (UG900) for more information.

When a Simulation is already open, you can activate the environment by clicking Simulation in the Flow Navigator.

From the Simulation environment you can:

  • Run behavioral, functional or timing simulation.
  • Run simulation for a defined period or through all events.
  • Use step and breakpoint commands to perform detailed debug of the design.
  • Force HDL objects to have specific values and defined states.
  • Quickly recompile and relaunch simulation to incorporate design updates.
  • Dump VCD and SAIF files for external use.
  • Save waveform configurations and simulation results to review at another time.
