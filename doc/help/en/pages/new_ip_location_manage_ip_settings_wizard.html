Manage IP Settings

  • Part: Select a target part or board for the Manage IP project. Enter the part or board name in the entry field directly, or click the browser button to open the Select Device dialog box to select the part or board.
  • Target Language: Select the target language used by the RTL project. IP added to the design is added in this target language if the language is supported.
  • Target Simulator: Select a simulator from the list, such as the Vivado® simulator or one of the supported third-party simulators.
  • Simulator Language: Select the HDL language supported by the simulator for use in validating the design. This option specifies the language to use for generating IP output products for simulation.
  • IP Location: Specify the location to create the Manage IP project. Both the Manage IP project and the IP customizations being managed are written to the specified location.
    Tip: The Vivado Design Suite creates the specified directory if it does not already exist.

Click Finish to create the Manage IP project.
