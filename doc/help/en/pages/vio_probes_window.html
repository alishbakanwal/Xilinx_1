VIO Probes

The VIO Probes window displays a list of probes that you have selected from a VIO debug core programmed into the current hardware device. With VIO probes added, you can use the window to monitor the activity on the input probe signals, or to force the output probe signals to a specified state.

The VIO core input probes are used to read values from a design that is running in the current hardware device. The VIO input probes are typically used as status indicators for a design-under-test. The value of the VIO input probes are periodically updated based on the refresh rate of the VIO core, which you can change by editing the Refresh Rate (ms) in the VIO Properties window.

The VIO core output probes are used to write values to a design that is running in the current hardware device. The VIO output probes are typically used as low-bandwidth control signals for a design-under-test. The output values are updated whenever a new value is entered into the VIO Probes window.

The toolbar for the VIO Probes window includes the following commands:

Search Opens a text field to enter a search pattern to display specific probes of interest.
Collapse all Collapses all bus probes to hide the individual bits on the probe.
Expand all Expands all bus probes to display the individual bits on the probe.
Add probes Opens the Add Probes dialog box, which lists the probes associated with the VIO debug core. Select probes and click OK to add them to the window.
Remove selected probes Removes selected probes from the VIO Probes window.
