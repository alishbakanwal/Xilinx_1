Specify Path Groups

The Specify Path Groups dialog box lets you select from currently defined path groups for timing analysis.

  • Pattern: Specifies search patterns to locate currently defined path groups. You can use the question mark (?) and asterisk (*) characters as wildcards.
  • Regular Expressions: Defines the Pattern field as a regular expression (regexp) search pattern. See this link for more information on regular expression syntax.
  • Ignore case: Applies to regular expressions searches only. Search expressions are generally case-sensitive.
  • Find: Runs the search defined in the Pattern field. The results of the search are populated into the Found field of the Results section.
  • Results:
    • Found: Shows the number of objects found by the Pattern and lists the objects.
    • The following toolbar buttons let you move objects between the Found and Selected fields:
      Move the selected items to the right Moves selected objects from Found to Selected.
      Move the selected items to the left Moves selected objects from Selected to Found.
      Move all the items to the right Moves all objects from Found to Selected.
      Move all the items to the left Moves all objects from Selected to found.
    • Selected: Shows the number of objects that are currently selected from the found items and lists the objects. The selected items define the start points of the timing path.
    • The following toolbar buttons let you sort the selected objects:
      Sort Sorts the list of selected objects alpha-numerically. Click once to sort in ascending order, and click a second time to sort in descending order.
      Move the selected items up Moves the currently selected objects up in the ordered list.
      Move the selected items down Moves the currently selected objects down in the ordered list.
      Move the selected items to the top Moves the currently selected objects to the top of the ordered list.
      Move the selected items to the bottom Moves the currently selected objects to the bottom of the ordered list.
  • Command: Displays the Tcl commands that get the specified path group objects. You can use the displayed command to help create Tcl scripts using the Vivado® IDE.
  • Enumerate: Enumerates the selected path group objects individually in the Tcl command, eliminating any wildcards used.

Click OK to add the Tcl command to the Report Timing dialog box.

Click Append to append the Tcl command to an existing start point command.
