Power Results

The Power Results window displays the results of the Report Power command. The window is divided into two sections. The left side has a tree view showing Settings, a Summary power report,details of the Power Supply, and Utilization Details for the device. The right side reports the specific details of the section selected on the left.

Settings and Power Supply provide a view of the various options and command settings used when the Report Power command was run. This lets you see the conditions used for analysis, and repeat the analysis if necessary.

The Summary view provides the total on-chip power and thermal properties. The On-Chip Power graph shows the power dissipated in each of the device resource types. With this high-level view you can determine which parts of your design contribute to power consumption.

The Utilization Details provides a device resource tree and specific utilization and power consumption of those resources. Use this detailed view to examine specific parts of your design.

The Power Results toolbar includes the following commands:

Search Opens a search field to enter a search pattern.
Collapse all Collapses the hierarchy of the Utilization Details displayed in the tree view.
Expand all Expands the hierarchy of Utilization Details displayed in the tree view.
Previous Goes back to select a previously selected power results. Use this command to quickly scroll through the power results that you selected to investigate.
Next Goes forward to select a previously selected timing check. Use this command to quickly scroll through the power results that you selected to investigate.
Save Saves the power report in an RPX file, which you can reload later to reevaluate the timing without rerunning the power analysis.
