Unplace Cells

This page of the Clear Placement wizard displays information related to the selected and unselected primitive cell objects in the design.
Tip: This page is only displayed if you have cells selected before running the Clear Placement command.
  • Cells to Unplace:
    • Unplace <#> selected cells: Directs the Vivado tool to clear the placement constraints for the currently selected primitive cells, where <#> is the count of currently selected cells.
    • Unplace all except for <#> selected cells: Directs the tool to clear the placement constraints for all of the primitive cells in the design, except the currently selected cells.
    • Unplace all cells: Directs the tool to clear the placement for all primitive cells in the design.
    • Do not unplace any cells: Do not clear the placement of any cell objects, selected or unselected.

Click Next to display the Cell Types to Unplace page.
