ILA Core Options

This page lets you configure properties of the ILA cores inserted by the Set Up Debug wizard:

  • Sample of data depth: Specify the number of clock cycle data samples to capture for the probes in the design. The sample depth and probe width combine to determine the device resources used by the ILA core.
  • Input pipe stages: Specify the number of pipeline registers to use in the ILA inputs to manage timing. Pipeline registers can be used to move the ILA core out of the critical timing path. This also adds to the resources used by the ILA core.
  • Capture control: Specifies the conditions that must be met for the ILA core to capture data. Refer to this link in the Vivado Design Suite User Guide: Programming and Debugging (UG908) for more information on configuring capture modes.
  • Advanced trigger: Specifies that advanced trigger mechanisms will be specified for the ILA core in the Vivado hardware manager. Refer to this link in the Vivado Design Suite User Guide: Programming and Debugging (UG908) for more information.
Important: These ILA configuration options are applied to all the ILA cores created by the Set Up Debug wizard.
