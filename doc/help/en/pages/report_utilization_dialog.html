Report Utilization

This dialog box lets you generate a report of the device resource utilization from the synthesized or implemented design netlist.

  • Results name: Specifies the name to apply to the report when opening the results in the Vivado® IDE.
  • Export to file: Specifies the path and name of the Utilization report file to create. You can use the file browser button () to specify the location and name of the file to create.
  • Open in a new tab: Opens the Utilization report in a new tab of the window in the Vivado IDE. If this checkbox is not enabled, the new report overwrites the last report that was opened.
