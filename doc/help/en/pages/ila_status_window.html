ILA Status

This window is one of five windows displayed in the Hardware ILA dashboard by default. The ILA Status window provides a real-time visual summary of the state of the ILA debug core. The window graphically displays Core status and Capture status so you can watch as the core goes through a trigger event and data capture cycle.

The toolbar of the ILA Status window includes the following commands:

Toggle Auto-Retrigger Enable or disable the automatic re-trigger to re-arm the ILA core trigger after a successful trigger, capture, display operation has completed.
Run Trigger Arm the selected ILA core to detect trigger events that are defined by the core's basic or advanced trigger settings.
Run Immediate Arm the selected ILA core to trigger immediately regardless of the trigger settings.
Stop Trigger Stop the trigger on the selected ILA core.
Compile Trigger State Machine Compile the trigger state machine without running or arming the ILA trigger.
Tip: The trigger state machine is compiled automatically when the trigger is run.
