Scope Window

The Scope window shows the design hierarchy of the current simulation. A simulation scope is a hierarchical partition of an HDL design, such as an entity or module, a block, or a package, or a subprogram such as a process or task. Selecting a scope in the Scope window displays all of the HDL objects contained within that scope in the Objects window, letting you quickly navigate the design.

The right-click menu of the Scope window lets you manage the various scopes of the simulation, by adding the objects inside a selected scope to the Waveform window, or opening the source file that defines or instantiates a selected scope. The Scope window toolbar provides commands to filter and manage the contents displayed in the Scope window.
Important: When you hide a scope using a filter button, all of the scopes inside that scope are also hidden regardless of type.

The toolbar includes the following commands:

Search Opens a search field to enter a string search pattern.
Collapse all Collapses all scope hierarchies.
Expand all Expands all scope hierarchies.
VHDL Entity Enable or disable to display or hide the VHDL entity scopes.
VHDL Package Display or hide the VHDL package scopes.
VHDL Block Display or hide the VHDL block scopes.
VHDL Process Display or hide the VHDL process scopes.
Verilog Module Display or hide the Verilog module scopes.
Verilog Package Display or hide the Verilog package scopes.
Verilog Block Display or hide the Verilog block scopes.
Verilog Task Display or hide the Verilog task scopes.
Verilog Process Display or hide the Verilog process scopes.
