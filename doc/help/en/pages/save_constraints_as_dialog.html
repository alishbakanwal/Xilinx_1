Save Constraints As

This dialog box lets you save the active constraint set of the current design to a new name, creating a copy of the constraint set.

  • New Constraint set name: Specifies the constraint set to create. All constraints files associated with the constraint set are also copied to new files with the same name, but located under the new constraint set.
  • Make Active: Makes the new constraint set the active constraint set for the project. The active constraint set provides all constraints applied to the current design, and is the target for saving new constraints.
