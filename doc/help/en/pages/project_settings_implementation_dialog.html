Project Settings - Implementation

The Project Settings dialog box lets you specify values for various settings used throughout the design flow. The settings are grouped according to the steps in the design flow, such as synthesis, simulation, and implementation.

The Implementation settings let you define the constraint set and strategy as well as customize the options for the active implementation run. The default options for the selected strategy are displayed in a table, and you can modify options or values to customize the strategy.

  • Default constraint set: Specifies the constraint set to use during implementation. You can create a new constraint set using the Add Sources wizard.
  • Incremental Compile: Specifies a design checkpoint (DCP) to use as a reference for the next implementation run. The reference design is usually an earlier iteration or variation of the current design that has been synthesized, placed, and routed. However, you can also use a checkpoint that is only partially placed or routed. See this link in the Vivado Design Suite User Guide: Implementation (UG904) for more information.
  • Strategy: Specifies the predefined Vivado® Design Suite strategies, or lets you create your own strategy by modifying the options for the selected strategy.
    Tip: When you change the options for a selected strategy, as indicated by an asterisk (*) in the strategy name, you can click the Save Strategy As button () to save those options as a new user-defined strategy.
  • Description: Provides a brief summary of the goal of the selected strategy. You can define the description when saving user-defined strategies.
  • Options: Shows a table view of the options available in the selected implementation strategy. Select an option in the table to display a description of the option.
    Tip: The Implementation Strategy is broken down into the various steps that make up the optimization, placement, and routing process. See this link in the Vivado Design Suite User Guide: Implementation (UG904) for more information.
