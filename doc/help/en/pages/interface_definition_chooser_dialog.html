Interface Definition Chooser

This dialog box lets you choose an interface from the currently defined interfaces available in the Vivado® Design Suite. The interface definitions are listed by name and description. You can use the Group by Interface Type command () to sort the interfaces by type or list them alphabetically by name.

Selecting an interface populates the Interface Logical Port list with the signal ports included in the interface definition. This lets you review the signals supported by the interface.

Click OK to choose the currently selected interface definition.
Tip: If multiple interfaces are selected, the first in the list will be the chosen interface definition.
