Implemented Design

The Implemented Design environment links the post-synthesis netlist files, the placement and routing data from implementation, and the device resources of the target part to create the in-memory implemented design.

When an Implemented Design is already open, you can activate the design environment by clicking Implementation in the Flow Navigator.

From the Implemented Design environment you can:

  • Run design rule checks and methodology checks to highlight and eliminate errors.
  • Generate the most accurate reports, including timing summary, design and power analysis, on the fully placed and routed design.
  • Perform functional and timing simulation with actual delays from the placed and routed design.
  • Generate a bitstream to program the target device.
