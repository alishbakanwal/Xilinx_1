Primary Clock Constraints

This dialog box lets you define the attributes of the currently selected primary clock or clocks. The attributes of the clock are associated with options of the create_clock command.

  • Frequency: Specifies the frequency of operation for the clock in megahertz.
  • Period: Specifies the clock period in nanoseconds.
    Tip: Either the Frequency or Period is required. The unspecified attribute is automatically derived from the specified attribute.
  • Rise at: Specifies the time of the rising edge of the clock to define the waveform.
  • Fall at: Specifies the time of the falling edge of the clock to define the waveform.
    Tip: The waveform of the clock is automatically defined as 50% of the period. Use the Rise at/Fall at values to change the default waveform.
  • Jitter: Optionally specifies the input jitter associated with the selected primary clock. The jitter value is associated with the clock through the set_input_jitter command.

Click OK to add the attribute of the clock to the currently selected clock or clocks in the Timing Constraints wizard.
