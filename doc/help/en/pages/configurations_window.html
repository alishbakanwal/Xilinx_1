Configurations Window

The Configurations window is displayed when the Partial Reconfiguration (PR) flow is enabled for the project using the Tools > Enable Partial Reconfiguration command, or by setting the PR_FLOW property to TRUE. The window lists the PR configurations by name, and displays the Partition Definitions (partitionDefs) in the current project. For each configuration you can you specify a Reconfigurable Module (RM) to assign to each partitionDef. In the right-click menu you can specify the active configuration, which is the default configuration for implementation.

The Configurations window also has a toolbar with the following commands:

Search Opens a search field to enter a search pattern.
Modify Configuration Lets you edit the selected PR configuration by opening the Edit Configurations page of the Partial Reconfiguration wizard.
