Place Pblocks

This dialog box lets the Vivado tool draw and place regions for selected Pblock. The tool analyzes the Pblock and sizes it based on the slice count and the target utilization rate you specify.

  • Parent block: Specify the top-level block or partition to evaluate. The default is the ROOT partition.
  • Pblocks to Place: Displays a list of Pblocks in the design in a table form, and lets you select which of the Pblocks to place.
    • Place: A checkbox to enable or disable a specific Pblock for placement.
    • Pblock: The name of the Pblocks in the design.
    • Utilization: The target utilization rate determines the size of the Pblock on the device. A high utilization rate will make the Pblock smaller as it consumes more of the device resources in the defined area. A low utilization rate will make the Pblock larger.
      Tip: You can start with a high utilization rate, even greater than 100%, for initial placement and to create a floorplan for the design, and lower the utilization rate for implementation.
  • Set Utilization on All Pblocks: Specify the target utilization rate for all Pblocks. This sets a general target that you can override using the Utilization for individual Pblocks.

Click OK to automatically draw and place regions in the Device window for the selected Pblocks. After the Pblocks have been placed, you can move them
