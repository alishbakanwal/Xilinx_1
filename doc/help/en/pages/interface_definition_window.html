Interface Definition Window

The Interface Definition window lets you define and edit a new interface definition. The top portion of the Interface Definition window displays the Vendor:Library:Name:Version (VLNV) information for the new interface definition. In addition, you can enter a Description and Display Name for the interface. The rest of the window is a table-style window that lists the defined ports included in the interface definition. You can display or hide columns of information related to the ports as needed.

The Interface Definition window toolbar includes the following commands:

Search Opens a search field to enter a search pattern to apply to the port table. Ports with fields matching the search pattern are displayed.
Add Interface Port Opens the Add Port dialog box to let you define a new port for the interface definition.
Remove Interface Port Removes the selected ports from the interface definition.
Tip: This command is only active when one or more ports are selected.
Copy Ports Opens the Interface Definition Chooser dialog box to let you select an interface to copy the port definitions from.
Important: This command overwrites all current port definitions with the port definitions of the copied interface. Use this option before adding or modifying any ports on your interface, or any changes will be lost.

The Interface Definition window has a Save button and a Discard button, which become active when any changes are made to the interface definition. Click Save to write the Interface Definition to the IP repository where it is stored, or click Discard to cancel any changes and reload the last saved version of the interface.
