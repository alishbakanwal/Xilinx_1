Sources Window

The Sources window provides multiple views of the design sources associated with the current project. The views include:

  • Hierarchy: Shows sources sorted from the top-level of the design down.
  • IP Sources: Shows IP source files and generated output for designs that include IP or block designs.
  • Libraries: Shows sources sorted by library.
  • Compile Order: Shows sources sorted into the order they will be compiled for synthesis, implementation, or simulation, based on the Used In selection menu that is located just below the toolbar.
    Tip: When the Partial Reconfiguration flow is enabled, the Sources window also displays a Partition Definitions tab. See the Vivado® Design Suite User Guide: Partial Reconfiguration (UG909) for more information in the PR flow.

The Sources window lets you manage the HDL design sources, IP and block design sources, constraints files and constraint sets, and simulation source files, such as test benches. The right-click menu in the Sources window provides access to commands that let you add or remove files from the current project, change libraries and compile orders, make specific filesets active, manage Partial Reconfiguration, and set targets for design constraints.

The Sources window toolbar includes the following commands:

Search Opens a search field to enter a search pattern.
Collapse all Collapses all source folder hierarchies.
Expand all Expands all source folder hierarchies.
Open Selected Opens the selected source files.
Tip: This command is only available when source files are selected.
Add Sources Opens the Add Sources wizard to let you add different types of source files to the design.
Show Only Missing Sources Available from the Hierarchy tab, this displays only the missing source files to let you see the problem more clearly.
Tip: This command is only available when there are missing source files in the project.
Sort Available from the Libraries tab, this sorts the listed folders and source files in increasing alphanumeric order.
Scroll Scrolls to the selected object in the Sources window when an object is selected in another window, such as the Device window.
