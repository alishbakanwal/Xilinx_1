Report Power Optimization

This dialog box lets you report the power optimizations run by the Vivado® Design Suite on the current design. Power optimization is an optional step that can be run during implementation using intelligent clock gating to reduce dynamic power in your design. It can be run after logic optimization or after placement to reduce power demand in the design. Power optimization does not change the clocks or logic of the design.

  • Results name: Specify the name to apply to the report in Power Opt Results window in the Vivado IDE.
  • Export to file: Write the results of the Power Optimization report into the specified file. You can specify the file format to export as either text or XML.
  • Open in a new tab: Open the Power Optimization report in a new tab of the Power Opt Results window in the Vivado IDE. If this checkbox is not enabled, the new report overwrites the last report that was opened.
