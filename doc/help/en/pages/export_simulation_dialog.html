Export Simulation

This dialog box lets you export simulation scripts and design files for the specified target simulator, such as Mentor Graphics ModelSim, Cadence Incisive Enterprise Simulator, and Synopsys VCS MX simulator. The generated script will contain simulator commands for compiling, elaborating and simulating the design.

  • Target simulator: Select the third-party simulator to export simulation files for.
    Tip: Third-party simulators are only listed if they are available for use on the operating system the Vivado tool is currently running on.
  • Compiled library location: Specify the path of the libraries compiled for use with your third-party simulator. The Xilinx libraries must be compiled for use with outside simulators using the Compile Simulation Libraries command to allow the device resources to be properly simulated. The simulation libraries contain the device and IP behavioral and timing models.
  • Export directory: Specify the directory to output the various design files and scripts to. If the directory already exists, you can update its contents by enabling the Overwrite files checkbox.
  • Use absolute paths: Specifies the use of absolute paths in the exported simulation scripts. By default, the design source file and include directory paths will be relative to the Export directory.
  • Copy source files to export directory: Copies the design source files along with the exported netlist and simulation scripts.
