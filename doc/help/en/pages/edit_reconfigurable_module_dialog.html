Edit Reconfigurable Module

This dialog box lets you edit an existing Reconfigurable Module (RM) by specifying the name, type, and top-level hierarchy of the module:

  • Partition Definition: Edit the partitionDef the RM is associated with.
    Tip: If there is only one partitionDef in the current project, this field will only display and not edit the partitionDef. You can use the Create Partition Definition dialog box to define a new partitionDef.
  • Reconfigurable Module: Edit the name of the RM.
  • Top Module Name: Specify the hierarchical cell associated with the RM.
  • Sources have been synthesized: Specify that the RM has a post-synthesis netlist as the design source, either a DCP or EDIF netlist.

This dialog box also displays a table of files associated with the RM, and lets you add and remove files to the module:

  • Add Files (): Opens a file browser letting you select one or more design source files to include in the module. Selecting multiple files will add the files in the specified order to the page.
  • Add Directories (): Opens a file browser letting you select one or more directories to search for design source files. Specified directories will be searched and any design source files found will be added to the module.
  • Remove (): Removes selected files from the module.
  • Scan and add RTL include files into project: This checkbox scans the added RTL files and adds any referenced Verilog 'include files into the module as well.
  • Copy sources into project: This checkbox lets you copy the specified design source files under the sources folder of the local project directory. Refer to this link in Vivado® Design Suite User Guide: System-Level Design Entry (UG895) for more information on using local and remote sources.
  • Add sources from subdirectories: This checkbox adds source files from the subdirectories under the directories specified using the Add Directories option.

Click OK to commit the changes to the Reconfigurable Module. Click Cancel to close the dialog box without making changes.
