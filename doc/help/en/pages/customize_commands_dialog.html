Customize Commands

The Customize Commands dialog box lets you add custom Tcl commands, or procedures, to the Vivado IDE with shortcut keystrokes to quickly access these commands. The added commands are available to use from the Tools > Customize Commands sub-menu.

The dialog box is split into two sections: the list of custom commands on the left, and the details of the command on the right.

  • Custom Commands: Displays the list of commands that will be added to the Vivado IDE when you click OK. The list includes custom commands that were previously added as well as any added at now.
    Add Enter a menu name for the new custom command. The new command is added to the bottom of the list when you press ENTER.
    Remove Removes the selected custom commands from the list.
    Move up Moves the selected commands up in the list.
    Tip: The order the commands are listed is the order they will appear in the Customize Commands menu in the Vivado IDE.
    Move down Moves the selected commands down in the list.
  • Edit Custom Commands: The right side of the dialog box lets you customize the details of the currently selected command.
    • Menu Name: Specifies the name of the command as it will appear in the Vivado IDE menu.
    • Shortcut: Specifies the keystroke shortcut for the current command.
      • Add: Opens the Add Shortcut dialog box to let you type the keystroke shortcut to use for the current command. Click OK to assign the shortcut to the current command,
        Important: If the keystroke combination typed is already assigned to a different command, the dialog box will report this under Shortcuts currently used by. You can change the shortcut, or click OK to assign the shortcut to the current command.
      • Remove: Removes the keyboard shortcut defined for the current command.
    • Run Command: Specifies the name of the Tcl proc to run. The proc must be pre-loaded into the Vivado Design Suite by sourcing the Tcl file. See this link in Vivado Design Suite User Guide: Using Tcl Scripting (UG894) for more information on loading and running Tcl scripts.
      • Define Args: Opens the Define Command Arguments dialog box. Sometimes a Tcl proc requires or optionally accepts arguments when the command is run. Define default values for those arguments, or define names for the arguments to prompt the user when the custom command is run.
    • Source Tcl file: As an alternative to specifying the Run Command, specify a Tcl file to source at the time the command is run. In this case you do not need to define a Tcl proc as the script is run at the time the file is sourced.
    • Toolbar Options: Configure the settings for optionally displaying the Tcl proc on the main toolbar of the Vivado IDE:
      • Add to the toolbar: Enable this checkbox to display the custom command on the toolbar.
      • Tooltip: The text that displays when the mouse is paused over the command icon on the toolbar.
      • Icon file path: Specifies the path to a small (16x16 pixel) JPG or PNG image file to be displayed as an icon on the toolbar.
Click OK, to add the new custom commands to the Customize Commands menu.
