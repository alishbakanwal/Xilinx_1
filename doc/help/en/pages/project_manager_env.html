Project Manager

The Project Manager environment lets you work in the current project without having an open in-memory design. You can access this environment any time by clicking Project Manager in the Flow Navigator.

In this environment you can:

  • Add design sources, constraints, and simulation sources.
  • Access the Vivado® IP Catalog, configure IP, and add user repositories.
  • Access Vivado Language Templates to define new RTL modules.
  • Examine the Project Summary to determine the state of the design, including resource utilization and timing.
  • Configure the current project using the Project Settings dialog box to change the target part, and configure synthesis, implementation, simulation, and bitstream options.
