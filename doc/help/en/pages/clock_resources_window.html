Clock Resources Window

For 7 series devices and the Zynq®-7000 family, the Clock Resources window provides a table-based representation of the I/O bank, clock, and GT resources on the device, which reflects the relative location of those resources on the device. You can use this window to quickly view or manually place I/O, clock, and GT objects in the design. You can drag and drop design objects from other windows, such as the Netlist window or the I/O Ports window, onto the device resources displayed in the Clock Resources window. You can select a device resource and use the right-click menu to mark or highlight the object to make it easy to locate in the Device window.

You can collapse or expand the cells of the Clock Resources window to hide or show the content in a specific region of the device. Click the minus sign (-) to hide, or click the plus sign (+) to show a specific section of the window.

The toolbar has the following commands:

Search Opens a search field to enter a search pattern. The Clock Resources window is searched for the specified value.
Collapse All Collapses the Clock Resources window cells.
Expand All Expands the Clock Resources window cells.
Scroll Scrolls the Clock Resources window to display the selected object.

The Clock Resources window appears in the Vivado® IDE workspace, to display with other workspace windows. For more information on workspace windows refer to this link in the Vivado Design Suite User Guide: Using the Vivado IDE (UG893).
