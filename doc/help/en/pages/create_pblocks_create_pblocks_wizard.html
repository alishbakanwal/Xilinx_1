Create Pblocks Wizard

The Create Pblocks wizard guides you through the process of creating Pblocks in order to create a floorplan for the current design.

Floorplanning is recommended when a design does not meet, or has never met timing. Floorplanning lets you place one or more levels of the design hierarchy, or other related logic, into a reserved region of the device called a Pblock. This Pblock guides high-level placement of cells assigned to a region, while letting the placer do the detailed placement within that region.

To create a Pblock you must define the name of the Pblock, assign cells from the design to the Pblock, and assign an area of the device to the Pblock to reserve device resources for placement of the assigned cells. The steps the wizard follows include the following:

  • Select Cells
  • Format Names
  • Create Pblocks Summary
