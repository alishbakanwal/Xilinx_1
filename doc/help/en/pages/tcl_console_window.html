Tcl Console

The Tcl Console provides a command line interface for the Vivado® IDE, letting you directly enter commands and interact with elements of the design. The command line auto-completes commands as you type them, and presents a menu of commands when there are multiple possible matches. Click one of the commands, or keep typing to narrow the selection to a specific command.

The Tcl Console also presents a transcript of commands that were entered as well as messages returned by the Vivado tools when running those commands. The transcript is updated in real-time, as Tcl commands are executed by the Vivado IDE.

Tip: The Tcl commands you use are written in the vivado.jou file, and a transcript of commands and their messages are written to the vivado.log file. See this link in the Vivado Design Suite User Guide: Using the Vivado IDE (UG893) for more information.

The Tcl Console also provides a help system for the Tcl commands that you can access by entering the following at the command line:

help command_name

The Tcl Console toolbar has the following commands:

Collapse all Collapses the transcript of the Tcl Console to display just the Tcl commands that were run and not the generated messages.
Tip: Use the collapsed view to copy and paste commands from the Tcl Console into a text editor to help in Tcl scripting.
Expand all Expands the transcript so that all of the messages from the various Tcl commands are displayed, which facilitates monitoring the tool and understanding your design.
Pause Pauses the transcript from scrolling to let you read a specific section. The Vivado IDE continues capturing the transcript even when paused. Click the Pause button again to resume scrolling the transcript.
Show Displays a search bar at the bottom of the Tcl Console to find and highlight a string of text in the transcript.
Clear Clears the Tcl Console of previous commands and messages.
