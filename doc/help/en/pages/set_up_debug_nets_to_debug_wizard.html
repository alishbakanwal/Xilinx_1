Nets to Debug

This page of the Set Up Debug wizard lets you specify nets in the current design to debug. You can select nets in other windows to add to the list of nets to debug. The Nets to Debug page is a modeless page, that stays open while you select nets in other windows of the Vivado® IDE, such as the Netlist window, or Schematic window, and add those nets to the list of nets to debug.

Tip: Nets that have the MARK_DEBUG property are automatically added to the list of nets to debug. Any nets that are currently selected in the Vivado IDE when the Set Up Debug wizard is launched are also added to the list of nets to debug.

When nets are added to the debug list, the Vivado Design Suite automatically determines the appropriate clock domain to use for the net. The clock domain is important to ensure that the ILA is clocked synchronous to the nets on the ILA so that you get cycle accurate data for the probed nets. Nets that have a common clock domain can be probed from a single ILA core. Nets that are in different clock domains require separate ILA cores, which are inserted automatically by the Set Up Debug wizard. You can select a net and change the clock domain from the one that the Vivado tool has automatically selected. The wizard creates a unique ILA core per clock domain, depending on the clock domain selections for the nets being probed.

Important: Whatever clock is selected for the ILA core should be a free-running clock. A clock that is only occasionally active will only trigger activity and capture data when the clock is active.

ILA probes are also added to the ILA core for each net or bus in the list. Scalar nets have a probe width of 1. Buses have probe widths equal to the width of the bus. The Vivado tool automatically determines the appropriate width of the probe for a signal. You can also configure the type of probe for a specific signal. Probes can trigger activity on a signal, and capture data from the signal. You can also set the probe to only trigger activity or capture data.

The toolbar includes the following commands:

Search Opens a search field to enter a search pattern.
Collapse all Collapses the hierarchy of debug nets.
Expand all Expands the hierarchy of debug nets.
Scroll Scroll to the selected signal when a net is selected in another window such as the Netlist window or Schematic window.
Filter Filter the nets that currently have an undefined clock domain. This lets you view only unassigned nets.
Select Clock Domain Opens the Select Clock Domain dialog box to let you choose or change the clock domain assigned to a selected net or bus.
Add selected nets Add nets that are currently selected in other windows, such as the Netlist window, or the Schematic window, to the list of nets to debug.
Remove nets Remove the currently selected nets from the list of nets to debug.

Click Next to open the ILA Core Options page.
