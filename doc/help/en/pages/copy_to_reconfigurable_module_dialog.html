Copy to Reconfigurable Module

Copy the files from a Reconfigurable Module, selected in the Sources window, into the specified RM. This lets you define a new empty RM in the Add Reconfigurable Module dialog, and populate it later with the file contents of an existing RM.

  • Copy to Reconfigurable Module: Specify the name of the RM to copy the file contents into.

Click OK to copy the files to the Reconfigurable Module. Click Cancel to close the dialog box without making changes.
