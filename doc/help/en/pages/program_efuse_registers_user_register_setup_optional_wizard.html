User Register Setup (Optional)

This page of the Program eFUSE Registers wizard lets you define both 32-bit and 128-bit user codes for the FUSE_USER and FUSE_USER128 registers.

  • Specify 32-bit value to program into USER register: Enable programming of the FUSE_USER register.
    • USER bits [31:0]: Specify the code as a hexadecimal value.
  • Specify 128-bit value to program into USER128 register: Enable programming of the FUSE_USER128 register.
    • USER bits [127:0]: Specify the code as a hexadecimal value.
