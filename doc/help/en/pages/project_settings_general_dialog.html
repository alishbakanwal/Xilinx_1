Project Settings - General

The Project Settings dialog box allows you to specify values for various settings used throughout the design flow. The settings are grouped according to the steps in the design flow, such as synthesis, simulation, and implementation.

Tip: This dialog box is similar to the Vivado Options dialog box, which specifies settings for the overall Vivado® tools, rather than the current project.

The General settings let you adjust the default part and language-based settings.

  • Project device: Changes the default part or board using the part browser.
  • Target language: Specifies the preferred language for generated files.
  • Default library: All files that do not have an explicit library specification are compiled into this library, even Verilog files.
  • Top module name: Specifies the top module for your project. This value is retained even if the hierarchy update mode is set to automatic in the Sources window.
  • Language Options: Defines values for Verilog and VHDL options.
    • Verilog options: Opens the Verilog Options dialog box to specify 'include and 'define values or set all identifiers to uppercase.
    • Generics/Parameters options: Opens the Generics/Parameters dialog box to specify values for VHDL generics and Verilog parameters defined in the current project.
    • Loop Count: Adjusts the maximum loop count used for elaboration.
    Important: You can also define Verilog and Generics/Parameters options from the Project Settings - Simulation dialog box. The simulation settings apply to the simulation fileset and affect simulation but not synthesis.
