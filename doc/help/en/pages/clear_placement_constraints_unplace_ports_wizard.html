Unplace Ports

This page of the Clear Placement wizard displays information related to the selected and unselected ports in the design.
Tip: This page is only displayed if you have ports selected before running the Clear Placement command.
  • Ports to Unplace:
    • Unplace <#> selected ports: Directs the Vivado tool to clear the placement constraints for the currently selected ports, where <#> is the count of currently selected ports.
    • Unplace all except for <#> selected ports: Directs the tool to clear the placement constraints for all of the ports in the design, except the currently selected ports.
    • Unplace all ports: Directs the tool to clear the placement for all ports in the design.
    • Do not unplace any ports: Do not clear the placement of any ports, selected or unselected.
