Edit Enumeration

This dialog box lets you define enumerated values to display for a selected probe, replacing the actual value on the probed signal. The enumerated value is used for display in the ILA Wave window for instance. The enumerated value is displayed when the probed signal has the defined value. If the probe has a value for which no enumeration has been defined, the actual value is displayed.

The Enumerations table of the dialog box lists the currently defined enumerated values for the probe, and lets you edit or add to these enumerated values. The table shows the Name, Radix, and Value for the enumeration. The Name is the enumerated value that will be displayed instead of the actual value on the probe. The Radix is the form of the value used for evaluating when the enumerated value should replace the actual value. The Value is the actual value on the probed signal that the enumerated value will replace.

The toolbar includes the following commands:

  • Add (): Adds a new line to the Enumerations table, letting you define a new Name/Radix/Value combination.
  • Remove (): Removes the currently selected enumerations.

Click OK to apply the enumerations and close the dialog box.
