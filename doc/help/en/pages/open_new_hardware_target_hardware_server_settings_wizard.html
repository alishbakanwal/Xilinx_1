Hardware Server Settings

When connecting to the hardware server, you must first specify if the application is running on the local machine, or on a remote host.

Connect to: This field lets you choose to connect to a hw_server running on the local server, or specify a remote server. The Vivado® Design Suite is preconfigured to connect to the hw_server running locally. Nothing further is required. However, when connecting to remote host, the following fields must also be defined:
  • Host name: Specifies the name of the remote host.
  • Port: Specifies the communication port for connection to the server. The default port setting is 3121, but can be changed when launching the hw_server application.
    Tip: Type hw_server -help for a list of command options.

Click Next to connect to the specified hw_server, and open the Select Hardware Target page of the wizard.
