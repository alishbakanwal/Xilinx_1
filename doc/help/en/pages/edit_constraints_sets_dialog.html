Edit Constraints Sets

This dialog box lets you define a new constraint set or edit an existing constraint set. It also lets you create and add new constraints files to the specified constraint set.

The order of files in a constraint set is important, as constraints are read in from the files in the order listed. Constraints defined early in the list can be overwritten by constraints defined later in the list. Refer to this link in the Vivado® Design Suite User Guide: Using Constraints (UG903) for more information on constraint precedence.

  • Specify constraint set: Select the constraint set to edit. By default, the active constraint set is selected, however you can also select another existing constraint set or create a new constraint set for the project. Selecting Create Constraint Set opens the Create Constraint Set dialog box letting you specify the name for the new constraint set.
    • Make Active: Makes the selected constraint set the active constraint set for the project. The active constraint set provides all constraints applied to the current design, and is the target for saving new constraints. This checkbox is not available when you are editing the active constraint set.
  • Add Files: Opens a file browser in which you can select one or more constraint files (.xdc) or unmanaged Tcl scripts (.tcl). Selecting multiple files adds the files in sorted order to the file list.
    Tip: Alternatively, you can click and select Add Files.
  • Create File: Opens the Create Constraints File dialog box in which you can specify the file type, file name, and file location to write the file to. You can add constraints to the empty constraints file by editing the file, by defining constraints using the Timing Constraints Wizard, or by modifying the placement and routing of the design and saving the project or constraints.
    Tip: Alternatively, you can click and select Create Files.
  • Remove (): Removes selected files from the current project. The files are removed from the project but are not removed from the hard drive.
  • Move Up: When one or more files have been selected in the file list display area, the Up button () is activated, letting you move the selected file or files up in the listed order.
  • Move Down: When one or more files have been selected in the file list display area, the Down button () is activated, letting you move the selected file or files down in the listed order.
  • Copy constraints file into project: This checkbox lets you copy the specified constraints file into the sources folder of the local project directory. Refer to this link in Vivado® Design Suite User Guide: System-Level Design Entry (UG895) for more information on using local and remote sources.
