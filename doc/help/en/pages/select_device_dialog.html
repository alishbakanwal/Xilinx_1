Select Device

Select the default part or platform board to use for the current project. The device resources that the design is synthesized against and placed onto are determined by the selection of the target part or board.

  • Select: Parts | Boards: Lists the available target parts or boards for the current project.
    Important: The devices and boards that are available are determined at the time the Vivado Design Suite is installed. You can add uninstalled parts by using the Help > Add Design Tools or Devices command from the main menu.
  • Filter: Displays or hides the filter fields to filter the listed parts or boards according to various criteria.
  • Reset All Filters: Clears the currently selected device or board filters.
  • Search: Lets you further filter the devices or boards displayed in the results table by pattern matching the specified search string.

Displays the target parts or boards that match the specified filters and/or search string in a table of results in the lower portion of the dialog box. The resulting table displays several columns of information, such as Device Name, I/O Pin Count, and the number of Block RAMs and DSPs. To sort the results table, click in the column header. To sort multiple columns, press and hold the Ctrl key while clicking in the column headers. If you click once, the column is sorted in ascending order. If you click twice, the column is sorted in descending order.

Select a target part or board for the design, and click OK.
