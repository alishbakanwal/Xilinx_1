Messages Window

The Messages window displays the history of messages generated by the Vivado® Design Suite in the processing of your design. Messages are assigned specific message IDs and provide hyperlinks to source files or design objects the message is related to. Click the hyperlink in a message, and the associated source file is opened in the Text Editor, or the associated design object is selected in other windows, such as the Netlist or Device window.

The top bar of the Message window lists statistics related to the different messages. Messages are listed according to their severity. You can enable or disable the checkbox associated with a specific message severity to show or hide messages of that severity. The Show All or Hide All button lets you display or hide all messages.

In the Messages window, you can right-click a message to suppress a specific message, messages with the same ID, or messages with the same severity. This limits the messages reported in the Messages window. You can also enable the Wrap Lines command from the right-click menu, so that messages wrap onto multiple lines if needed.

The Messages window toolbar includes the following commands:

Search Opens a search field to enter a search pattern. Messages matching the search pattern are displayed.
Collapse Collapses all message groupings.
Expand Expands all message groupings.
Group by File Groups messages according to the file they are associated with.
Group by ID Groups messages by their message IDs.
Filter Applies a filter to the Messages window to limit the number of messages displayed. Messages can be filtered by suppression or severity.
Manage Message Suppression Opens the Manage Suppression dialog box to let you define and manage rules for suppressing messages.
Discard User Created Messages Clears the Messages window of messages related to previously executed Tcl commands and user scripts.
Tip: This command does not discard messages related to Vivado Design Suite processes such as synthesis and implementation runs.
