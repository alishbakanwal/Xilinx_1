Generated Clock Constraints

This dialog box lets you define the divide_by option of the clock created by the create_generated_clock command.

  • Divide: Specifies the integer value to divide the specified primary clock by in order to define the period of the generated clock.

Click OK to add the attribute of the clock to the currently selected clock or clocks in the Timing Constraints wizard.
