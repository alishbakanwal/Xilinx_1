Combinational Constraints

This dialog box lets you define the input and output delays of the currently selected combinatorial path. The delays of the path are associated with the set_input_delay and set_output_delay commands.

  • Input min: Specifies the minimum input delay for the selected path (in nanoseconds).
  • Input max: Specifies the maximum input delay for the selected path.
  • Output min: Specifies the minimum output delay for the selected path.
  • Output max: Specifies the maximum output delay for the selected path.

Click OK to add the delays to the currently selected paths.
