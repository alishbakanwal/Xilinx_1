Select Project Template

The Vivado® Design Suite includes multiple design templates to let you quickly create an example design featuring different design constructs, such as IP, block designs, and embedded processors. The design templates support a variety of target parts or platform boards.

  • Template: Select a template from the menu of design templates to create a new instance of the example design.
  • Description: This field displays a brief description of the design and in some cases, an image representing the design.
