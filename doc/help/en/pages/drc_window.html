DRC Results

The DRC Results window displays the results of the Report DRC command. The window lists the violations of the design rule checks performed, arranged by the associated rule name. Each violation shows the assigned severity and details of the violation, such as a list of related design objects. Selecting a violation displays details in the Violation Properties window and selects related design objects in other windows, such as the Netlist or Device windows.

The DRC Results window is a table window, which presents data about design rule violations in column form. To display and hide columns as well as manage column widths, right-click in the header of any column. You can sort information in the table by clicking once or twice in the column header. See this link in the Vivado® Design Suite User Guide: Using the Vivado IDE (UG893) for more information.

The top bar of the DRC Results window lists statistics related to the different violation severities. You can enable or disable the checkbox associated with a specific violation severity to show or hide those violations. The Show All or Hide All button lets you display or hide all violations.

The DRC Results toolbar includes the following commands:

Search Opens a search field to enter a search pattern. DRC violations matching the search pattern are displayed.
Collapse all Enabled when the DRC violations are grouped by rule. Collapses the violation hierarchy.
Expand all Enabled when the DRC violations are grouped by rule. Expands the violation hierarchy.
Group by Rule Groups the DRC violations by rule hierarchy. Violations are simply listed by name when this command is not enabled.
