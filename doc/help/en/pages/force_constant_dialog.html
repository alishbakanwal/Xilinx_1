Force Constant

This dialog box lets you establish a constant value for a selected signal, wire, or register to be applied during simulation. The constant value can be specified in a particular radix format, and can be applied to the object after a specified simulation runtime, and for a specified amount of time. After the Force Constant command has expired, the object resumes normal operation as defined in your HDL design.
Tip: To apply a recurring value to an object use the Force Clock command.
  • Signal name: Displays the name of the selected HDL object.
  • Value radix: Select the radix format that the value will be specified in.
  • Force value: Define the value to assign to the signal for the specified time range.
  • Starting after time offset: Specify a time offset to apply before assigning the constant value. The offset should be specified as nanoseconds, and the default offset is 0.
    Important: Both start and cancel time offset values are applied from the current simulation runtime, current_time, at the time the Force Constant command is applied.
  • Cancel after time offset: Specify the time offset to cancel the constant value. The object is set to the defined value for the duration of the simulation, until the offset has expired. If no Cancel value is specified, the constant is applied for the duration of simulation runtime.
