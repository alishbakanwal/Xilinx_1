Peripheral Details

  • Name: Specify the name of the new AXI4 peripheral
  • Version: Specify a version number that reflects the <major#.minor#> version scheme.
  • Display Name: Specify the display name to use when listing the peripheral in the Vivado® IP catalog.
  • Description: Enter a description of the peripheral.
  • IP Location: Specify the directory to create the new AXI IP in.
  • Overwrite existing: Enable to overwrite an existing directory of the same name.
The name and version of the AXI peripheral IP combine to create the directory name of the IP that is written to disk at the specified location.
Tip: If the specified combination of name and version already exists, an error is returned unless you enable the Overwrite existing option.
