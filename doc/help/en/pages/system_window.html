Design Canvas Window

The Vivado® IP integrator provides a design canvas that you can use to construct your block design. The Design Canvas window displays the block design, and lets you edit the design.

For editing the block design, the toolbar menu includes the following commands:

Add IP Add IP from the Vivado IP Catalog.
Make External Connect pins to external ports.
Customize Block Opens the IP customization dialog box for the currently selected IP. You can also re-customize IP by double-clicking a selected IP.
Validate Design Runs integrated design rule checks on the block design to identify any potential problems.
Regenerate Layout Redraws the whole block design to clean up the design.
Optimize Routing Reroutes the connections to clean up the design.
Show Interface Connections Only Displays only the connections between interface pins and ports, and hide all other connections.

You can pan and zoom the Design Canvas window to display the level of detail needed in the block design. The toolbar menu has the following commands for controlling what is displayed.

Zoom In Display more detail of the block design.
Zoom Out Display more area of the block design.
Zoom Fit View the whole block design.
Select Area Draw a rectangle to select objects in the Design Canvas window.
Fit Selection Zoom into and center the Design Canvas window around the currently selected objects.
Autofit Selection Automatically pan and zoom the Design Canvas window to center an object that has been selected in another window.
The Design Canvas window also supports the use of mouse strokes to zoom the display. Click and hold the left mouse button, and draw a line or box to zoom into or out of the block design. You can use the scroll bars shown on the right side or bottom of the Design Canvas window to scroll the display and see a new area.
Tip: The scroll bars are only visible when the display area is zoomed into the block design.

The toolbar also has commands to let you configure the IP Catalog and what is displayed in the Design Canvas window:

Block Design Options Configure display options and colors for objects in the block design. See this link in the Vivado Design Suite User Guide: Designing Subsystems Using IP Integrator (UG994) for more information.
IP Settings Opens the Project Settings - IP dialog box to let you configure IP-related tool settings for the current project.
