Older Project Version

This dialog box appears when you are opening a project that was created, or last updated, in a previous version of the Vivado® Design Suite. Changes to the internal data structure require the project to be updated to the latest version or for the project to be opened in read-only mode for viewing purposes only.

Tip: Archive the original project, or create a copy of the project prior to updating to the latest version. Xilinx also recommends archiving older releases of the Vivado Design Suite for use with older project to ensure compatibility.
  • Automatically upgrade to the current version: Upgrades the internal project data structure to the latest Vivado Design Suite version. After upgrading the project, you will also need to separately upgrade any IP or block designs used in the design. Refer to this link in the Vivado Design Suite User Guide: Designing with IP (UG896), or this link in the Vivado Design Suite User Guide: Designing IP Subsystems Using IP Integrator (UG994) for more information.
    Important: This process cannot be reversed.
  • Open project in read-only mode: Does not upgrade the internal project data, but opens the project for viewing or for saving a copy of the project.
