Open New Hardware Target

You can open the Vivado® Design Suite Hardware Manager to download your design bitstream to a target device, and use the Vivado logic analyzer and Vivado serial I/O analyzer features of the Hardware Manager to debug your design. To open a new hardware target, you must connect to a hardware server that is running either on the local host, or on a remote system. The hardware server manages the connection from the Vivado Design Suite, or Vivado Lab Edition, to the physical programming target. It should be running on the machine connected to the hardware programmer, or test board.

The hw_server command must be launched as a separate application, and can be found in the /bin folder of your Vivado Design Suite installation directory. Refer to this link in the Vivado® Design Suite User Guide: Programming and Debug (UG908) for more information.

To open the hardware target, the wizard steps you through the following pages:

  • Hardware Server Settings
  • Select Hardware Target
  • Open Hardware Target Summary

Click Next to proceed to the Hardware Server Settings page.
