New Suppression Rule

This dialog box lets you define a rule for suppressing messages in the Message window by matching specified search strings with message contents. The case of the search string is important, because the search is case-sensitive. The search matches partial words, without the need of wildcards, so you must take care in defining your suppression rules, or you might suppress messages unintentionally.

Enter a Match String to use as a search pattern to suppress messages that contain that string. You can also enter multiple match strings in a single suppression rule, which are used in an AND capacity.

Important: Only messages that match all of the specified search strings are suppressed.

Click the Add command () to add a search string to the message suppression rule.

If you specified multiple Match Strings, you can use the Remove command () to remove specific strings. The order of the strings does not matter for the suppression rule. If each of the search strings is matched in a message, that message is suppressed.

Click OK to save the new rule to the Manage Suppression dialog box.

Click Cancel to close without saving the new suppression rule.
