Export Constraints

This dialog box lets you export the in-memory design constraints to a new Xilinx® Design Constraints file (XDC).

  • Output file name: Specify the path and file name for the XDC file to write.
  • Include unconstrained placement locations: By default only the fixed locations (LOCs) will be written to the specified XDC file. This checkbox lets you export both fixed and unfixed placement locations to the constraint file being written.
    Tip: Fixed LOCs are associated with user-assigned placements, while unfixed LOCs are associated with tool assigned placements. See this link in Vivado® Design Suite User Guide: Using Constraints (UG903) for more information.
