Noise Results

The Noise Results window displays the results of the Report Noise command. The window is divided into two sections. The left side has a table of contents, including Summary, Messages, I/O Bank Details, and links to additional relevant information. The right side displays the information for the current selection.

In the I/O Bank Details, the noise is presented as a percent of the remaining noise margin on the package pin. Selecting an I/O bank or package pin from the Noise Results table also selects it in other windows, such as the I/O Ports window or the Package window.

The I/O Bank Details are displayed in table form. To display and hide columns as well as manage column widths, right-click in the header of any column. You can also sort information in the table by clicking once or twice in the column header. See this link in the Vivado® Design Suite User Guide: Using the Vivado IDE (UG893) for more information.

The Noise Results toolbar includes the following commands:

Search Opens a search field to enter a search pattern. I/O banks and package pins that match the search value are displayed.
Collapse all Collapses the I/O bank hierarchy.
Expand all Expands the I/O bank hierarchy.
Scroll Scrolls the Noises window to display the selected I/O bank or package pin when an object is selected in another window, such as the Device window.
