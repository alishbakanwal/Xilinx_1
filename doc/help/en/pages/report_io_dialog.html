Report I/O

This dialog box lets you report the I/O assignments in the current design. The report includes details of the I/O banks of the current design and each pin of the device, and also includes device specific information such as target part, package, and speed grade.

  • Results name: Specify the name to apply to the report in Report I/O window in the Vivado IDE.
  • Export to file: Write the results of the I/O report into the specified file. You can specify the file format to export as either text or XML.
  • Open in a new tab: Open the I/O report in a new tab of the Report I/O window in the Vivado IDE. If this checkbox is not enabled, the new report overwrites the last report that was opened.
