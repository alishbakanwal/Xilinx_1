Export IBIS Model

IBIS models describe the signal characteristics of device interconnects based on V/I curve data, as constructed for each IOB standard on a device. When you export an IBIS model the Vivado tool outputs a file (.ibs) with a list of pins used in the design, the signals connected to those pins, and the IBIS buffer models for the IOBs connected to the pins.

  • Output File: Specify the path and file name for the IBIS model file.
  • Options:
    • Include all models: Include all IOB models for the current device. By default, only IOBs used in the design are included.
    • Disable per pin modeling: Disable per-pin modeling of the transmission path from the die pad to the package pin. The IBIS model will instead include a single RLC transmission line model representation for all pins in the [Package] section.
    • Maximum length of signal names: Specify the length of signal names in the output file.
    • Updated generic IBIS models file: Specify an IBIS model file for the device to override the IBIS models found in the Vivado installation for the current part. The model specified will be used instead of the installed models when generating the output IBIS file.
    • Updated parasitic package data file: Specify a parasitic package file (.pkg extension) to override the parasitic package file found in the Vivado installation for the current part.
    • Component name: Specify a new component name to write to the output file.
