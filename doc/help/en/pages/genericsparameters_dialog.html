Generics/Parameters

Specify Name/Value pairs for generic statements or parameters. This dialog box has a toolbar menu with the following commands:.

  • Add: Opens the Add Value dialog box to define a new Name/Value pair.
  • Remove: Removes the selected Name/Value pair.
  • Edit: Opens the Edit Value dialog box to edit the selected Name/Value pair.

Click OK to add the options to the Simulation settings, and close the dialog box.
