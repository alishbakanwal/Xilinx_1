Real Settings

The real-type radix causes bus values to be interpreted as either fixed point or floating point real numbers, based on settings of this dialog box. In the right-click menu of the Waveform window you can define the Radix of a signal as one of a number of types including real.
Important: Before you can assign the real radix to a signal, you must use the Real Settings dialog box to configure the real number.
  • Fixed Point
  • Floating Point

Fixed Point

Fixed point real numbers are numbers that have a fixed number of digits after the binary point, or decimal point, and are commonly used in DSP applications. You define the number of fixed bits to represent the fractional portion of the real number by specifying the number of bits following the binary point.

  • Signed: The fixed point number is a signed real number, with a sign bit.
  • Unsigned: The fixed point number is an unsigned real number, with no sign bit.
  • Binary Point: The binary point acts as a divider between the integer portion and the fractional portion of the real number. The specified number represents the bit at which the binary point occurs, counting from the least significant bit (LSB).

Floating Point

Floating point real numbers are represented by a fixed number of significant digits for the fractional portion (the significand) scaled by an exponent. The number's binary point, or decimal point, can float relative to the significant digits of the number.

  • Single precision: The real number is represented in a single-precision floating-point format occupying 32 bits, with a 24 bit fractional portion and 8 bit exponent.
  • Double precision: The real number is represented in a double-precision floating-point format occupying 64 bits, with a 53 bit fractional portion and 11 bit exponent.
  • Custom precision: Custom define the bits assigned to the exponent and the bits assigned to the fractional portion of the real number.
    • Exponent width: The number of bits assigned to the exponent
    • Fraction width: The number of bits assigned to the significand.
      Tip: The maximum value for the width of the exponent and fractional portion is 80 bits each.

Click OK or Apply to assign the Real Settings to the selected signal in the Waveform window. This will also set the signal radix to Real.
