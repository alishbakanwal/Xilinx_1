Project Settings - Synthesis

The Project Settings dialog box allows you to specify values for various settings used throughout the design flow. The settings are grouped according to the steps in the design flow, such as synthesis, simulation, and implementation.

The Synthesis settings let you define the constraint set and strategy as well as customize the options for the active synthesis run. The default options for the selected synthesis strategy are displayed in a table, and you can modify options or values to customize the strategy.

  • Default constraint set: Specifies the constraint set to use during synthesis. You can create a new constraint set using the Edit Constraints Set command from the right-click menu of the Sources window, or by using the Add Sources wizard.
  • Strategy: Specifies the predefined Vivado® Design Suite strategies, or lets you create your own strategy by modifying the options for the selected strategy.
    Tip: When you change the options for a selected strategy, you can click the Save Strategy As button () to save those options as a new user-defined strategy.
  • Description: Provides a brief summary of the goal of the selected strategy. You can define the description when saving user-defined strategies.
  • Synth Design options: Shows a table view of the options available in the selected synthesis strategy. Select an option in the table to display a description of the option.
