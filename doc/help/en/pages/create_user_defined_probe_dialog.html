Create User Defined Probe

This dialog box lets you create custom user-defined probes. The new probe can combine specific bit values of existing probes to simplify or clarify the data presented in the waveform viewer. Physical ILA probe ports and constant values can be added to the Probe Bits table in the dialog box to define a new custom probe.

User-defined probes can map bit values from a single physical probe on the ILA core, or can combine bit values from multiple physical probes onto a single user-defined probe. Probes that map bits from a single probe can be used to create triggers and view data. Probes that combine bits from multiple physical probes can only be used for viewing data in the ILA Waveform window and exporting to an ILA data file.

  • ILA Core: Select from a list of ILA Debug cores programmed into the current device.
  • New Probe Name: Specify a name for the user-defined probe. The probe name is auto-generated by the tool, but can be changed as needed.
    Important: For bus probes, the name must include the range of bits.
  • Add ():
    • Add Probe: Opens the Add Probes dialog box letting you choose from the available probes defined on the current ILA debug core. The selected probes are added to the user-defined probe at the current location.
    • Add Constant: Inserts a single bit with a constant value into the user-defined probe at the current location. The constant value defaults to 1, but can be changed as needed.
  • Remove (): Removes the currently selected probes and/or constants.
    Tip: You can select and remove individual bits of a bus probe.
  • Move Up: When one or more probes have been selected in the Probe Bits table, the blue Up arrow button () is activated, letting you move the selected probe or probes up in the listed order.
  • Move Down: When one or more probes have been selected in the Probe Bits table, the blue Down arrow button () is activated, letting you move the selected probe or probes down in the listed order.

Click OK to create the user-defined probe and close the dialog box.
