Project Settings - Elaboration

The Project Settings dialog box allows you to specify values for various settings used throughout the design flow. The settings are grouped according to the steps in the design flow, such as synthesis, simulation, and implementation.

For RTL designs containing out-of-context (OOC) hierarchical modules, IP, and block designs, the Elaboration settings let you choose between a smaller design that opens faster or a more detailed elaborated design.

  • Link IP Module Options
    • Blackbox model: If your design contains OOC design modules, IP cores, or block designs, use this model for a high-level view of the elaborated design, without the added hierarchical details.
      Tip: Use this view if you do not have an IP generated and ready to use in a top-down or team-based design flow.
    • Netlist model: Use this model for a complete elaboration of the design hierarchy, including the synthesized netlist for out-of-context IP or modules.
  • Load constraints: Load and preview constraints in the elaborated design.
    Important: Not all design constraints are valid in the elaborated design. Invalid constraints are ignored.
