Partial Reconfiguration Wizard

Important: You must first use the Tools > Enable Partial Reconfiguration command, or set the PR_FLOW property to TRUE on the project, and use the Create a Partition Definition command prior to using this wizard.

Partial Reconfiguration allows Reconfigurable Modules (RMs) to be swapped into and out of a Partition Definition (partitionDef) to create a unique configuration of the design based on the combination of static logic and an RM. The PR design flow requires the implementation of each PR configuration, resulting in partial bitstreams for the RMs, but complete bitstreams for each integrated configuration.

The Partial Reconfiguration wizard walks you through the steps of defining RMs to plug into a previously created partitionDef, defining PR configurations, and defining implementation runs for the PR configurations. The wizard includes the following pages:

  • Edit Reconfigurable Modules
  • Edit Configurations
  • Edit Configuration Runs
  • Partial Reconfiguration Summary
