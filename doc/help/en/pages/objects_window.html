Objects Window

The Objects window lists the various HDL objects that are defined in the current or selected scope as shown in the Scope window. Icons displayed beside the listed HDL objects show the type or port mode of each object. The window lists the Name, Value, and Data Type of the simulation objects.

The right-click menu of the Objects window lets you manage the various HDL objects within the current or selected scope, by adding the objects to the Waveform window, forcing the object to a specific value, or changing the radix for the object. To help with debugging a selected signal type HDL object, the Report Drivers command in the right-click menu prints the name and value of the driving signal, as well as the current value of the selected object.

The Objects window toolbar provides the following commands to filter and manage the displayed objects:

Search Opens a search field to enter a search pattern.
Input Enable or disable to display or hide the INPUT type objects.
Output Enable or disable to display or hide the OUTPUT type objects.
Inout Enable or disable to display or hide the INOUT type objects.
Internal Signal Enable or disable to display or hide the INTERNAL SIGNAL type objects.
Constant Enable or disable to display or hide the CONSTANT type objects.
Variable Enable or disable to display or hide the VARIABLE type objects.
