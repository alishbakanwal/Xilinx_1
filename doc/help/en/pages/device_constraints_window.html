Device Constraints Window

The Device Constraints window offers two modes to let you define the voltage levels for I/O Banks in the design (VREF) or define DCI_CASCADE chains in the design. Use the drop-down menu just below the toolbar menu to select either Internal VREF mode or DCI Cascade mode.

Internal VREF

The Internal VREF mode displays the I/O banks on the device and lets you drag and drop them onto the appropriate voltage levels to assign an internal voltage reference to the I/O bank. Initially, all I/O banks might be listed under the heading of NONE, indicating that no VREFs are assigned. Select one of the displayed I/O banks, and drag it onto a specific internal reference voltage. This assigns the INTERNAL_VREF property to the selected I/O banks.

DCI Cascade

The DCI Cascade mode lets you select two or more high-performance (HP) I/O banks to assign to a DCI cascade chain. This assigns the DCI_CASCADE property to the selected I/O banks. Select two or more I/O banks, and click the Add () button to define them as a chain. The Add DCI Cascade dialog box opens to let you define the master I/O bank for the cascade chain.

Each mode also has a toolbar that supports one or more of the following commands:

Search Opens a search field to enter a search pattern.
Expand all Expands all VREF or DCI Cascade categories.
Collapse all Collapses all VREF or DCI Cascade categories.
Scroll Scrolls the display to the selected object.
Add Adds a DCI Cascade rule. You must select two or more I/O banks prior to using this command.
Remove Removes the selected VREF or DCI Cascade rule.
