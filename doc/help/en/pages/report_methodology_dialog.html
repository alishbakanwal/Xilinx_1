Report Methodology

The Report Methodology dialog box lets you run design methodology checks to find errors or problems in the current design. Methodology analysis is a special form of design rule check that specifically checks for compliance with design methodology, and identifies common errors that arise from that process.

  • Results name: Specify the name to apply to the report in Methodology Results window in the Vivado® IDE.
  • Export to file: Write the results of the Noise report into the specified file. You can specify the file format to export as either CSV, HTML, or text.
  • Open in a new tab: Open the report in a new tab of the Methodology Results window in the Vivado IDE. If this checkbox is not enabled, the new report overwrites the last report that was opened.
