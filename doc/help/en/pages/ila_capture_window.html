ILA Capture Setup

This window is one of five windows displayed in the Hardware ILA dashboard by default. The content of the ILA Capture Setup window is determined by the Capture Mode defined in the ILA Settings window. When the Capture Mode is set to ALWAYS, there are no capture conditions to define, so the ILA Capture window is blank. When the Capture Mode is set to BASIC, the window has the following features:

Basic Capture

The basic capture condition is a boolean expression that evaluates the values on each debug probe included in the capture condition, and triggers the ILA data capture sequence when the expression evaluates to TRUE. The first step in defining the capture condition for the ILA debug core is to decide what debug probes to include. The ILA Capture Setup window lists the probes assigned to the capture condition in a table that displays the following information:
  • Name: Specifies the name of the probe on the ILA debug core.
  • Operator: Specifies the operator used to evaluate the capture condition on the probe. The operator can be changed by clicking in the field and selecting an operator from the drop down menu. The list of available operators depends upon the radix and possible values on the probe.
  • Radix: Specifies the radix for the value on the probe. You can specify the radix by clicking in the field and selecting the radix from the drop down menu.
  • Value: Specifies the probe value to use in the capture condition. The boolean expression defined by the Value and the Operator determine if the capture condition for the probe evaluates to true or false.
  • Port: Reports the port in the design that the probe is assigned to.
The boolean expression for each probe is evaluated according to the specified operator and value to determine if the expression is true or false. The capture condition of the ILA debug core is the result of a global boolean "AND", "OR", "NAND", or "NOR" expression applied to all of the ILA probe capture comparator results, as defined by the Set capture condition command of the toolbar menu.
Important: Only probes that are added to the ILA Capture Setup window participate in the capture condition. Any probes that are not in the window are not considered as part of the capture condition.

The toolbar for the ILA Capture Setup window includes the following commands:

Search Opens a text field to enter a search pattern to display specific probes of interest.
Add probes Opens the Add Probes dialog box, which lists the probes associated with the ILA debug core. Select probes and click OK to add the probes to the capture condition.
Remove selected probes Removes selected probes from the ILA Capture Setup window.
Set capture condition Lets you define the global capture condition for all probes.
