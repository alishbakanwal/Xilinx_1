Define Command Arguments

This dialog box lets you define required or optional arguments accepted by the Tcl procedure assigned to a custom command, and define default values for those arguments. For instance, you can define the arguments needed to run the following example command: 

example_proc {arg1 arg2 {arg3}}

In this example, the first and second arguments (arg1 and arg2) are required arguments, and arg3 is optional. In this dialog box you would define the argument names, comments, default values, and specify arg3 as optional using the features described below.

Add Lets you enter a display name for an argument needed by the Tcl proc. The new argument is added to the bottom of the list when you press ENTER.
Note: The specified name is for display and reference only. The argument is passed to the Tcl proc by position, and so must be listed in the order required by the command footprint of the Tcl procedure.
Remove Removes the selected arguments from the list.
Move up Moves the selected arguments up in the list.
Tip: The order the arguments are listed is the order they are passed to the Tcl command. In the example_proc, this means the first argument listed is passed as arg1, the second argument listed is passed as arg2, and the last argument listed is passed as arg3.
Move down Moves the selected commands down in the list.
  • Name: Specifies the display name of the argument. You can also edit the name directly in this field.
  • Comments: Specifies comments to be displayed in the Custom Commands Arguments Entry dialog box that opens when a custom command with arguments is run. The comments are displayed at the bottom of the dialog box when an argument is selected to provide some explanation of the argument or expected value.
  • Default value: Specifies a default value to be passed to the custom command unless overridden by the user at the time the command is run.
  • Optional: A checkbox specifying that the argument is optional.
    Note: Optional arguments should appear last in the argument list of a Tcl proc, after required arguments.

Click OK to define the arguments and close the dialog box.
