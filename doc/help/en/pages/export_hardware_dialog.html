Export Hardware

This dialog box simplifies the movement of designs from the Vivado Design Suite to the Xilinx® Software Development Kit (SDK) to create your software application, and associate the executable linkable file (ELF) file with the hardware design. It is used to export a top-level design that includes a block design from IP Integrator with an embedded processor like MicroBlaze, Zynq®-7000 All Programmable SoC, or Zynq UltraScale+™ MPSoC.

  • Include bitstream: Include the bitstream file in the exported hardware specification, or hardware description file. After the design has been implemented and the bitstream generated, you can export the bitstream to SDK for software application development.
    Tip: In the unusual case where the programming logic (PL) block does not contain any logic, you can export the design after synthesis.
  • Export to: Specify the path to the hardware specification file (.hdf) for creating a hardware project in SDK. The default is to write the file into the local project directory structure.
