Autoplace I/O Ports Wizard

The Autoplace I/O Ports wizard will automatically assign ports in the current design to I/O blocks (IOB) and package pins on the device. The Vivado tool follows the I/O standards, differential pair rules, and clocking requirements of the design while placing the ports.

The process that the Autoplace I/O Ports wizard follows depends upon the state of the design. Are there fixed and unfixed ports in the design? Have you selected ports prior to launching the wizard?

  • Which ports do you want to place?
    • All <#> ports: Autoplace all ports, where <#> is the number of ports in the design.
    • <#> selected ports: Place only the currently selected ports, where <#> is the number of selected ports.
    • Place unconnected ports: Enable autoplacement of unconnected ports.
