Report DRC

The Report DRC dialog box lets you verify the design against a selection of design rule checks and output the results to a named report in the DRC Results window or to a text file.

  • Results name: Specifies the name to apply to the DRC Results window in the Vivado® IDE.
  • Output File: Writes the results of the DRC report into the specified file.
  • Rule Decks: Lists the various rule decks that you can select to run. A DRC rule deck is a collection of design rule checks that can be run at different stages of the design flow. The Vivado Design Suite provides a set of predefined rule decks, and you can also create user-defined rule decks. The default rule deck covers the commonly needed rules for checking a design.
    Tip: Selecting a rule deck also selects all of the rule checks defined in that collection. You can enable or disable individual checks in the Rules field.
  • Rules: Specifies the individual checks to run during the Report DRC command. The checks are grouped into categories, which you can expand or collapse to view the individual checks. Select some or all checks in a category as needed.
  • Open in a new tab: Opens the DRC Results in a new tab of the Results window in the Vivado IDE. If this checkbox is not enabled, the new report overwrites the last report that was opened.
