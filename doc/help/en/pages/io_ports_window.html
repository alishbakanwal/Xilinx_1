I/O Ports Window

The I/O Ports window displays a list of the I/O ports defined in the current design and lets you configure attributes of the ports, such as I/O standard, drive strength, and slew. The I/O Ports window is a table window that presents data about the ports in column form. To display and hide columns as well as manage column widths, right-click in the header of any column. You can also sort information in the table by clicking once or twice in the column header. See this link in the Vivado Design Suite User Guide: Using the Vivado® IDE (UG893) for more information.

The right-click menu of the I/O Ports window includes commands to help you define new ports in an I/O planning project, configure existing ports, place I/O ports either automatically or manually, and fix I/O port assignment.
Important: All ports must have a defined I/O standard, other than default, and must be fixed to a package pin site prior to bitstream generation. Fixed ports are assigned or fixed by a user, while unfixed ports are assigned to a package pin location, but have not been fixed by the user.

The I/O Ports window toolbar includes the following commands:

Search Opens a search field to enter a search pattern. The I/O Ports table is searched for the specified value.
Collapse All Collapses the listed IP.
Expand All Expands the listed IP.
Tip: Expand and Collapse are only active when the I/O ports are grouped by Interface and Bus.
Group by Interface and Bus Groups the I/O ports according to the bus or interface definitions. Ports are listed by name when this command is not enabled.
Create I/O Port Interface Lets you define a new port interface to group selected ports under.
Schematic Opens a schematic view containing the selected ports.
Scroll Scrolls the I/O Port window to display the selected object.
Fix Ports Marks the selected ports as Fixed so that they will not be moved by any automatic placement commands or placement optimization.
