Edit Reconfigurable Modules

This page of the Setup Partial Reconfiguration (PR) wizard lets you create, edit, and delete Reconfigurable Modules (RMs) for a PR project. A Partition Definition (partitionDef) must first be created using the Create Partition Definition dialog box, or create_partition_def Tcl command.

The page displays a table of existing Reconfigurable Module Associations with Partition Definitions in the current project. This table shows the existing RMs in the current project and shows the partitionDefs they are associated with. The table also has a toolbar for managing the RMs:

  • Add (): Add a new RM to the current project. This command opens the Add Reconfigurable Module dialog box. Newly added RMs are displayed in the table.
  • Remove (): Remove the selected RM from the current project.
  • Edit (): Edit the attributes of the selected RM. This command opens the Edit Reconfigurable Module dialog box.
Click Next to proceed to the Edit Configurations page of the Setup Partial Reconfiguration wizard.
Note: The commands of the Setup Partial Reconfiguration wizard are not run after each page, but are saved until all the pages have been visited and the whole setup is complete. Click Skip to Finish to go to the Setup Partial Reconfiguration Summary page, and complete the setup without visiting all pages.
