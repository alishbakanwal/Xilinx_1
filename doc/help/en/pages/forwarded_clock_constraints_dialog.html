Forwarded Clock Constraints

This dialog box lets you define the invert option of the clock created by the create_generated_clock command.

  • Invert: Inverts the forwarded clock from the specified primary clock.

Click OK to add the attribute of the clock to the currently selected clock or clocks in the Timing Constraints wizard.
