Create I/O Port

The Create I/O Port dialog box lets you define a new external port to add to the current I/O Planning project.

  • Port name: Specifies the name of the port to create.
  • Direction: Defines the port as an input, output, or bidirectional port.
  • Diff pair: Defines the I/O port as a differential pair, creating two related ports. The port name defines the positive side of the pair, and the negative side is automatically created.
  • Create bus: Specifies the port as a bus port and defines the width of the port.
    • From: Specify the LSB of the bus to define the bus width.
    • To: Specify the MSB of the bus.
  • I/O standard: Defines the IOSTANDARD property of the port. Select from a list of available standards for the target device.
    Important: All ports must have a defined I/O standard, other than default, prior to bitstream generation.
  • Drive strength: Defines the DRIVE property of an output or bidirectional port.
  • Slew type: Defines the SLEW property of an output or bidirectional port.
  • Pull type: Defines the PULLTYPE property of the port.
  • In Term type: Defines the IN_TERM property of the port.
  • ODT type: Defines the ODT property of the port.
    Tip: ODT is used for UltraScale™ devices, and In Term type is specified for 7 series devices.
  • Location: Defines the PACKAGE_PIN property of the port.
    • Fixed: This checkbox indicates that the location assignment is fixed, and should not be changed by the Vivado tool.

Click OK to create the new I/O port.
