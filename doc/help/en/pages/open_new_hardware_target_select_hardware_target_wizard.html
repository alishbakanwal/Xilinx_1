Select Hardware Target

After connecting to the hardware server, the Hardware Manager examines the hardware programmer, or test board identify available targets on the JTAG chain. The results are displayed in the Hardware Targets table. Select the hardware target to open in the Hardware Manager.

Hardware Targets: Lists the available targets on the JTAG chain of the hardware programmer, or test board connected to the hardware server. The order of targets in a scan chain is determined by reading the ID codes from the chain and then matching the code to device properties such as device name and instruction register (IR) length.
  • Type: Displays the type of the device on the JTAG chain.
  • Name: Displays the name of the device on the chain.
  • JTAG Clock Frequency: You should use the default JTAG clock frequency, or 15 MHz for the Digilent cable connection and 6 MHz for the USB cable connection. If you cannot connect to the target at the default speed, you can decrease the JTAG clock frequency as described at this link in the Vivado Design Suite User Guide: Programming and Debug (UG908).

Hardware Devices: Displays the Name, ID Code, and IR Length of the currently selected hardware target.
