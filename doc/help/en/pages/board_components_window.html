Board Window

The Board window displays a list of all the components defined in the current target board. You can drag and drop the listed components onto the current block design. You can also select the components to see the interfaces they are currently connected to in the design.

When viewing the list of components in the Board window, you can use the Show Connected Board Interfaces Only command () to display only the currently connected components. If you select an unconnected component, you can use Connect Board Component command () to quickly add the component IP to the current block design. If you select a connected component, you can remove it with the Disconnect Board Component command ().
