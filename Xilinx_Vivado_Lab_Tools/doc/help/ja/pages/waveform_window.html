[Waveform] ウィンドウ

Vivado シミュレータでは、HDL オブジェクトはワイヤ、信号、レジスタなどの値を保持できるエンティティです。Vivado シミュレータを起動すると、[Waveform] ウィンドウにテストベンチからの最上位 HDL オブジェクトが表示されます。[Scope] または [Objects] ウィンドウからその他の HDL オブジェクトを追加して、シミュレーションを実行できます。

重要: [Waveform] ウィンドウには、キャプチャされたすべての HDL オブジェクトのシミュレーション データが必ず表示されるわけではなく、指定したオブジェクトのみを表示させることができます。信号がシミュレーション中に記録またはキャプチャされていれば、シミュレーション終了後に [Waveform] ウィンドウで信号を追加または削除できます。信号の記録に関する詳細は、『Vivado Design Suite ユーザー ガイド : ロジック シミュレーション』 (UG900) のこのセクションか、Tcl コマンドの log_wave を参照してください。

[Waveform] ウィンドウは 2 つの領域に分けられ、左側にオブジェクト名と値がリストされ、右側にシミュレーション時間と波形が表示されます。右クリック メニューは、2 つの領域間で異なります。左のオブジェクト側には、オブジェクト名と値の表示を設定したり、オブジェクトをグループ化したり、シミュレーション結果をよりわかりやすくするためのドライバーを追加するコマンドが含まれます。信号オブジェクトを選択して右クリック メニューの [Report Drivers] コマンドを使用すると、駆動信号の名前と値のほか、選択したオブジェクトの現在の値が表示されます。右の波形側には、タイムラインを管理したり、マーカーを追加したり、別の遷移イベントにタイムラインを上下に移動したりするコマンドが含まれます。左側でオブジェクトを選択すると、右側の波形コマンドの一部が使用可能になります。

波形ウィンドウのツールバーには、表示された波形のシミュレーション タイムラインを確認するための次のコマンドが含まれます。

Go to Cursor 波形が拡大されている際に使用します。現在のカーソル位置が中心に表示されるようになります。
Go to Time 0 波形カーソルがシミュレーション タイムラインの開始地点に移動されます。
Go to Last Time 波形カーソルがシミュレーション タイムラインの終了地点に移動されます。
Previous Transition [Waveform] ウィンドウ左のオブジェクト側で信号を選択すると、現在の位置から選択した信号の遷移よりも 1 つ前の地点にカーソルが移動されます。
Next Transition 波形ウィンドウ左のオブジェクト側で信号を選択すると、現在の位置から選択した信号の次の遷移の地点にカーソルが移動されます。
Add Marker 現在のカーソル位置にマーカーが挿入されます。
Previous Marker タイムラインに任意の数のマーカーが追加されている場合に、カーソルを現在の位置から前のマーカーの地点に移動します。
Next Marker タイムラインに任意の数のマーカーが追加されている場合に、カーソルを現在の位置から次のマーカーの地点に移動します。
Swap Cursors タイムラインに複数のカーソルが表示される場合に、カーソル間を切り替えます。
注: Ctrl キーを押しつつカーソルをクリックして選択し、マウスをドラッグすると、タイムラインに 2 つ目のカーソルを追加できます。詳細は、『Vivado Design Suite ユーザー ガイド : ロジック シミュレーション』 (UG900) のこのセクションを参照してください。
Snap to Transition カーソルを移動する際に、そのカーソルがタイムラインの一番近い遷移位置に置かれます。オフににすると、カーソルをタイムラインにそって自由に移動できるようになります。
Floating Ruler タイムラインにカーソルが複数表示される場合に、波形表示の一番下のフロート ルーラーが表示/非表示になります。ルーラーは、イベント間の時間間隔を計るために使用できます。

ユーザーは、波形ウィンドウの表示内容を指定したり、波形の特定エリアを表示するのに必要な詳細レベルまで拡大したり、シミュレーションの時間範囲を指定したりできます。ツールバー メニューには、[Waveform] ウィンドウの表示を制御するための次のコマンドが含まれます。

Options 信号の基数を設定したり、信号名を切り捨てたりといった [Waveform] ウィンドウの表示内容を設定するのに選択可能なオプションが表示されます。詳細は、『Vivado Design Suite ユーザー ガイド : 合成』 (UG900) のこのセクションを参照してください。
Save [Waveform] ウィンドウが波形設定ファイルに保存されます。
Zoom Fit [Waveform] ウィンドウでシミュレーション ランタイム全体が表示されます。
Zoom In より小さなシミュレーション ランタイムの周期に表示が拡大され、信号波形とシミュレーション イベントの詳細が表示されます。
Zoom Out 表示が縮小されて、より多くのシミュレーション ランタイムが表示されます。
[Waveform] ウィンドウでは、クリックによる表示の拡大/縮小機能がサポートされています。クリックしてドラッグしながら線やボックスを描くと、デバイスを拡大または縮小表示できます。[Waveform] ウィンドウの右側または下側のスクロールバーを使用すると、画面をスクロールして別の新しいエリアを表示できます。
ヒント: アナログ波形ではさらに多くのマウス動作パターンが使用できます。 詳細は、『Vivado Design Suite ユーザー ガイド : ロジック シミュレーション』 (UG900) のこのセクションを参照してください。

[Waveform] ウィンドウは Vivado® IDE のワークスペースに表示されます。ワークスペースの詳細は、『Vivado Design Suite ユーザー ガイド : Vivado IDE の使用』 (UG893) のこのセクションを参照してください。
