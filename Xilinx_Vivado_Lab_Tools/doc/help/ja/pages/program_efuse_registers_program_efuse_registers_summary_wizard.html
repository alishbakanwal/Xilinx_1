[Program eFUSE Registers Summary] ページ

[Program eFUSE Registers Summary] ページには、1 度に書き込み可能な eFUSE レジスタへ書き込まれる変更点がレポートされます。

  • [Back] をクリックすると、前のページに戻って必要に応じて選択を変更できます。
  • [Finish] をクリックすると、指定どおりに eFUSE レジスタに書き込まれます。最終的な警告ダイアログ ボックスが eFUSE ビットの実際の書き込み前に表示されます。これは最後の警告になります。
  • [Cancel] をクリックすると、eFUSE への書き込みなくウィザードは閉じます。
