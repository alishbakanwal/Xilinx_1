[Specify Top Level Cell] ダイアログ ボックス

[Specify Top Level Cell] ダイアログ ボックスでは、現在のデザインのセル オブジェクトを見つけるための検索条件が定義できます。

このダイアログ ボックスには、2 つのセクションがあります。

  • [Query] セクション
  • [Results] セクション

[Query] セクション

  • [Options] : ドロップダウン リストからプロパティ、演算子、および値を選択すると、現在のデザイン内のオブジェクトが検索されます。
    • [Property] : セルの NAME または IS_PRIMITIVE プロパティのいずれかがリストされます。
    • [Operator] : 指定した [Property] によって文字列またはブール型演算子が表示されます。
    • [Value] : 指定した [Property] の既知の値または使用可能な値がリストされるか、直接入力した文字列または値が表示されます。このフィールドには、ワイルドカードとしてクエスチョン マーク (?) およびアスタリスク (*) 文字を使用できます。
    • [Add a new criterion] () : [Options] セクションに新しい行が追加されます。
    • [Remove this criterion] () : [Options] セクションから選択した行が削除されます。
    ヒント: [Options] セクションに複数の行がある場合は、条件間の論理関係を AND または OR のいずれかとして指定する必要もあります。AND の場合は指定した条件すべてに一致するオブジェクトが戻され、OR の場合は指定した条件のいずれかに一致するオブジェクトが戻されます。
  • [Regular Expressions] : [Value] フィールドが正規表現 (regexp) 検索パターンとして定義されます。正規表現構文の詳細については、このリンクを参照してください。
  • [Ignore case] : 正規表現の検索にのみ適用されます。大文字と小文字は区別されません。通常は、検索パターンの大文字/小文字は区別されます。
  • [Search hierarchically] : デザイン階層の各レベルが検索されます。デフォルトでは、現在のインスタンス current_instance、またはデザイン階層の最上位が検索されます。このオプションは、[Find] フィールドに指定したオブジェクトのタイプによって異なります。
  • [Of Objects] : [Find] 検索をデザイン全体ではなく、選択したオブジェクトのみに限定します。たとえば、特定のネットに接続されたセルか特定の Pblock に割り当てられたセルを検索できます。参照ボタン () をクリックして [Of Objects - Find] ダイアログ ボックスを開いて検索するか、該当する Tcl コマンドを手動で入力します。
  • [Find] : [Options] フィールドで定義された検索が実行され、結果がダイアログ ボックスの [Results] セクションに表示されます。

[Results] セクション

[Results] セクションには、2 つのリスト ボックスが表示され、その間にツールバー メニューが含まれます。左側のリスト ボックスには、定義した検索条件に合った結果が表示されます。右側のリスト ボックスには選択した特定のセルが含まれ、それらのセルが [Specify Top Level Cells] ダイアログ ボックスから戻されるようになります。

ツールバー メニューには、左側と右側の間でセル オブジェクトを移動するためのコマンドがあります。
Move right 左側のリスト ボックスから右側のリスト ボックスに選択したセルが移動されます。
Move left 右側のリスト ボックスから左側のリスト ボックスに選択したセルが移動されます。
Move all objects right 左側のリスト ボックスから右側のリスト ボックスにすべてのセルが移動されます。
Move all objects left 右側のリスト ボックスから左側のリスト ボックスにすべてのセルが移動されます。
ヒント: タイミング パスの始点は緑色、終点は赤色、通過点は選択した色で表示されます。

左側のリスト ボックスから右側のリスト ボックスに必要なセルを移動して [OK] をクリックすると、ダイアログ ボックスが閉じて、選択したセルが戻されます。
