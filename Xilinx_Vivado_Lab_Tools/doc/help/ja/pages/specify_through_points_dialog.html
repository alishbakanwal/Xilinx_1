[Specify Through Points] ダイアログ ボックス

[Specify Through Points] ダイアログ ボックスでは、タイミング パスの通過点オブジェクトを定義できます。Vivado タイミング エンジンでは、指定したピン、セル インスタンス、またはネットを通過するパスのみがタイミング解析で考慮されます。シーケンス内の複数の通過点を指定し、特定のパスを解析できます。パスを定義するには、通過点の指定順序が重要です。通過点を複数のオブジェクトで指定することもできます。この場合、指定の通過オブジェクトのいずれかを通過するタイミング パスが考慮されます。

  • [Find names of type] : 通過点として使用するセル、セル ピン、ネットを検索できます。
  • [Options] : ドロップダウン リストからプロパティ、演算子、および値を選択すると、現在のデザイン内のオブジェクトが検索されます。
    • [Property] : リストされるプロパティは、[Find names of type] で選択したオブジェクトのタイプによって異なります。[More Properties] をクリックすると、リストにプロパティを追加できます。
    • [Operator] : 指定したプロパティ用にドロップダウン リストに表示される既知の値または使用可能な値から選択するか、文字列または値を直接入力できます。
    • [Value] : リストされる値は、既知の値、または指定した [Property] に対して使用可能な値から定義するか、直接入力した文字列または値にできます。このフィールドには、ワイルドカードとしてクエスチョン マーク (?) およびアスタリスク (*) 文字を使用できます。
    • [Add a new criterion] () : [Options] セクションに新しい行を追加します。
    • [Remove this criterion] () : [Options] セクションから選択した行を削除します。
    重要: [Options] セクションに複数の行がある場合は、条件間の論理関係を AND または OR のいずれかとして指定する必要もあります。AND の場合は指定した条件すべてに一致するオブジェクトが戻され、OR の場合は指定した条件のいずれかに一致するオブジェクトが戻されます。
  • [Regular Expressions] : [Value] フィールドが正規表現 (regexp) 検索パターンとして定義されます。正規表現構文の詳細については、このリンクを参照してください。
  • [Ignore case] : 正規表現の検索にのみ適用されます。大文字と小文字は区別されません。通常は、検索パターンの大文字/小文字は区別されます。
  • [Search hierarchically] : デザイン階層の各レベルが検索されます。デフォルトでは、現在のインスタンス current_instance、またはデザイン階層の最上位が検索されます。このオプションは、[Find names of type] フィールドに指定したオブジェクトのタイプによって異なります。
  • [Of Objects] : 検索をデザイン全体ではなく、選択したオブジェクトのみに限定します。たとえば、特定のセル ピンのネットを見つけます。参照ボタン () をクリックして [Of Objects - Find] ダイアログ ボックスを開いて検索するか、該当する Tcl コマンドを手動で入力します。
  • [Find] : [Options] セクションで定義した検索が実行されます。検索結果は、[Results] セクションの [Found] フィールドに表示されます。
  • [Results] :
    • [Found] : ダイアログ ボックスの [Options] セクションで定義した検索で見つかったオブジェクトの数が表示され、そのオブジェクトがリストされます。
    • 次のツールバー ボタンを使用すると、[Found] と [Selected] リスト ボックス間でオブジェクトを移動できます。
      Move the selected items to the right 選択したオブジェクトが [Found] から [Selected] に移動されます。
      Move the selected items to the left 選択したオブジェクトが [Selected] から [Found] に移動されます。
      Move all the items to the right すべてのオブジェクトが [Found] から [Selected] に移動されます。
      Move all the items to the left すべてのオブジェクトが [Selected] から [Found] に移動されます。
    • [Selected] : 見つかったアイテムから現在選択されているオブジェクトの数が示され、そのオブジェクトがリストされます。選択したアイテムにより、タイミング パスの通過点が定義されます。
    • 次のツールバー ボタンを使用すると、選択したオブジェクトを並べ替えることができます。
      Sort 選択したオブジェクトがアルファベット順に並べ替えられます。1 度クリックすると昇順で、もう 1 度クリックすると降順になります。
      Move the selected items up 現在選択しているオブジェクトがリストの 1 つ上に移動されます。
      Move the selected items down 現在選択しているオブジェクトがリストの 1 つ下に移動されます。
      Move the selected items to the top 現在選択しているオブジェクトがリストの一番上に移動されます。
      Move the selected items to the bottom 現在選択しているオブジェクトがリストの一番下に移動されます。
  • [Command] : 指定した通過点オブジェクトを取得する Tcl コマンドが表示されます。表示されたコマンドを使用すると、Vivado® IDE で使用される Tcl コマンドを作成しやすくなります。
  • [Enumerate] : Tcl コマンドに選択した通過点オブジェクトを個別に列挙でき、ワイルドカードを使用する必要をなくします。
    重要: オンにすると、通過点オブジェクトを選択するときの Tcl コマンドの実行速度がかなり遅くなる可能性があります。

[OK] をクリックすると、[Report Timing] ダイアログ ボックスに Tcl コマンドが追加されます。

[Append] をクリックすると、Tcl コマンドが既存の通過点コマンドに追加されます。
