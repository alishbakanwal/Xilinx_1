[Properties] ウィンドウ

[Properties] ウィンドウには、現在選択されているプロジェクトの概要とそれに関する詳細な情報が表示されます。[Properties] ウィンドウの内容は、選択しているオブジェクトによって異なります。たとえば、選択した CELL オブジェクトに関して表示される情報は、NET や SITE オブジェクトを選択した場合とは異なります。

[Properties] ウィンドウには複数のビューが含まれます。 これらのビューには、選択したオブジェクトに基づいて異なり、そのオブジェクトの詳細が表示されます。たとえば、CELL オブジェクトを選択すると次のビューが表示されます。

  • [General] : 名前およびオブジェクト タイプなど、セルに関する概要が表示されます。
  • [Properties] : 選択したセルで定義されているプロパティがリストされます。プロパティによっては、[Properties] ウィンドウ内で直接編集できるものもあります。それ以外は読み取り専用なので、ツールでのみ変更できます。
  • [Nets] : 現在のセルに接続されたネットがリストされます。
  • [Cell Pins] : 現在のセルに接続されたピンがリストされます。

[Properties] ビューを選択すると、[Properties] ウィンドウ ツールバーに次のコマンドが表示されます。

Search 検索パターンを入力する検索フィールドが開きます。
Collapse all すべてのプロパティ階層の展開が閉じられます。
Expand all すべてのプロパティ階層が展開されます。
Show/Hide Hierarchy 関連するプロパティがグループ化されるか、グループが解除されます。
Add 現在選択しているオブジェクトに定義済みプロパティのリストからプロパティが追加されます。このボタンをクリックすると、[Add Properties] ダイアログ ボックスが開き、使用可能なプロパティのリストが表示されます。
Remove 現在のオブジェクトから選択したプロパティが削除されます。
Help [Properties] ウィンドウの一番下に小さなヘルプ エリアが表示されて、プロパティが示されます。
Sort アルファベット順にリストされたプロパティが並べ替えられます。
