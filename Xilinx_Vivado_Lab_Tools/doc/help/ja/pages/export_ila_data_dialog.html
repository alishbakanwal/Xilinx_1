[Export ILA Data] ダイアログ ボックス

このダイアログ ボックスでは、選択した ILA デバッグ コアからキャプチャされたデータをエクスポートできます。ILA デバッグ コアからキャプチャされたデータがアップロードされ、ILA アーカイブ ファイル (波形データベース ファイルおよびコンフィギュレーション ファイル、波形 CSV および VCD ファイル、デバッグ プローブ ファイルを含む) に書き込まれます。

  • [ILA Core] : 現在のデバイスにプログラムされた ILA デバッグ コアのリストから選択します。
  • [Format] : 書き込むデータ ファイルのファイル形式を指定します。使用可能な形式は、ネイティブ ILA アーカイブ、CSV、VCD です。
  • [File] : 書き込むデータ ファイルのパスとファイル名を指定します。
