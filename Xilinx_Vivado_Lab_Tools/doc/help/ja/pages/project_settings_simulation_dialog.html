[Project Settings] ダイアログ ボックス : [Simulation] ページ

[Project Settings] ダイアログ ボックスでは、デザイン フローを通して使用されるさまざまな設定に対して値を指定できます。設定ページは、合成、シミュレーション、インプリメンテーションなどのデザイン フローの手順別に分類されています。

[Simulation] ページでは、シミュレータ、言語、および詳細なシミュレーション設定が指定できます。

  • [Target simulator] : シミュレーションに使用するシミュレータを指定します。デフォルトは Vivado® シミュレータですが、多くのサードパーティ シミュレータもサポートされています。詳細は、『Vivado Design Suite ユーザー ガイド : ロジック シミュレーション』 (UG900) のこのセクションを参照してください。
  • [Simulator language] : シミュレータの言語モードを指定します。デザインのさまざまな IP に使用されるシミュレーション モデルは、IP でサポートされる言語によって異なります。詳細は、『Vivado Design Suite ユーザー ガイド : ロジック シミュレーション』 (UG900) のこのセクションを参照してください。
  • [Simulation set] : シミュレーション コマンドによりデフォルトで使用されるシミュレーション セットを指定します。
  • [Simulation top module name] : シミュレーション中に使用する別のトップ モジュール名を指定します。
  • [Clean up simulation files] : シミュレーション ディレクトリ全体が削除されるので、次の run がきれいなディレクトリで実行されます。
  • [Compilation tab] : Verilog および VHDL ソース ファイルをシミュレーション用にコンパイルするため、シミュレーション ファイルセットのプロパティとして格納され、xvlog および xvhdl ユーティリティで使用されるコンパイラ指示子を定義および管理できます。
    • [Verilog options] : すべての識別子を大文字に設定したりできる [Verilog Options] ダイアログ ボックスを開いて、'include および 'define 値を指定します。
    • [Generics/Parameters options] : VHDL ジェネリック値および Verilog パラメーター値を定義するための [Generics/Parameters] ダイアログ ボックスが開きます。
    • [Properties] : シミュレーション ファイルセットに格納するコンパイラ指示子プロパティが表示されます。表のプロパティを選択すると、そのプロパティの詳細が表示されます。
    重要: Verilog オプションおよびジェネリック/パラメーター オプションは、[Project Settings] ダイアログ ボックスの [General] ページからも定義できます。[General] ページの設定はデザイン ソース ファイルセット (sources_1) に適用されます。 これは合成には影響しますが、シミュレーションには影響しません。
  • [Elaboration tab] : エラボレーション指示子が定義および管理されます。これらの指示子はシミュレーション ファイルセットのプロパティとして格納され、xelab ユーティリティでシミュレーション スナップショットをエラボレートおよび生成する際に使用されます。表のプロパティを選択すると、そのプロパティの詳細が表示され、値を変更できます。
  • [Simulation tab] : シミュレーション指示子が定義および管理されます。これらの指示子はシミュレーション ファイルセットのプロパティとして格納され、xsim アプリケーションで現在のプロジェクトをシミュレーションする際に使用されます。表のプロパティを選択すると、そのプロパティの詳細が表示され、値を変更できます。
  • [Netlist tab] : Verilog ネットリストの SDF アノテーションおよび SDF 遅延で取り込まれるプロセス コーナーに関するネットリスト コンフィギュレーション オプションにアクセスできます。これらのオプションは、シミュレーション ファイルセットのプロパティとして格納され、シミュレーション用ネットリストを書き出す際に使用されます。
  • [Advanced tab] : シミュレーション用のすべてのデザイン ソースが含まれます。このオプションはオンのままにしておきます。
