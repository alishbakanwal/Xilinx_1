[ILA Core Options] ページ

このページでは、Set Up Debug ウィザードで挿入された ILA コアのプロパティを設定できます。

  • [Sample of data depth] : デザインのプローブ用にキャプチャするクロック サイクルのデータ サンプル数を指定します。ILA コアで使用されるデバイス リソースは、サンプルの深さおよびプローブの幅で決まります。
  • [Input pipe stages] : タイミングを管理するために ILA 入力で使用するパイプライン レジスタ数を指定します。パイプライン レジスタは、ILA コアをクリティカル タイミング パス外に移動するために使用できます。これにより、ILA コアで使用されるリソースは増えます。
  • [Capture control] : データをキャプチャするために ILA コアで満たす必要のある条件を指定します。キャプチャ モードのコンフィギュレーションに関する詳細は、『Vivado Design Suite ユーザー ガイド : プログラムおよびデバッグ』 (UG908) のこのセクションを参照してください。
  • [Advanced trigger] : Vivado ハードウェア マネージャーで ILA コアに指定するアドバンス トリガー メカニズムを指定します。詳細は、『Vivado Design Suite ユーザー ガイド : プログラムおよびデバッグ』 (UG908) のこのセクションを参照してください。
重要: これらの ILA コンフィギュレーション オプションは Set Up Debug ウィザードで作成されたすべての ILA コアに適用されます。
