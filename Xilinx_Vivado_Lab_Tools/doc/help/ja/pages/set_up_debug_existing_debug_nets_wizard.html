[Existing Debug Nets] ページ

Set Up Debug ウィザードの [Existing Debug Nets] ページでは、デザインに既存のデバッグ ネットがある場合に実行する動作を指定できます。

ヒント: このページは、現在のデザインに既存のデバッグ ネットがある場合にのみ開きます。

使用できる動作は、次のとおりです。

  • [Continue debugging ## nets connected to existing debug core] : 既存の ILA コアとネット割り当てが保持され、デバッグする新しいネットを追加できます。
  • [Only debug new nets] : 既存の ILA コアとネット割り当てが削除され、デバッグする新しいネットを追加できます。
  • [Disconnect all nets and remove debug cores] : 既存の ILA コアとネット割り当てが削除され、デバッグする新しいネットも追加できません。
