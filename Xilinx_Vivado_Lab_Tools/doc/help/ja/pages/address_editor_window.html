[Address Editor] ウィンドウ

[Address Editor] ウィンドウには、マスター インターフェイスとそのマスターによりアドレス指定可能な接続ペリフェラル (スレーブ) のツリー ビューがリストされます。ブロック デザインのスレーブおよびマスター インターフェイスのメモリ アドレス範囲は自動的に割り当てられますが、このウィンドウで編集できます。

[Address Editor] ウィンドウのツリー ビューには、次の列が表示されます。
  • [Cell] : アドレス空間マスターとそのマスターでアドレス指定されたペリフェラル インターフェイスがリストされます。
  • [Slave Interface] : ペリフェラル インスタンスのスレーブ インターフェイス ピンの名前がリストされます。
  • [Base Name] : メモリのタイプ Mem または Reg が表示されます。
  • [Offset Address] : アドレス ブロックの開始からのメモリ オフセットがリストされます。
  • [Range] : アドレス空間に割り当てられるメモリのサイズまたは容量がリストされます。
  • [High Address] : オフセットおよび範囲に基づいて計算されたアドレスが表示されます。
アドレス オフセットと範囲は手動で割り当てたり、[Auto Assign Address] コマンド () で自動的に割り当てることができます。
ヒント: 複数のマスターが同じスレーブをアドレス指定している場合は、マスターのアドレス指定可能なメモリ空間からスレーブを除外できます。
