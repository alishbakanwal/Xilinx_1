[Export I/O Ports] ダイアログ ボックス

このダイアログ ボックスを使用すると、現在のデザインの I/O ポート割り当てを任意の数の出力ファイル形式にエクスポートできます。

  • [CSV] : サードパーティ デザインおよび解析ツールで使用可能な CSV (Comma Separated Value) ファイルが出力されます。詳細は、『Vivado® Design Suite ユーザー ガイド : I/O およびクロック プランニング』 (UG899) のこのセクションを参照してください。
  • [XDC] : IO 規格、方向、ドライブ、スキュー、およびパッケージ ピンの割り当てなど、I/O ポートの特性を定義する制約が出力されます。
  • [Verilog] : I/O 割り当てに基づいて、デザインの最上位レベルの Verilog スタブ ファイルが出力されます。
  • [VHDL] : I/O 割り当てに基づいて、デザインの最上位レベルの VHDL スタブ ファイルが出力されます。
