[Configure Project] ページ

[Configure Project] ページでは、Vivado IP インテグレーターのブロック デザイン例に追加された MicroBlaze プロセッサと IP を設定できます。

  • [Microblaze] : プロセッサのエレメントを設定します。
    • [Local Memory] : プロセッサ専用のメモリ容量を指定します。
    • [DDR4] : デザインに DDR4 Memory Interface IP が追加されます。
      ヒント: DDR4 スイッチは、イーサネット コントローラーのデータおよび命令キャッシュと DMA アクセス モードに影響します。
    • [Cache] : データ キャッシュと命令キャッシュの両方のサイズを指定します。DDR4 がオフになっている場合は、このオプションもオフになります。
    • [Ethernet] : FIFO または DMA を使用した AXI Ethernet IP がブロック デザインに追加されます。デフォルト モードは FIFO です。DDR4 がオンになっている場合は、DMA モードを使用できます。
  • [General Purpose IOs] : 指定した I/O インターフェイスの IP がブロック デザイン例に追加されます。UART IP を設定することもできます。
