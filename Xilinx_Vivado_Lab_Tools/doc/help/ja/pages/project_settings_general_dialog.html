[Project Settings] ダイアログ ボックス : [General] ページ

[Project Settings] ダイアログ ボックスでは、デザイン フローを通して使用されるさまざまな設定に対して値を指定できます。設定ページは、合成、シミュレーション、インプリメンテーションなどのデザイン フローの手順別に分類されています。

ヒント: これは、現在のプロジェクトではなく、Vivado® ツール全体の設定を指定する [Vivado Options] ダイアログ ボックスと同様です。

[General] ページでは、デフォルトのパーツおよび言語に基づいた設定を変更できます。

  • [Project device] : パーツ ブラウザーを使用してデフォルトのパーツまたはボードを変更します。
  • [Target language] : 生成したファイルに使用する言語を指定します。
  • [Default library] : 明確なライブラリ指定のないファイルはすべて (Verilog ファイルでも) このライブラリでコンパイルされます。
  • [Top module name] : プロジェクトの最上位モジュールを指定します。この値は、階層アップデート モードが自動に [Sources] ウィンドウで設定されていても保持されます。
  • [Language Options] : Verilog および VHDL オプションの値を定義します。
    • [Verilog options] : すべての識別子を大文字に設定したりできる [Verilog Options] ダイアログ ボックスを開いて、'include および 'define 値を指定します。
    • [Generics/Parameters options] : [Generics/Parameters] ダイアログ ボックスを開いて、現在のプロジェクトで定義される VHDL ジェネリックおよび Verilog パラメーターの値を指定します。
    • [Loop Count] : エラボレーションに使用する最大ループ カウントを変更できます。
    重要: Verilog オプションおよびジェネリック/パラメーター オプションは、[Project Settings] ダイアログ ボックスの [Simulation] ページからも定義できます。シミュレーション設定は、シミュレーション ファイルセットに適用されます。 シミュレーションには影響しますが、合成には影響しません。
