[Project Settings] ダイアログ ボックス : [Synthesis] ページ

[Project Settings] ダイアログ ボックスでは、デザイン フローを通して使用されるさまざまな設定に対して値を指定できます。設定ページは、合成、シミュレーション、インプリメンテーションなどのデザイン フローの手順別に分類されています。

[Synthesis] ページでは、制約セットおよびストラテジを定義したり、アクティブな合成 run のオプションをカスタマイズしたりできます。選択した合成ストラテジのデフォルト オプションは表形式で表示され、カスタマイズするストラテジのオプションや値が変更できるようになっています。

  • [Default constraint set] : 合成中に使用する制約セットを指定します。新しい制約セットは、[Sources] ウィンドウの右クリック メニューの [Edit Constraints Set] コマンドを使用するか、Add Sources ウィザードを使用すると作成できます。
  • [Strategy] : 定義済み Vivado® Design Suite ストラテジを指定するか、選択したストラテジのオプションを変更して独自のストラテジを作成できます。
    ヒント: 選択したストラテジのこのオプションを変更した場合は、[Save Strategy As] ボタン () をクリックすると、それらのオプションを新しいユーザー定義のストラテジとして保存できます。
  • [Description] : 選択したストラテジの目的のサマリを簡単に記述しておきます。この部分は、ユーザー定義のストラテジを保存する際に定義できます。
  • [Synth Design options] : 選択した合成ストラテジで使用可能なオプションが表形式で表示されます。表のオプションを選択すると、そのオプションの詳細が表示されます。
