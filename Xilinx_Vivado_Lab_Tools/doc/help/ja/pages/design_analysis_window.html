[Design Analysis Results] ウィンドウ

[Design Analysis Results] ウィンドウには、[Report Design Analysis] コマンドの結果が表示されます。このウィンドウは、2 つのセクションに分割されます。左側には、デザインと report コマンドに関する一般情報と、レポートされるタイミング パスのセットアップおよびホールド パス特性などの目次が表示されます。

デザイン解析結果は右側に表形式で表示されます。選択したオプションによって、レポートで実行されたタイミング パス、複雑性解析、密集解析などが表示されます。列のヘッダーを右クリックすると、列を表示および非表示にするコマンドと列幅を管理するコマンドが表示されます。表の情報は、列ヘッダーを 1 回または 2 回クリックすると並び替えることができます。表の詳細は、『Vivado® Design Suite ユーザー ガイド : Vivado IDE の使用』 (UG893) のこのセクションを参照してください。

[Design Analysis Results] ウィンドウには、左側に表形式のツール バーが、右側に結果の表が表示されます。

  • 左側の表形式のツールバーには、次のコマンドが含まれます。
    Search 検索パターンを入力する検索フィールドが開きます。
    Collapse all [Design Analysis Results] ウィンドウの左側の階層の展開が閉じられます。
    Expand all [Design Analysis Results] ウィンドウの左側の階層が展開されます。
    Previous [Design Analysis Results] ウィンドウの前の表示が復元されます。
    Next [Previous] で表示していた前の表示が元の表示に戻されます。
    ヒント: このコマンドは、[Previous] コマンドを使用した後にのみ使用可能です。
    Rerun [Report Design Analysis] コマンドが再実行されます。
    Show only sections with content オンまたはオフにして、内容や結果のないデザイン解析レポートのセクションを表示または非表示にします。
  • 結果の表のツールバーには、次のコマンドが含まれます。
    Search 検索パターンを入力する検索フィールドが開きます。検索パターンに合うように内容がフィルターされ、結果の表に表示されます。
    Schematic [Schematic] ウィンドウが開き、現在選択しているオブジェクトが表示されます。
    Mark Selected Objects 結果の表の選択オブジェクトが [Device] および [Netlist] ウィンドウなどのほかのウィンドウでマークされます。
    ヒント: タイミング パスの始点は緑色、終点は赤色、通過点は選択した色で表示されます。
    Show Percentage 結果の表の値が値ではなく % で表示されます。
    Select Clock Paths when Timing Paths are selected ソースおよびデスティネーション クロック パスをデータパスと一緒に選択します。
    Hide Unused 結果の表の空または未使用の列の表示を切り替えることができます。
