デザイン キャンバス ウィンドウ

Vivado® IP Pインテグレーターには、ブロック デザインを構築するために使用できるデザイン キャンバスが表示されます。デザイン キャンバス ウィンドウにはブロック デザインが表示され、デザインを編集できます。

ツールバー メニューには、ブロック デザインを編集するために次のコマンドが含まれます。

Add IP Vivado IP カタログから IP を追加します。
Make External ピンを外部ポートに接続します。
Customize Block 現在選択している IP の IP カスタマイズ用ダイアログ ボックスを開きます。選択した IP をダブルクリックしても IP をカスタマイズし直すことができます。
Validate Design ブロック デザインで統合デザイン ルール チェックを実行し、発生する可能性のある問題が検出されます。
Regenerate Layout ブロック デザイン全体を再描画してデザインがクリーンアップされます。
Optimize Routing 接続を再配線して、デザインがクリーンアップされます。
Show Interface Connections Only インターフェイス ピンとポート間の接続のみが表示され、その他すべての接続は非表示になります。

デザイン キャンバス ウィンドウは拡大/縮小して、ブロック デザインで必要な詳細レベルを表示できます。ツールバー メニューには、何を表示するかを制御するための次のコマンドが含まれます。

Zoom In ブロック デザインの詳細が拡大されて表示されます。
Zoom Out ブロック デザインのエリアがより多く表示されます。
Zoom Fit ブロック デザイン全体が表示されます。
Select Area デザイン キャンバス ウィンドウで長方形を描画してオブジェクトを選択します。
Fit Selection 現在選択しているオブジェクトがデザイン キャンバス ウィンドウの中央に拡大表示されます。
Autofit Selection 別のウィンドウで選択したオブジェクトがデザイン キャンバス ウィンドウの中央に自動的に拡大表示されます。
デザイン キャンバス ウィンドウでは、クリックによる表示の拡大/縮小機能がサポートされています。クリックしてドラッグしながら線やボックスを描くと、ブロック デザインを拡大または縮小表示できます。デザイン キャンバス ウィンドウの右側または下側のスクロールバーを使用すると、画面をスクロールして新しいエリアを表示できます。
ヒント: スクロールバーは、表示エリアがブロック デザインで拡大された場合にのみ表示されます。

ツールバーには、IP カタログをコンフィギュレーションしたり、デザイン キャンバス ウィンドウに表示するものを指定したりするコマンドも含まれます。

Block Design Options ブロック デザインのオブジェクトの表示オプションおよび色をコンフィギュレーションできます。詳細は、『Vivado Design Suite ユーザー ガイド : IP インテグレーターを使用した IP サブシステムの設計』 (UG994) のこのセクションを参照してください。
IP Settings [Project Settings] ダイアログ ボックスの [IP] ページを開き、現在のプロジェクトの IP 関連のツール設定をコンフィギュレーションできます。
