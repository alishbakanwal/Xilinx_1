テキスト エディター

テキスト エディターは、HDL ソース ファイル、XDC 制約ファイル、Tcl スクリプト、およびその他のテキスト ベースのファイルを編集する環境です。テキスト エディターでは、特定のファイル形式 (Verilog、SystemVerilog、VHDL、SDC、XDC、および Tcl) の言語構文が認識され、ファイルを編集中にその言語のキーワードおよび識別子がハイライトされます。認識されたファイル形式の場合は、リアルタイムの構文チェックも実行されるので、エラーを見つけて修正しやすくなります。
ヒント: 言語構文が認識されない場合は、テキスト エディターでそのファイルが単純なテキスト ファイルとして処理されます。

テキスト エディターには、タブおよびスペースを管理する方法を設定できる右クリック メニューがあるほか、階層の言語構文のコードを折り返す機能があり、Verilog モジュール、VHDL エンティティ、および Tcl proc の展開を閉じたり、開いたりできます。ファイルを変更した場合は、メモリ内のファイルとディスクに保存されたファイルが比較され、ファイルを保存する前にどのような変更があったかが確認されます。

テキスト エディターのツールバーには、[Copy] ()、[Cut] ()、[Paste] ()、[Delete] () などのファイルを編集する一般的なコマンドが含まれます。[Save] () コマンドでは変更を編集したテキスト ファイルに保存でき、[Undo] () および [Redo] () コマンドで必要に応じて変更を取り消したり、取り消した変更を元に戻したりできます。

また、テキスト エディターのツールバーには Verilog、VHDL、Tcl および XDC ファイルなどのコード ファイルを編集するためのコマンドが含まれます。

Toggle Line Comments コードの行をコメントに変更したり、1 行のコメントをテキスト行にすばやく変更したりできます。テキスト エディターでは、編集されるファイルの言語構文に基づいて、適切な文字列が挿入または削除され、1 行のコメントが作成または削除されます。
  • // for Verilog and SystemVerilog
  • -- for VHDL
  • # for Tcl and XDC
Toggle Column Selection Mode 各行の文字を選択するのではなく、列および行に基づいてグリッド ベースで文字を選択できるようになります。
Find テキスト エディターの一番下に [Find] フィールドが表示されるようになり、現在のテキスト ファイルを検索できるようになります。
Language Templates [Language Template] ダイアログ ボックスを開いて、前もって定義済みの Verilog、SystemVerilog、VHDL、XDC デザイン テンプレートのコレクションから参照および選択できるようにします。
Insert Template [Language Template] ダイアログ ボックスで現在選択されているテンプレートがテキスト ファイルのカーソル位置に追加されます。
注: 選択したテンプレートを挿入するために [Language Template] ダイアログ ボックスを開く必要はありません。

テキスト エディターは、Vivado® IDE のワークスペースに表示されます。ワークスペースの詳細は、『Vivado Design Suite ユーザー ガイド : Vivado IDE の使用』 (UG893) のこのセクションを参照してください。
