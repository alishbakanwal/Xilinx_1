[Specify Path Groups] ダイアログ ボックス

[Specify Path Groups] ダイアログ ボックスでは、タイミング解析に使用するパス グループを現在定義されたパス グループから選択できます。

  • [Pattern] : 現在定義されているパス グループを見つけるための検索パターンを指定します。ワイルドカードとしてクエスチョン マーク (?) およびアスタリスク (*) 文字を使用できます。
  • [Regular Expressions] : [Pattern] フィールドを正規表現 (regexp) 検索パターンとして定義します。正規表現構文の詳細については、このリンクを参照してください。
  • [Ignore case] : 正規表現の検索にのみ適用されます。大文字と小文字は区別されません。通常は、検索パターンの大文字/小文字は区別されます。
  • [Find] : [Pattern] フィールドで定義した検索が実行されます。検索結果は、[Results] セクションの [Found] フィールドに表示されます。
  • [Results] :
    • [Found] : [Pattern] で見つかったオブジェクトの数が示され、オブジェクトがリストされます。
    • 次のツールバー ボタンを使用すると、[Found] と [Selected] リスト ボックス間でオブジェクトを移動できます。
      Move the selected items to the right 選択したオブジェクトが [Found] から [Selected] に移動されます。
      Move the selected items to the left 選択したオブジェクトが [Selected] から [Found] に移動されます。
      Move all the items to the right すべてのオブジェクトが [Found] から [Selected] に移動されます。
      Move all the items to the left すべてのオブジェクトが [Selected] から [Found] に移動されます。
    • [Selected] : 見つかったアイテムから現在選択されているオブジェクトの数が示され、そのオブジェクトがリストされます。選択したアイテムにより、タイミング パスの始点が定義されます。
    • 次のツールバー ボタンを使用すると、選択したオブジェクトを並べ替えることができます。
      Sort 選択したオブジェクトがアルファベット順に並べ替えられます。1 度クリックすると昇順で、もう 1 度クリックすると降順になります。
      Move the selected items up 現在選択しているオブジェクトがリストの 1 つ上に移動されます。
      Move the selected items down 現在選択しているオブジェクトがリストの 1 つ下に移動されます。
      Move the selected items to the top 現在選択しているオブジェクトがリストの一番上に移動されます。
      Move the selected items to the bottom 現在選択しているオブジェクトがリストの一番下に移動されます。
  • [Command] : 指定したパス グループ オブジェクトを取得する Tcl コマンドが表示されます。表示されたコマンドを使用すると、Vivado® IDE で使用される Tcl コマンドを作成しやすくなります。
  • [Enumerate] : Tcl コマンドに選択したパス グループ オブジェクトを個別に列挙でき、ワイルドカードを使用する必要をなくします。

[OK] をクリックすると、[Report Timing] ダイアログ ボックスに Tcl コマンドが追加されます。

[Append] をクリックすると、Tcl コマンドが既存の始点コマンドに追加されます。
