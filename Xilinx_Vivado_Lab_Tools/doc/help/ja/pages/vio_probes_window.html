[VIO Probes] ウィンドウ

[VIO Probes] ウィンドウには、現在のハードウェア デバイスにプログラムされる VIO デバッグ コアから選択したプローブのリストが表示されます。VIO プローブを追加した場合、このウィンドウを使用して入力プローブ信号の動作を監視するか、出力プローブ信号を指定したステートにできます。

VIO コアの入力プローブは、現在のハードウェア デバイスで実行されているデザインからの値を読み出すために使用されます。VIO 入力プローブは通常、テスト中のデザインのステータス インジケーターとして使用されます。VIO 入力プローブの値は、VIO コアのリフレッシュ レートに基づいて定期的にアップデートされます。これは、[VIO Properties] ウィンドウの [Refresh Rate (ms)] で変更できます。

VIO コアの出力プローブは、現在のハードウェア デバイスで実行されているデザインからの値を書き込むために使用されます。VIO 出力プローブは通常、テスト中のデザインの低帯域幅の制御信号として使用されます。出力値は、[VIO Probes] ウィンドウに新しい値が入力されるとアップデートされます。

[VIO Probes] ウィンドウのツールバーには、次のコマンドが含まれます。

Search 特定のプローブが表示されるように、検索パターンを入力するテキスト フィールドが開きます。
Collapse all すべてのバス プローブの展開が閉じられ、プローブの個別ビットが非表示になります。
Expand all すべてのバス プローブが展開され、プローブの個別ビットが表示されます。
Add probes VIO デバッグ コアに関連するプローブがリストされた [Add Probes] ダイアログ ボックスが開きます。プローブを選択して [OK] をクリックすると、ウィンドウにそのプローブが追加されます。
Remove selected probes [VIO Probes] ウィンドウから選択したプローブが削除されます。
