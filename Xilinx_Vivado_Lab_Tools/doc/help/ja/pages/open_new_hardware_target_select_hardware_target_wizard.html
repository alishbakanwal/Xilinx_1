ハードウェア サーバーに接続したら、ハードウェア マネージャーでハードウェア プログラマが検証されるか、テスト ボードの JTAG チェーンで使用可能なターゲットが見つけられます。結果は、[Hardware Targets] の表に表示されます。ハードウェア マネージャーで開くハードウェア ターゲットを選択します。

[Hardware Targets] : ハードウェア プログラマの JTAG チェーンで使用可能なターゲット、またはハードウェア サーバーに接続されたテスト ボードがリストされます。スキャン チェーンのターゲットの順番は、チェーンから ID コードが読み出され、そのコードがデバイス名や命令レジスタ (IR) 長などのデバイス プロパティとマッチされることで決まります。
  • [Type] : JTAG チェーンのデバイスのタイプが表示されます。
  • [Name] : JTAG チェーンのデバイスの名前が表示されます。
  • [JTAG Clock Frequency] : デフォルトの JTAG クロック周波数 (または Digilent ケーブル接続の場合は 15MHz、USB ケーブル接続の場合は 6MHz) を使用する必要があります。デフォルトの速度でターゲットに接続できない場合は、『Vivado Design Suite ユーザー ガイド : プログラムおよびデバッグ』 (UG908) のこのセクションに記述されているように、JTAG クロック周波数を減らすことができます。

[Hardware Devices] : 現在選択しているハードウェア ターゲットの名前、ID コード、および IR 長が表示されます。

[Next] をクリックしてウィザードの [Open Hardware Target Summary] ページを表示します。
