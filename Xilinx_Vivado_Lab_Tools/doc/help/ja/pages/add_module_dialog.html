[Add Module] ダイアログ ボックス

[Add Module] ダイアログ ボックスでは、モジュールをリファレンスとして現在のブロック デザインに追加できます。これには、モジュールを定義するソース ファイルが既にデザイン ソースの最上位として現在のプロジェクトに追加されている必要があります。Add Sources ウィザードを使用すると、HDL ソースをプロジェクトに追加できます。

重要: サポートされるファイル形式は、VHDL および Verilog です。

リスト ボックスには、ブロック デザインに追加できるモジュールが表示されます。検索フィールドには、リストされるモジュールをフィルターするための検索パターンを入力できます。

モジュールを選択して [OK] をクリックすると、そのモジュールがブロック デザインに追加されます。
