[Run Block Automation] ダイアログ ボックス

[Run Block Automation] リンクは、MicroBlaze および Zynq などの IP や一部の階層 IP を IP インテグレーターに含めると、キャンバスのバナー部分に表示されます。この設計アシスタンス機能では、該当する IP に必要な基本的な接続が設定されて実行されます。

[Run Block Automation] ダイアログ ボックスには、接続可能なブロックがツリー ビューで表示されます。ブロックの横のチェック ボックスをオン/オフにすると、設計アシスタンス機能をオン/オフにできます。ツリー ビューでクロックをクリックして選択すると、その選択したブロック用にコンフィギュレーションできる接続オプションが表示されます。また、そのブロックに実行されるオートメーション プロセスの簡単な説明も表示されます。使用可能な接続オプションは、選択した IP ブロックによって異なります。

[OK] をクリックすると、選択したオプションでブロック オートメーションが実行されます。
