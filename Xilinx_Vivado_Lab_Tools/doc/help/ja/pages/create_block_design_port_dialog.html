[Create Block Design Port] ダイアログ ボックス

[Create Block Design Port] ダイアログ ボックスでは、現在のブロック デザインに追加する新しい外部ポートを定義します。

  • [Port name] : 作成するポートの名前を指定します。
  • [Direction] : 入力、出力、双方向ポートのいずれかとして、ポートを定義します。
  • [Type] : ポートを通る信号のタイプを指定します。タイプは、クロック、リセット、割り込み、データ、クロック イネーブル、またはその他のいずれかにできます。
  • [Create vector] : バス ポートとしてポートを定義し、ポートの幅を定義します。
  • [Frequency (MHz)] : このオプションは、入力がクロック ポートとしてコンフィギュレーションされた場合にのみ使用できます。クロック ポートの周波数をメガヘルツで指定します。
  • [Polarity] : このオプションは、入力ポートがリセットとしてコンフィギュレーションされた場合にのみ使用できます。リセット ポートは、アクティブ High かアクティブ Low リセットとして指定できます。
  • [Interrupt type] : このオプションは、入力ポートが割り込みとしてコンフィギュレーションされた場合にのみ使用できます。割り込みポートをエッジ タイプかレベル タイプの割り込みとして定義します。
  • [Sensitivity] : このオプションは、入力ポートが割り込みとしてコンフィギュレーションされた場合にのみ使用できます。レベル タイプの割り込みのセンシティビティを High か Low に定義したり、エッジ タイプの割り込みのセンシティビティを立ち上がりエッジか立ち下がりエッジに定義できます。
  • [Connect to matching selected ports] : 新しいポートがその新しいポートと互換性のある選択した互換性のあるポートまたはピンに接続されます。
    ヒント: このチェック ボックスは、[Create Port] コマンドを実行する際にデザインのその他のピンまたはポートが使用可能な場合にオンになります。

[OK] をクリックすると、新しいポートが作成され、選択した ポートにオプションで接続されます。
