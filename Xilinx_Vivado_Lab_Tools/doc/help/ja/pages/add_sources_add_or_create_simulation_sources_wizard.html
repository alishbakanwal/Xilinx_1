[Add or Create Simulation Sources] ページ

[Add or Create Simulation Sources] ページでは、既存のシミュレーション ファイルセットを指定したり、シミュレーション ソース (デザインのテストベンチまたはデザインのモジュールなど) を追加する新しいシミュレーション ファイルセットを作成したりできます。サポートされるシミュレーション ソース ファイルは、Ｖerilog、Verilog ヘッダー、SystemVerilog、VHDL、および メモリ ファイル形式です。

  • [Specify simulation set] : 追加されるシミュレーション ファイルのデフォルト ファイルセットは現在アクティブのシミュレーション ファイルセットになりますが、別の既存 sim_set を選択したり、プロジェクト用に新しいシミュレーション ファイルセット作成することもできます。[Create Simulation Set] をクリックすると [Create Simulation Set ] ダイアログ ボックスが開き、新しいシミュレーション ファイルセットの名前を指定できます。
    • [Make Active] : 新しいシミュレーション ファイルセットを作成した場合は、このチェック ボックスをオンにすると、プロジェクトのアクティブ sim_set に設定できます。
  • [Add Files] : ファイル参照ダイアログ ボックスが開き、プロジェクトに含める既存のシミュレーション ソース ファイルを任意の数だけ選択できます。複数のファイルを選択した場合は、指定した順に追加されます。
    ヒント: をクリックして [Add Files] をクリックしても同じ操作を実行できます。
  • [Add Directories] : ファイル参照ダイアログ ボックスが開き、シミュレーション ソース ファイルを検索するディレクトリを任意の数だけ選択できます。指定したディレクトリ内でシミュレーション ソース ファイルが検索されて現在のプロジェクトに追加されます。
    ヒント: をクリックして [Add Directories] をクリックしても同じ操作を実行できます。
  • [Create File] : [Create Source File] ダイアログ ボックスが開き、ファイル タイプ、ファイル名、ファイルを保存するディレクトリを指定できます。空のソース ファイルが作成されて、プロジェクトに追加されます。新規作成したソース ファイルは、Add Sources ウィザードの終了時にモジュール エディターで開きます。詳細は、[Define Module] ダイアログ ボックスまたは『Vivado® Design Suite ユーザー ガイド : システム レベル デザイン入力』 (UG895) のこのセクションを参照してください。
    ヒント: をクリックして [Create Files] をクリックしても同じ操作を実行できます。
  • [Remove] () : 現在のプロジェクトから選択したファイルを削除します。ファイルは現在のプロジェクトから削除されますが、ハード ドライブからは削除されません。
  • [Move Up] : ファイル リストでファイルが選択されている場合、青色の上方向矢印ボタン () が使用できるようになり、リスト内でファイルを上に移動できます。シミュレーション ソース ファイルはデザインに読み込まれて、この順序でコンパイルされます。コンパイルの順序および [Hierarch Update] コマンドの詳細は、『Vivado Design Suite ユーザー ガイド : Vivado IDE の使用』 (UG893) のこのセクションを参照してください。
  • [Move Down] : ファイル リストでファイルが選択されている場合、左側のメニューから青色の下方向矢印ボタン () が使用できるようになり、リスト内でファイルを下に移動できます。シミュレーション ソース ファイルはデザインに読み込まれて、この順序でコンパイルされます。
  • [Scan and add RTL include files into project] : オンにすると追加した RTL ファイルがスキャンされます。また、参照される Verilog のインクルード ファイルがローカルのプロジェクト ディレクトリに追加されます。
  • [Copy sources into project] : ローカル プロジェクト ディレクトリの sources フォルダーに指定したデザイン ソース ファイルをコピーできます。ローカルおよびリモート ソースの使用に関する詳細は、『Vivado® Design Suite ユーザー ガイド : システム レベル デザイン入力』 (UG895) のこのセクションを参照してください。
  • [Add sources from subdirectories] : [Add Directories] で指定したディレクトリのサブディレクトリに含まれるソース ファイルがすべて追加されます。
  • [Include all design sources for simulation] : すべてのデザイン ソース ファイルをシミュレーション ファイルセットに含め、サードパーティ シミュレータで使用するためにデザインをエクスポートできるようにします。
