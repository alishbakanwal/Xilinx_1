[Report I/O] ダイアログ ボックス

このダイアログ ボックスを使用すると、現在のデザインの I/O 割り当てがレポートできます。レポートには、現在のデザインの I/O バンクとデバイスの各ピンの詳細が含まれるほか、ターゲット パーツ、パッケージ、スピード グレードなどのデバイス特有の情報も含まれます。

  • [Results name] : Vivado IDE の [Report I/O] ウィンドウのレポートに適用する名前を指定します。
  • [Export to file] : I/O レポートの結果が指定ファイルに書き込まれます。ファイル形式はテキストまたは XML のいずれかとして指定できます。
  • [Open in a new tab] : Vivado IDE の [Report I/O] ウィンドウに新しいタブで I/O レポートが開きます。オフの場合は、新しいレポートで最後に開いたレポートが上書きされます。
