[Verilog Options] ダイアログ ボックス

このダイアログ ボックスでは、次のフィールドを定義できます。

  • [Verilog Include Files Search Paths] : このフィールドでは、Verilog `include 文の追加の検索パスを定義およびリストできます。このフィールドのツールバー メニューには、次のコマンドが含まれます。
    Add 新しく含める検索パスを指定するためのファイル ブラウザーが開きます。
    Remove 選択したパスが削除されます。
    Move up 選択したパスが表示リストの上の方へ移動されます。
    Move down 選択したパスが表示リストの下の方へ移動されます。
  • [Defines] : Verilog `define 文に名前/値のペアを指定します。このフィールドのツールバー メニューには、次のコマンドが含まれます。
    Add 名前/値のペアを指定するための [Add Value] ダイアログ ボックスが開きます。
    Remove 指定した名前/値のペアが削除されます。
    Edit 選択した名前/値のペアを編集するための [Edit Value] ダイアログ ボックスが開きます。
  • [Uppercase all identifiers] : デザイン内のすべての識別子を大文字に変更します。これはインメモリ デザインに適用されます。 HDL ソース ファイルには影響しません。

[OK] をクリックすると、[Simulation] 設定にオプションが追加され、ダイアログ ボックスが閉じます。
