[Device Constraints] ウィンドウ

[Device Constraints] ウィンドウには、デザインの I/O バンクの電圧レベル (VREF) を定義するモードと、デザインの DCI_CASCADE チェーンを定義する 2 つのモードがあります。ツールバー メニューすぐ下のドロップダウン メニューから、[Internal VREF] モードか [DCI Cascade] モードのいずれかを選択します。

[Internal VREF] モード

[Internal VREF] モードでは、デバイスの I/O バンクが表示され、適切な電圧レベルにそれらをドラッグ アンド ドロップして、内部電圧リファレンスを I/O バンクに割り当てることができます。最初は、すべての I/O バンクが [NONE] ヘディングの下にリストされ、割り当てられた VREF がないことが示されます。表示された I/O バンクの 1 つを選択し、それを特定の内部リファレンス電圧にドラッグします。これで選択した I/O バンクに INTERNAL_VREF プロパティが割り当てられます。

[DCI Cascade] モード

[DCI Cascade] モードでは、DCI カスケード チェーンに割り当てるため、2 つ以上のハイ パフォーマンス (HP) I/O バンクを選択できます。これで選択した I/O バンクに DCI_CASCADE プロパティが割り当てられます。2 つ以上の I/O バンクを選択して [Add] ボタン () をクリックすると、これらをチェーンとして定義できます。[Add DCI Cascade] ダイアログ ボックスが開いて、カスケード チェーンのマスター I/O バンクを定義できます。

各モードにはそれぞれツールバーもあり、次のコマンドの 1 つまたは複数の次のコマンドが実行できるようになっています。

Search 検索パターンを入力する検索フィールドが開きます。
Expand all すべての VREF または DCI カスケードのカテゴリが展開されます。
Collapse all すべての VREF または DCI カスケードのカテゴリの展開が閉じられます。
Scroll 選択したオブジェクトまで表示がスクロールされます。
Add DCI カスケードの規則が追加されます。このコマンドを使用する前に、2 つ以上の I/O バンクを選択しておく必要があります。
Remove 選択した VREF または DCI カスケードの規則が削除されます。
