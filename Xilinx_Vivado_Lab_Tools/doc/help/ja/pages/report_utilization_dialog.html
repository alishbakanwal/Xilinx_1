[Report Utilization] ダイアログ ボックス

このダイアログ ボックスでは、合成済みまたはインプリメント済みデザイン ネットリストからのデバイス リソースの使用率のレポートが生成できます。

  • [Results name] : 結果を Vivado® IDE で開いたときに、レポートに付ける名前を指定します。
  • [Export to file] : 作成する使用量レポートのパスと名前を指定します。作成するファイルのディレクトリと名前を指定するのに、ファイル参照ボタン () を使用できます。
  • [Open in a new tab] : Vivado IDE のウィンドウに新しいタブで使用量レポートが開きます。オフの場合は、新しいレポートで最後に開いたレポートが上書きされます。
