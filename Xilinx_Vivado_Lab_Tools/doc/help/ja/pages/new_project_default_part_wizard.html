[Default Part] ページ

現在のプロジェクトで使用するデフォルト パーツまたはプラットフォーム ボードを選択します。デザインで使用可能なデバイス リソースは、選択するターゲット パーツまたはボードにより異なります。

  • 次のいずれかを実行します。[Parts] | [Boards] : 現在のプロジェクトで使用可能なターゲット パーツまたはボードがリストされます。
    重要: 使用可能なデバイスおよびボードは、Vivado Design Suite がインストールされた時点で決まっています。アンインストールしたパーツは、メイン メニューから [Help] → [Add Design Tools or Devices] をクリックすると追加できます。
  • [Filter] : さまざまな条件で表示されているパーツまたはボードをフィルターできます。このフィールドは [Filter] をクリックすると非表示にできます。
  • [Reset All Filters] : 現在選択されているパーツまたはボードのフィルター条件をクリアします。
  • [Search] : 表に示されているパーツまたはボードを指定した検索文字列に一致するもののみに絞りこむことができます。

指定したフィルターおよび検索文字列に一致したターゲット パーツまたはボードのみがページ下部の表に表示されます。この表には、デバイス名、I/O ピン数、ブロック RAM 数、および DSP 数が示されています。結果の表を並び替えるには、列ヘッダーをクリックします。複数の列を並び替えるには、Ctrl キーを押しながら列ヘッダーをクリックしていきます。1 回クリックした場合は昇順に、２ 回クリックした場合は降順に列が並べ替えられます。

デザインのターゲット パーツまたはボードを選択したら [Next] をクリックして、[New Project Summary] ページに進みます。
