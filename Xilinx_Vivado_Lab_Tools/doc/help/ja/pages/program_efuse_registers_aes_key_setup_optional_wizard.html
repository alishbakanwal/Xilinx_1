[AES Key Setup] ページ (オプション)

7 シリーズ デバイスの場合、Program eFUSE Registers ウィザードのこのページでは、現在のデバイスの FUSE_KEY および FUSE_USER レジスタをプログラムできます。AES (Advanced Encryption Standard) 暗号キーは、指定したキー ファイルから読み出され、256 ビットの FUSE_KEY レジスタに書き込まれます。ビットストリームの暗号解読は、デバイスをプログラム、検証、読み戻す際に有効になります。HEX 値を指定して 32 ビットの FUSE_USER レジスタに割り当てることもできます。

インプリメント済みデザインには、write_bitstream コマンドを使用してビットストリームを生成する前に、暗号化 (ENCRYPTION) プロパティを適用しておく必要があります。ENCRYPTION プロパティは、Vivado IDE で [Tools] → [Edit Device Properties] をクリックし、[Edit Device Properties] ダイアログ ボックスの [Encryption] ページから追加できます。

  • [Enable AES key programming] :
    • [AES key file (.nky)] : AES 暗号キー ファイルのファイル名とパスを指定します。
      ヒント: デフォルトでは、このファイルは暗号化されたビットストリームの生成時にインプリメンテーション run ディレクトリに書き込まれます。
    • [AES key] : 上記で指定したファイルから読み出された AES 暗号キー値が表示されます。
    • [User Bits [7:0]] : 7 シリーズ デバイスの場合、一度にプログラムするには、AES キーと同時に USER レジスタの下位 8 ビットをプログラムする必要があります。これらのレジスタは別々にプログラムできません。
  • [Enable Programming of User Bits [31:8]] : FUSE_USER レジスタの上位 24 ビットの 16 進数値を入力するテキスト フィールドが表示されます。デザインまたはデバイスのリビジョンの 32 ビット ID を指定できる USER eFUSEs ビットが提供されます。FUSE_USER の下位 8 ビットは 256 ビットの AES (Advanced Encryption Engine) キーと同時にプログラムされます。上位 24 ビットは、AES キーと同時にプログラムできるほか、後でプログラムすることもできます。
暗号化されたビットストリームを使用してデバイスをプログラムするには、まず暗号キーを BBR または eFUSE レジスタにプログラムしてから、その暗号化されたビットストリームをデバイスにプログラムするという 2 段階のプロセスが使用されます。
program_hw_devices -key {efuse} [current_hw_device]
program_hw_device [current_hw_device]
