[DRC Results] ウィンドウ

[DRC Results] ウィンドウには、[Report DRC] ダイアログ ボックスの結果が表示されます。このウィンドウには、実行されるデザイン ルール チェックの違反がリストされ、関連するルール名ごとに分類されます。各違反ごとに、関連するデザイン オブジェクトのリストなど、割り当てられた重要度と違反の詳細が表示されます。違反を選択すると、[Violation Properties] ウィンドウにその詳細が表示され、[Netlist] または [Device] ウィンドウなどのその他のウィンドウで関連するデザイン オブジェクトが選択されます。

[DRC Results] ウィンドウは表形式のウィンドウで、デザイン ルール違反に関するデータが列形式で表示されます。列を表示および非表示にするコマンドと、列幅を管理するコマンドは、列のヘッダーを右クリックすると表示されます。表の情報は、列ヘッダーを 1 回または 2 回クリックすると並び替えることができます。詳細は、『Vivado® Design Suite ユーザー ガイド : Vivado IDE の使用』 (UG893) のこのセクションを参照してください。

[DRC Results] ウィンドウのトップバーには、さまざまな違反の重要度に関連する統計が表示されます。特定の重要度を示すチェック ボックスをオンまたはオフにすると、その違反を表示または非表示にできます。[Show All] および [Hide All] ボタンをクリックすると、すべての違反をすばやく表示または非表示にできます。

[DRC Results] ツールバーには、次のコマンドが含まれます。

Search 検索パターンを入力する検索フィールドが開きます。検索パターンと一致する DRC 違反が表示されます。
Collapse all DRC 違反がルールごとに分かれているときに使用できます。違反の階層を閉じます。
Expand all DRC 違反がルールごとに分かれているときに使用できます。違反の階層を展開します。
Group by Rule DRC 違反をルールごとにグループ分けします。このコマンドを使用しない場合、違反は単に名前別にリストされます。
