[Place Pblocks] ダイアログ ボックス

このダイアログ ボックスでは、選択した Pblock の領域を描画して配置できます。Pblock は解析され、指定したスライス カウントとターゲット使用率に基づいてサイズが変更されます。

  • [Parent block] : 検証する最上位ブロックまたはパーティションを指定します。デフォルトは、ROOT パーティションです。
  • [Pblocks to Place] : デザイン内の Pblock が表形式で表示され、配置する Pblock を選択できます。
    • [Place] : 特定の Pblock を配置の有効/無効を切り替えるチェック ボックスです。
    • [Pblock] : デザイン内の Pblock の名前です。
    • [Utilization] : ターゲット使用率は、デバイスの Pblock のサイズによって異なります。使用率が高いほど、定義されたエリアのデバイス リソースが消費されるので、Pblock は小さくなります。使用率が低いと、Pblock は大きくなります。
      ヒント: まず最初の配置では高い使用率 (100% 以上など) から始めて、デザインのフロアプランを作成し、インプリメンテーションで使用率を下げることをお勧めします。
  • [Set Utilization on All Pblocks] : すべての Pblock のターゲット使用率を指定します。これにより一般的なターゲットが設定されます。これは、各 Pblock ごとに [Utilization] を使用すると上書きできます。

[OK] をクリックし、選択した Pblock の [Device] ウィンドウに領域が自動的に描画されて配置されます。Pblock は配置した後、移動できます。
