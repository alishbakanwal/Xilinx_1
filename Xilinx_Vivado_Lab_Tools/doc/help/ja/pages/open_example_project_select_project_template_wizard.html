Vivado® Design Suite には複数のデザイン テンプレートが含まれており、RTL ソース コードとカタログからの IP、ブロック デザイン、エンベデッド プロセッサなどのさまざまなデザイン エレメントを含むサンプル デザインをすばやく作成できます。デザイン テンプレートでは、さまざまなターゲット パーツまたはプラットフォーム ボードがサポートされています。

  • [Template] : リストされるデザイン テンプレートからテンプレートを選択し、新しいサンプル デザインを作成します。
  • [Description] : デザインの簡単な説明を表示します。デザインを表す図が表示される場合もあります。
