[Objects] ウィンドウ

[Objects] ウィンドウには、[Scope] ウィンドウの現在のスコープまたは選択したスコープで定義されているさまざまな HDL オブジェクトがリストされます。リストされる HDL オブジェクトの横のアイコンは、各オブジェクトのタイプまたはポート モードを示します。このウィンドウにはシミュレーション オブジェクトの名前、値、データ タイプなどがリストされます。

[Objects] ウィンドウの右クリック メニューには、現在のスコープまたは選択したスコープ内のさまざまな HDL オブジェクトを管理するためのコマンドがあります。 たとえば、[Waveform] ウィンドウにオブジェクトを追加したり、オブジェクトに特定の値を付けたり、オブジェクトの基数を変更したりできます。右クリック メニューの [Report Drivers] コマンドを使用すると、駆動信号の名前と値のほか、選択したオブジェクトの現在の値が表示されるので、選択した信号タイプの HDL オブジェクトがデバッグしやすくなります。

[Objects] ウィンドウのツールバーには、表示されたオブジェクトをフィルターおよび管理するための次のコマンドが含まれます。

Search 検索パターンを入力する検索フィールドが開きます。
Input INPUT タイプのオブジェクトの表示または非表示を指定します。
Output OUTPUT タイプのオブジェクトの表示または非表示を指定します。
Inout INOUT タイプのオブジェクトの表示または非表示を指定します。
Internal Signal INTERNAL SIGNAL タイプのオブジェクトの表示または非表示を指定します。
Constant CONSTANT タイプのオブジェクトの表示または非表示を指定します。
Variable VARIABLE タイプのオブジェクトの表示または非表示を指定します。
