[Xilinx Tcl Store] ダイアログ ボックス

ザイリンクス Tcl Store は、Vivado® Design Suite を使用するために設計された Tcl コードまたはアプリのオープン ソース リポジトリです。アプリは、ザイリンクスまたはサードパーティによりリリースされて管理される 1 つまたは複数の Tcl スクリプトのグループまたはコレクションです。ほかの人の共有したアプリをダウンロードおよびインストールしたり、アプリを記述してザイリンクス Tcl Store で共有したりできます。

[Xilinx Tcl Store] ダイアログ ボックスには、インストール可能なアプリがリストされます。 これらはダウンロードして、Vivado ツールで使用できます。このダイアログ ボックスには、2 つのセクションがあります。左側には使用可能なアプリのリストが、右側には現在選択しているアプリの詳細が、アプリの説明とそのアプリに含まれる特定の Tcl プロシージャー (proc) のリストと共に表示されます。

このダイアログ ボックスには、次の機能が含まれます。

  • [All] | [Installed] : 使用可能なアプリのすべてを表示するタブと、システムにインストールされたアプリのみを表示するタブがあります。
  • () [Search] : 検索文字列を入力してリストされるアプリを制限したり、特定のアプリを検索したりできます。
  • [Install] : 現在選択されているアプリをインストールするボタンです。アプリが既にインストールされていれば、そのように表示されます。
    ヒント: アプリをアンインストールするには、アプリを選択して、右側の [Details] フィールドの [Uninstall Application] リンクをクリックします。
  • [Refresh] : 現在のアプリのリストをアップデートするボタンで、オンライン ザイリンクス Tcl Store から使用可能なアプリのリストを読み込み直します。リストはアプリをインストールする前に更新して、最新バージョンを取得するようにしてください。
  • [Support on GitHub] : ザイリンクス Tcl Store のサポート ページへのリンクです。
  • [Details] ビュー : 右側には、現在選択している Tcl アプリに関する次の情報が表示されます。
    • [Name] : アプリの名前
    • [Description] : アプリの簡単な説明
    • [Revision] : アプリのリビジョン番号
    • [Required] : そのアプリに最低限必要な Vivado Design Suite のバージョン
    • [Company] : アプリを提供および管理する企業の名前
    • [URL] : 選択したアプリの資料、テスト データ、スクリプトなどへのリンク
    • [Tcl Procs] : アプリに含まれる Tcl プロシージャのリスト
      ヒント: Tcl アプリが Vivado Design Suite にインストールされると、リストされた Tcl proc はクリックしたときにコマンド メニューを開くリンクになります。このリンクから、Tcl スクリプトのヘルプ テキストやソース コードにアクセスしたり、[Customize Commands] ダイアログ ボックスを開いて Vivado IDE にプロシージャを追加できます。
