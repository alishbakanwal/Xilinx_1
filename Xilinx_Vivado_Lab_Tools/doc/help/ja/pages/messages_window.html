[Messages] ウィンドウ

[Messages] ウィンドウには、デザインのプロセス中に Vivado® Design Suite で生成されるメッセージの履歴が表示されます。メッセージには、特定のメッセージ ID が付いており、そのメッセージに関連するソース ファイルまたはデザイン オブジェクトへのハイパーリンクが含まれます。メッセージ内のハイパーリンクをクリックすると、関連するソース ファイルがテキスト エディターで開くか、関連するオブジェクトが [Netlist] や [Device] ウィンドウなどの別のウィンドウで選択されます。

[Messages] ウィンドウのトップバーには、異なるメッセージに関連する統計が表示されます。メッセージは、その重要度によってリストされます。重要度を示すチェック ボックスをオンまたはオフにすると、その重要度のメッセージを表示または非表示にできます。[Show All] および [Hide All] ボタンをクリックすると、すべてのメッセージをすばやく表示または非表示にできます。

[Messages] ウィンドウの右クリック メニューには、特定のメッセージ、同じ ID を持つメッセージ、同じ重要度のメッセージの表示を制限するコマンドが含まれます。これにより、[Messages] ウィンドウにレポートされるメッセージが制限されます。右クリック メニューから [Wrap Lines] をクリックすると、メッセージが必要に応じて複数行に折り返しで表示されるようになります。

[Messages] ウィンドウのツールバーには、次のコマンドが含まれます。

Search 検索パターンを入力する検索フィールドが開きます。検索パターンと一致するメッセージを表示します。
Collapse すべてのメッセージ グループの展開を閉じた状態で表示します。
Expand すべてのメッセージ グループが展開されます。
Group by File 関連するファイル別にメッセージがグループ分けされます。
Group by ID メッセージがメッセージ ID 別にグループ分けされます。
Filter [Messages] ウィンドウにフィルターを適用して、表示されるメッセージ数を制限します。メッセージは、表示制限または重要度でフィルターできます。
Manage Message Suppression 表示制限メッセージの規則を定義および管理できる [Manage Suppression] ダイアログ ボックスを開きます。
Discard User Created Messages [Messages] ウィンドウに表示されている前に実行した Tcl コマンドおよびユーザー スクリプトに関連するメッセージを削除します。
ヒント: このコマンドでは、合成およびインプリメンテーション run などの Vivado Design Suite プロセスに関連するメッセージは削除されません。
