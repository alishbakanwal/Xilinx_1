[XADC Plot] ウィンドウ

[XADC Plot] ウィンドウには、現在のハードウェア デバイスに含まれる選択したシステム モニター (SYSMON) コアのセンサーがグラフで表示されます。SYSMON Analog-to-Digital Converter (ADC) では、オンチップ温度と供給センサーを介してダイ温度および電圧が測定されます。ADC では、広範囲のアプリケーションに高精度のアナログ インターフェイスが提供されます。

[XADC Plot] ウィンドウのツールバーには、次のコマンドが含まれます。

Add selected sensors SYSMON のセンサーのリストが表示され、線グラフに選択および追加できます。
ヒント: グラフの Y 軸は、表示されるさまざまなセンサーのスケールを反映させるために変更されます。
Remove selected sensors [XADC Plot] ウィンドウから選択したセンサーが削除されます。
Refresh System Monitor SYSMON デバッグ コアのプロパティが現在のデバイスの物理的なシステム モニターからの値で更新されます。
Set JTAG Scan Rate [XADC Plot] ウィンドウが現在のデバイスの物理的なシステム モニターからアップデートされる頻度を指定します。
