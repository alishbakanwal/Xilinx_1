[Hardware Server Settings] ページ

ハードウェア サーバーに接続する場合は、まずアプリケーションがローカル マシンで実行されるのか、リモート ホストで実行されるのかを指定する必要があります。

[Connect to] : [local server] で実行されている hw_server に接続するか、[remote server] を指定します。Vivado® Design Suite は、ローカルで実行される hw_server に接続するようにあらかじめ設定されていますので、ローカルの場合はこれ以上設定する必要はありませんが、リモート ホストに接続する場合は、次のフィールドも定義する必要があります。
  • [Host name] : リモート ホストの名前を指定します。
  • [Port] : サーバーへの接続用の通信ポートを指定します。デフォルトのポート設定は 3121 ですが、hw_server アプリケーションを起動する際に変更することもできます。
    ヒント: hw_server -help と入力して、コマンド オプションのリストを参照してください。

[Next] をクリックすると、指定した hw_server に接続され、ウィザードの [Select Hardware Target] ページが開きます。
