[Edit Enumeration] ダイアログ ボックス

このダイアログ ボックスでは、選択したプローブに対して表示される列挙値を定義できます。列挙値は、たとえば [ILA Wave] ウィンドウでの表示に使用されます。列挙値は、プローブ信号に定義された値がある場合に表示されます。プローブに列挙が定義されていない値がある場合は、実際の値が表示されます。

ダイアログ ボックスの [Enumerations] の表には、プローブの現時点で定義済みの列挙値がリストされます。これらの列挙値は編集したり、追加したりできます。表には、列挙の [Name]、[Radix]、[Value] が表示されます。[Name] は、実際の値の代わりにプローブに表示される列挙値で、[Radix] は、いつ実際の値を列挙値に置き換えるのかを評価するために使用する値の形式です。[Value] は列挙値に置き換わるプローブ信号の実際の値を示しています。

ローカル ツールバーには、次のコマンドが含まれます。

  • [Add] () : [Enumerations] の表に新しい行が追加され、新しい [Name]/[Radix]/[Value] の組み合わせが定義できるようになります。
  • [Remove] () : 現在選択されている列挙を削除します。

[OK] をクリックすると、列挙が適用され、ダイアログ ボックスが閉じます。
