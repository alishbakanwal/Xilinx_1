[Package a Specified Directory] ページ

  • [Directory] : IP のソース ファイルのディレクトリを指定します。
    重要: Vivado Design Suite では、推論ルールを使用してディレクトリを IP としてパッケージします。指定したディレクトリは、このルールに準拠している必要があります。
  • [Package as a library core] : パッケージした IP をライブラリ コアとして定義します。ライブラリ コアはそれ自体だけでは Vivado IP カタログからはカスタマイズできませんが、別の IP を開発するときに階層モジュールとして組み込むことができます。
