[Select Clock Domain] ダイアログ ボックス

[Select Clock Domain] ダイアログ ボックスは、現在のデザインでクロックをすばやく見つけることができる定義済みクエリーです。

Search 検索パターンを入力する検索フィールドが開きます。
Collapse all クロックの階層が閉じられます。
Expand all クロックの階層が展開されます。
Clock Type クロックのタイプを次のように指定します。
  • ALL_CLOCKS : デザインに含まれるクロックすべてをリスト
  • GLOBAL_CLOCK : デバイスの BUFG に接続されるクロックをリスト
  • REGIONAL_CLOCK : デバイスの BUFR に接続されるクロックをリスト。リージョナル クロック ネットワークは、グローバル クロック ネットワークから独立しています。
  • LOCAL_CLOCK : 汎用配線リソースに配線されるクロック ネットワークをリスト
Search Hierarchically デザイン階層中のクロックが検索されます。

[Select Clock Domain] ダイアログ ボックスにはクロックのリストが表示され、その 1 つを Set Up Debug ウィザードの [Nets to Debug] ページで選択した信号に割り当てることができます。

リストからクロックを 1 つ選択して [OK] をクリックすると、ダイアログ ボックスが閉じます。
