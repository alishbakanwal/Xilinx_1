Clear Placement Constraints ウィザード

Clear Placement Constraints ウィザードでは、現在のデザインから前に配置したセルや I/O ポート、またはその両方の配置を削除できます。Clear Placement Constraints ウィザードで表示されるページは、デザインの状態とウィザードでの選択によって異なります。たとえば、デザインに固定されたセルまたはポートと固定されていないセルまたはポートがあるかどうか、ウィザードを起動する前にポートまたはセルを選択したかどうかによって異なります。

  • [What type of placement do you want to clear?] (削除する配置の選択)
    • [Cell placement] : 現在のデザインのリーフ セルまたはプリミティブ セルの配置制約が削除されます。
    • [I/O port placement] : デザインのポートの I/O ポート割り当てが削除されます。
    • [Both] : 現在のデザインのセル配置とポート割り当ての両方が削除されます。

削除する配置をセルにするか、ポートにするか、両方にするかを選択したら、[Next] をクリックします。次の表示されるページは、ここでの選択と、[Clear Placement] を実行する前に選択したオブジェクトによって異なります。このウィザードでは、次のページが表示されます。

  • [Unplace Ports] ページ
  • [Unplace Cells] ページ
  • [Cell Types to Unplace] ページ
  • [Fixed Placement] ページ
