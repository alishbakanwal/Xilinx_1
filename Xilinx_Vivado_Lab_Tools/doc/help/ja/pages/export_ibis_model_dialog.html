[Export IBIS Model] ダイアログ ボックス

IBIS モデルでは、デバイスの各 IOB 規格用に作成したとおり、V/I 曲線データに基づいたデバイス インターコネクトの信号特性が記述されます。IBIS モデルをエクスポートすると、デザインで使用されるピン、これらのピンに接続される信号、そのピンに接続される IOB の IBIS バッファー モデルのリストされたファイル (.ibs) が Vivado ツールで生成されます。

  • [Output File] : IBIS モデル ファイルのパスとファイル名を指定します。
  • [Options] :
    • [Include all models] : 現在のデバイスの IOB モデルすべてが含まれるようになります。デフォルトでは、デザインで使用される IOB のみが含まれます。
    • [Disable per pin modeling] : ダイ パッドからパッケージ ピンへの送信パスのピンごとの記述がされないようになります。IBIS モデルには、[Package] セクションのすべてのピンが 1 本の RLC 伝送ラインで記述されるようになります。
    • [Maximum length of signal names] : 出力ファイルの信号名の長さを指定します。
    • [Updated generic IBIS models file] : デバイスの IBIS モデル ファイルを指定すると、現在のパーツの Vivado インストールにある IBIS モデルが上書きされます。出力 IBIS ファイルが生成される際には、インストールされたモデルではなく、指定したモデルが使用されます。
    • [Updated parasitic package data file] : 寄生パッケージ ファイル (.pkg) を指定すると、現在の Vivado インストールに含まれる寄生パッケージ ファイルが上書きされます。
    • [Component name] : 新しいコンポーネント名を指定すると、それが出力ファイルに書き込まれます。
