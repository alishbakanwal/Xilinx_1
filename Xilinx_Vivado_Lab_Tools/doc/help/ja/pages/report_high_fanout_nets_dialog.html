[Report High Fanout Nets] ダイアログ ボックス

このダイアログ ボックスを使用すると、現在のデザインのファンアウトの大きなネットがレポートできます。合成後および物理最適化後に、ファンアウトの大きい信号のリストを確認することは重要です。エンドポイントが何千もあるようなネットでは、タイミングを満たすのが困難なことがよくあります。ファンアウトの大きいネットはファブリック レジスタ (FD*) で駆動することをお勧めします。 これを使用すると、簡単に物理最適化中に複製および移動できます。

  • [Results name] : Vivado IDE の [High Fanout Nets] ウィンドウのレポートに適用する名前を指定します。
  • [Export to file] : ファンアウトの大きいネットのレポート結果が指定ファイルに書き込まれます。ファイル形式はテキストまたは XML のいずれかとして指定できます。
  • [Open in a new tab] : Vivado IDE の [High Fanout Nets] ウィンドウに新しいタブでファンアウトの大きなネットのレポートを開きます。オフの場合は、新しいレポートで最後に開いたレポートが上書きされます。
