[Interface Definition Chooser] ダイアログ ボックス

このダイアログ ボックスでは、現在定義されている Vivado® Design Suite で使用可能なインターフェイスからインターフェイスを選択できます。インターフェイス定義は、名前と説明別にリストされます。[Group by Interface Type] コマンド () を使用すると、インターフェイスをタイプ別に分類したり、名前をアルファベット順にリストしたりできます。

インターフェイスを選択すると、インターフェイス定義に含まれる信号ポートを使用して [Interface Logical Port] リストが生成され、インターフェイスでサポートされる信号を確認できます。

[OK] をクリックすると、現在選択されているインターフェイス定義を選択できます。
ヒント: 複数のインターフェイスが選択されている場合は、リストの最初のものがインターフェイス定義として選択されます。
