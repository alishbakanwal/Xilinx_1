[Clock Networks] ウィンドウ

[Clock Networks] ウィンドウには、[Report Clock Networks] コマンドの結果が表示されます。各クロック ツリーにはソースから終点までのクロック ネットワークが示され、終点のタイプ別に並べられます。ツールバーの [Filter] コマンドを使用すると、入力ポート、セルおよび入力ピン、クロックおよびクロック以外の終点などのクロック ツリーのオブジェクトを表示および非表示にして、クロック ツリー全体を確認できます。
ヒント: 完全なツリーの詳細は [Clock Networks] ウィンドウにのみ表示されます。[Report Clock Networks] コマンドのテキスト版には、クロック ルートの名前のみが表示されます。

[Clock Networks] ウィンドウのツールバーには、次のコマンドが含まれます。

Collapse [Clock Networks] ウィンドウに表示されるクロック ツリーをすべての展開が閉じられます。
Expand Selected Clock Trees 選択したクロック ツリーが展開されます。
Schematic 選択したクロックを含む回路図が開きます。
Filter Ports クロックの入力ポートが表示されます。
Filter Net クロック ツリーのロード間のクロック ネットが表示されます。
Filter Cell MMCM およびクロック バッファーなどのクロック ツリーのネットリスト セル オブジェクトが表示されます。
Filter Input Pins セルの入力ピンが表示されます。
Filter Output Pins セルの出力ピンが表示されます。
Filter Clock Endpoints クロック ツリーのクロック パスの終点が表示されます。
Filter Non-Clock Endpoints クロック ツリーのクロック以外のパスの終点が表示されます。
Rerun クロック ネットワーク レポートが生成し直されます。
