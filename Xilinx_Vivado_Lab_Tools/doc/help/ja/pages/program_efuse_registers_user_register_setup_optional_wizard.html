[User Register Setup] ページ (オプション)

Program eFUSE Registers ウィザードのこのページでは、FUSE_USER および FUSE_USER128 レジスタの 32 ビット コードと 128 ビット コードの両方を定義できます。

  • [Specify 32-bit value to program into USER register] : FUSE_USER レジスタのプログラムを有効にします。
    • [USER bits [31:0]] : コードを 16 進数値として指定します。
  • [Specify 128-bit value to program into USER128 register] : FUSE_USER128 レジスタのプログラムを有効にします。
    • [USER bits [127:0]] : コードを 16 進数値として指定します。

[Next] をクリックして [Control Register Setup (Optional)] ページに進みます。
