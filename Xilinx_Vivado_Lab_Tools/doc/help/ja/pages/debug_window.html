[Debug] ウィンドウ

[Debug] ウィンドウには、現在のデザインでデバッグ用にマークされたデバッグ コア、ポート、およびネットが表示されます。この情報にアクセスするには、ウィンドウの一番下で次のタブのいずれかをクリックします。
  • [Debug Cores] : 現在のデザインに追加された ILA デバッグ コアの階層ツリー ビューが表示され、ILA コアに割り当てられたプローブ ポートとそれらのプローブ ポートに割り当てられたクロックおよびネットがリストされます。
  • [Debug Nets] : MARK_DEBUG プロパティの付いたデバッグ用にマークしたネットがリストされ、ILA コアに割り当てられたネットと、割り当てられていないネットが表示されます。

ローカル ツールバーには、次のコマンドが含まれます。

Search 検索パターンを入力する検索フィールドを開きます。[Debug] ウィンドウから指定した値が検索されます。
Collapse All [Clock Resources] ウィンドウのセルの展開が閉じられます。
Expand All [Clock Resources] ウィンドウのセルが展開されます。
Set Up Debug Set Up Debug ウィザードが起動されます。 デバッグ用にマークするネットを選択すると、デバッグ コアおよびポートをデザインに追加するネットリスト挿入プロセスが順を追って示されます。
Create Debug Core 新しい ILA デバッグ コアがデザインに追加されます。
Create Debug Port 選択したデバッグ コアのプローブ ポートが定義されます。
Assign to Debug Port プローブ ポートにネットが割り当てられ、デバッグ中に信号にアクセスできるようになります。
