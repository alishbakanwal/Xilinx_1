[Clock Resources] ウィンドウ

7 シリーズ デバイスおよび Zynq®-7000 ファミリの場合、[Clock Resources] ウィンドウにデバイスの I/O バンク、クロック、および GT リソースがタブで表示され、デバイス上のこれらのリソースの相対的な位置が示されます。このウィンドウは、I/O、クロック、および GT オブジェクトをすばやく確認したり、デザインに手動で配置するために使用します。[Netlist] ウィンドウまたは [I/O Ports] ウィンドウなどのほかのウィンドウからデザイン オブジェクトを [Clock Resources] ウィンドウに表示されているデバイス リソースにドラッグ アンド ドロップできます。デバイス リソースを選択して、右クリックで [Mark] または [Highlight] をクリックすると、[Device] ウィンドウで簡単にそのオブジェクトを見つけることができます。

[Clock Resources] ウィンドウのセルを展開/非展開することで、デバイスの特定領域の内容を表示/非表示にできます。ウィンドウの特定セクションを非表示にする場合はマイナス (-) を、表示するにはプラス (+) をクリックします。

ローカル ツールバーには、次のコマンドが含まれます。

Search 検索パターンを入力する検索フィールドを開きます。[Clock Resources] ウィンドウから指定した値が検索されます。
Collapse All [Clock Resources] ウィンドウのセルの展開が閉じられます。
Expand All [Clock Resources] ウィンドウのセルが展開されます。
Scroll [Clock Resources] ウィンドウをスクロールして選択したオブジェクトが表示されます。

[Clock Resources] ウィンドウは Vivado® IDE のワークスペースに表示されます。ワークスペースの詳細は、『Vivado Design Suite ユーザー ガイド : Vivado IDE の使用』 (UG893) のこのセクションを参照してください。
