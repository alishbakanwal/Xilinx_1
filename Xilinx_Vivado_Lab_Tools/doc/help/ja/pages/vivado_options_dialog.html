[Vivado Options] ダイアログ ボックス

[Vivado Options] ダイアログ ボックスでは、Vivado® Design Suite のアクションおよびビヘイビアーを制御するツール レベルのオプションを設定できます。オプションは、次のカテゴリに分類されます。

  • [General] : 起動ビヘイビアー、デフォルト ディレクトリおよび自動保存などのツールの一般的な機能を設定します。
  • [Colors] : Vivado IDE の文字表示色、背景色、およびさまざまなデザイン オブジェクトの色など、ツールで使用されるカラー スキームを定義します。
  • [Selection Rules] : [Device] や [Package] ウィンドウなどのさまざまなウィンドウでクリックしたときのオブジェクトの選択動作を指定します。これは、ウィンドウ内で表示が重なっていて特定のオブジェクトを選択するするのが困難な場合に役立ちます。
  • [Shortcuts] : ツール内の特定のアクションに使用するキーボード ショートカットを定義します。
  • [Strategies] : 合成およびインプリメンテーション run のストラテジを定義および管理します。
  • [Remote Hosts] : LSF を介した分散プロセッシングまたはネットワーク内のリモート コンピューターでの分散プロセッシングに対して、合成およびインプリメンテーション run を設定できます。詳細は、『Vivado Design Suite ユーザー ガイド : インプリメンテーション』 (UG904) のこのセクションを参照してください。
    ヒント: このオプションは、Linux オペレーティング システムでのみ使用できます。
  • [Text Editor] : デザイン ソース ファイル、Tcl スクリプトおよびその他のテキスト ファイルを開くときに使用するテキスト エディターを指定します。その際使用する標準フォントおよびテキスト エディターに関するその他の機能もここで設定できます。
  • [Window Behavior] : 特定アクションに対する警告や質問を表示するメッセージ ウィンドウを設定できます。こういった確認メッセージは、ここでオンまたはオフにできます。

[OK] をクリックすると、オプションが vivado.ini ファイルに書き込まれ、ツールを起動するたびに使用されるようになります。
