[Export Simulation] ダイアログ ボックス

このダイアログ ボックスでは、Mentor Graphics 社の ModelSim、Cadence 社の Incisive Enterprise Simulator、および Synopsys 社の VCS MX などの選択したターゲット シミュレータのシミュレーション スクリプトおよびデザイン ファイルがエクスポートできます。生成されたスクリプトには、デザインをコンパイル、エラボレート、シミュレーションするためのシミュレータ コマンドが含まれます。

  • [Target simulator] : シミュレーション ファイルをエクスポートするサードパーティ シミュレータが選択できます。
    ヒント: Vivado ツールが現在実行されている OS で使用できるサードパーティ シミュレータのみがリストされます。
  • [Compiled library location] : サードパーティ シミュレータで使用するライブラリ コンパイルのパスを指定します。ザイリンクス ライブラリを外部ライブラリを使用するには、[Compile Simulation Libraries] コマンドを使用してコンパイルし、デバイス リソースが正しくシミュレーションされるようにする必要があります。シミュレーション ライブラリには、デバイスおよび IP のビヘイビアーおよびタイミング モデルが含まれます。
  • [Export directory] : さまざまなデザイン ファイルとスクリプトを出力するディレクトリを指定します。ディレクトリが既にある場合は、[Overwrite files] チェックボックスをオンにすると内容を更新できます。
  • [Use absolute paths] : エクスポートしたシミュレーション スクリプトで絶対パスが使用されるようになります。デフォルトでは、デザイン ソース ファイルとインクルード ディレクトリ パスが [Export directory] に対して相対的になります。
  • [Copy source files to export directory] : エクスポートされたネットリストとシミュレーション スクリプトと一緒にデザイン ソース ファイルがコピーされます。
