[Package Pins] ウィンドウ

[Package Pins] ウィンドウでは、パッケージのピンがリストされ、各ピンに割り当てられた I/O ポートなどのピンに関連する属性が表示されます。[Package Pins ] ウィンドウは表形式のウィンドウで、ピンに関するデータが列形式で表示されます。列のヘッダーを右クリックすると、列を表示および非表示にするコマンドと列幅を管理するコマンドが表示されます。表の情報は、列ヘッダーを 1 回または 2 回クリックすると並び替えることができます。詳細は、『Vivado® Design Suite ユーザー ガイド : Vivado IDE の使用』 (UG893) のこのセクションを参照してください。

[Package Pins] ウィンドウの右クリック メニューには、パッケージ ピンへのポートの割り当て、パッケージ ピンへのポートの割り当て禁止などのコマンドが含まれます。

[Package Pins] ツールバーには、次のコマンドが含まれます。

Search 検索パターンを入力する検索フィールドが開きます。[Package Pins] の表から指定した値が検索されます。
Collapse all パッケージ ピン階層の展開が閉じられます。
ヒント: このコマンドは、ピンを I/O バンク別に分けた場合にのみ使用できます。
Expand all パッケージ ピン階層が展開されます。
Group by I/O Bank パッケージ ピンがそれらに関連する I/O バンクに分けられます。このコマンドを使用しない場合は、ピンは名前別にリストされます。
Scroll [I/O Port] ウィンドウをスクロールされて選択したオブジェクトが表示されます。
