[Timing Path Report] ウィンドウ

[Timing Path Report] ウィンドウには、パスを定義する始点、通過点、終点などを含めた特定のタイミング パスの詳細なタイミング レポートが表示されます。ソース クロックとデスティネーション クロックのパスもレポートされます。この情報は、パスのタイミング違反の原因を理解しやすくするために提供されます。ウィンドウには、スラック、タイミング要件、パス遅延合計、パスのロジックによる遅延、見積もりまたは実際の配線遅延、クロック パス スキューなどのパスのサマリが表示されます。

[Timing Path Report] ウィンドウには、スラックやクロック パス スキューなどの特定の値の詳細へのリンクも含まれています。値をクリックすると、それがどのように計算されたのかがわかります。クリックするとそのオブジェクトを [Netlist] や [Device] ウィンドウなどのその他のウィンドウで表示できます。

[Timing Path Report] ウィンドウのさまざまなエレメントの説明と読み方については、『Vivado® Design Suite ユーザー ガイド : ザイン解析およびクロージャ テクニック』 (UG906) のこのセクションを参照してください。
