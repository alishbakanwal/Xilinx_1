[Format Names] ページ

[Format Names] ページでは、Create Pblocks ウィザードで指定した Pblock 名の形式を定義できます。
重要: 名前の形式は、現在の [Create Pblock] コマンドで使用され、ほかの手段で作成される Pblock には使用されません。
  • [Prefix] : Pblock 名に付ける接頭辞に使用するテキスト文字列を入力します。
  • [Suffix] : Pblock 名の接尾語を指定します。 これは Pblock に関連付けられた階層モジュールの [Cell name] またはツールで自動的に割り当てられる [Numeric] の値で定義されます。

[Sample] 領域には、選択した [Prefix] および [Suffix] に従ってフォーマットされた Pblock 名の例が表示されます。[Next] をクリックして Create Pblocks ウィザードの [Summary] ページに進みます。
