[Control Register Setup] ページ (オプション)

Program eFUSE Registers ウィザードのこのページでは、eFUSE ビットの主な機能へのアクセスを提供する FUSE_CNTL レジスタを管理できます。

  • [Enable control register programming] : チェック ボックスでさまざまな eFUSE 制御へのアクセスを個別にオンまたはオフにできます。使用可能な制御のリストは、UIltraScale® アーキテクチャと 7 シリーズで異なります。このページには各制御レジスタの簡単な説明が記述されます。特定の設定の詳細は、次のマニュアルを参照してください。
