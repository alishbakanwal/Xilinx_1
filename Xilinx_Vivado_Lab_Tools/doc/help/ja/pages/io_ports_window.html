[I/O Ports] ウィンドウ

[I/O Ports] ウィンドウには、現在のデザインで定義された I/O ポートがリストされ、I/O 規格、駆動電流、およびスルーなどのポートの属性を設定できます。[I/O Ports] ウィンドウは表形式のウィンドウで、ポートに関するデータが列形式で表示されます。列を表示および非表示にするコマンドと、列幅を管理するコマンドは、列のヘッダーを右クリックすると表示されます。表の情報は、列ヘッダーを 1 回または 2 回クリックすると並び替えることができます。詳細は、『Vivado® Design Suite ユーザー ガイド : Vivado IDE の使用』 (UG893) のこのセクションを参照してください。

[I/O Ports] ウィンドウの右クリック メニューには、I/O プランニング プロジェクトで新しいポートを定義したり、既存ポートをコンフィギュレーションしたり、I/O ポートを自動または手動で配置したり、割り当てを固定したりといったコマンドが表示されます。
重要: すべてのポートに デフォルト以外の I/O 規格が定義され、ビットストリーム生成前にパッケージ ピン サイトに固定されている必要があります。固定されたポートとはユーザーにより割り当てまたは固定されたポートです。 固定されないポートとは、パッケージ ピン ロケーションに割り当てられていても、ユーザーによって固定されていないポートです。

[I/O Ports] ウィンドウのツールバーには、次のコマンドが含まれます。

Search 検索パターンを入力する検索フィールドが開きます。[I/O Ports] の表から指定した値が検索されます。
Collapse All リストされた IP の展開が閉じられます。
Expand All リストされた IP が展開されます。
ヒント: これらは、I/O ポートがインターフェイスおよびバス別にグループ分けされる場合にのみ使用できます。
Group by Interface and Bus I/O ポートがバスまたはインターフェイス定義に従ってグループ分けされます。このコマンドを使用しない場合は、ポートは名前別にリストされます。
Create I/O Port Interface 選択したポートをグループ分けするため、新しいポート インターフェイスを定義します。
Schematic 選択したポートを含む回路図が開きます。
Scroll [I/O Port] ウィンドウをスクロールして選択したオブジェクトが表示されます。
Fix Ports 選択したポートを [Fixed] とマークして、自動配置コマンドや配置最適化で移動されないようにします。
