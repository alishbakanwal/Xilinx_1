Create New Runs ウィザード

Create New Runs ウィザードでは、新しい合成およびインプリメンテーション run の作成プロセスが手順を追って示されます。最初にまず、どのタイプのデザイン run を作成するのか指定します。

  • [Synthesis] : 新しい合成 run をインプリメンテーション run なしで作成します。
  • [Implementation] : 新しいインプリメンテーション run を作成します。
  • [Both] : 合成およびインプリメンテーション run の両方を作成します。

Flow Navigator コンテキスト メニューから [Create Synthesis Runs] または [Create Implementation Runs] コマンドを選択すると、Create New Runs ウィザードの最初のページに合成 run を作成しているのかインプリメンテーション run を作成しているのかが表示されます。選択はできません。

ヒント: ネットリスト ベースのプロジェクトの場合、合成 run は必要ないので、最初の Create New Runs ウィザードのページは飛ばされ、[Configure Implementation Runs] ページが開きます。

[Next] をクリックして [Configure Synthesis Runs] ページまたは [Configure Implementation Runs] ページを開きます。
