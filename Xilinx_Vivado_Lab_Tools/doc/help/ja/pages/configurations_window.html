[Configurations] ウィンドウ

[Configurations] ウィンドウは、[Tools] → [Enable Partial Reconfiguration] コマンドを使用するか、PR_FLOW プロパティを TRUE に設定して、プロジェクトでパーシャル リコンフィギュレーション (PR) フローを有効にすると表示されます。ウィンドウには、PR コンフィギュレーションが名前順にリストされ、現在のプロジェクトのパーティション定義 (partitionDefs) が表示されます。コンフィギュレーションごとにリコンフィギャラブル モジュール (RM) を指定して、partitionDef をそれぞれ割り当てることができます。右クリック メニューからは、アクティブ コンフィギュレーション (インプリメンテーションのデフォルト コンフィギュレーション) を指定できます。

[Configurations] ウィンドウのツールバー メニューには、次のコマンドが含まれます。

Search 検索パターンを入力する検索フィールドが開きます。
Modify Configuration Partial Reconfiguration ウィザードの [Edit Configurations] が開き、選択した PR コンフィギュレーションを変更できます。
