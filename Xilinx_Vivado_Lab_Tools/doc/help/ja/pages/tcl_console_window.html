[Tcl Console] ウィンドウ

[Tcl Console] ウィンドウは Vivado® IDE のコマンド ライン インターフェイスであり、直接コマンドを入力してデザインのエレメントを処理できます。コマンドを入力していくと、自動コンプリート機能により、入力した文字で開始するコマンド (複数ある場合はそれらのリスト) が表示されます。コマンドの 1 つをクリックするか、入力し続けて特定のコマンドに選択肢を狭めていきます。

[Tcl Console] ウィンドウには、入力されたコマンドのログと、これらのコマンドを実行した際に Vivado ツールから戻されたメッセージも表示されます。このログは、Tcl コマンドが Vivado IDE で実行されるとリアルタイムでアップデートされます。

ヒント: 使用した Tcl コマンドは vivado.jou ファイルに、コマンドのログおよびメッセージは vivado.log ファイルに記述されます。詳細は、『Vivado Design Suite ユーザー ガイド : Vivado IDE の使用』 (UG893) のこのセクションを参照してください。

[Tcl Console] ウィンドウに次のように入力すると、Tcl コマンドのヘルプが表示されます。

help command_name

[Tcl Console] ウィンドウのツールバーには、次のコマンドがあります。

Collapse all [Tcl Console] ウィンドウのログの展開が閉じられ、実行された Tcl コマンドのみが表示されます。 生成されたメッセージは表示されません。
ヒント: これにより、[Tcl Console] ウィンドウからコマンドをテキスト エディターにコピーして貼り付けることができるので、Tcl スクリプトが作成しやすくなります。
Expand all [Tcl Console] ウィンドウのログが展開され、さまざまな Tcl コマンドからのメッセージすべてが表示されるので、ツールをモニターしたりデザインを理解しやすくなります。
Pause 特定のセクションを読めるように、ログのスクロールが一時停止されます。一時停止しても、メッセージは取り込まれています。[Pause] ボタンをもう 1 度クリックすると、スクロールが再開されます。
Show [Tcl Console] ウィンドウの一番下に検索バーが表示され、テキスト文字列を検索してハイライトすることができます。
Clear 前のコマンドおよびメッセージが [Tcl Console] ウィンドウから削除されます。
