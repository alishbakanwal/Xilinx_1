New Project ウィザード

New Project ウィザードでは、RTL デザイン、合成後のネットリスト デザイン、I/O プランニング プロジェクト、およびデザイン テンプレートに基づいたサンプル デザインを使用して Vivado® Design Suite プロジェクトを作成できます。また、ISE® Design Suite、XST、または Synplify の既存プロジェクトをインポートすることもできます。新規プロジェクトの作成で使用されるフローには、次が含まれています。

  • プロジェクト名およびディレクトリを定義
  • プロジェクト タイプを定義
  • デザイン ソースを追加または作成
  • 既存 IP を追加
  • 制約の追加または作成
  • デザインのデフォルト パーツまたはボードを定義
