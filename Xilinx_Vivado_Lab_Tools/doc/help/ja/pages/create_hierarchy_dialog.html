[Create Block Design Hierarchy] ダイアログ ボックス

[Create Block Design Hierarchy] ダイアログ ボックスでは現在のブロック デザインに追加する新しい階層セルを定義します。

  • [Cell name] : 作成する階層セルの名前を指定します。
  • [Move selected blocks to new cell] : 現在選択している IP が新しい階層セルに移動されます。
    ヒント: このオプションは、[Create Hierarchy] コマンドを使用する前に、ブロック デザインで IP を選択した場合にのみ使用できます。IP を新しい階層セルの作成後にドラッグ アンド ドロップすることもできます。

[OK] をクリックすると、新しい階層セルが作成され、現在のブロック デザインに追加されます。新しい階層セルは、ピンまたはインターフェイスなしで作成されます。
