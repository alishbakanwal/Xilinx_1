[Find Results] ウィンドウ

[Find Results] ウィンドウには、[Edit] → [Find] コマンドの検索結果が表示されます。[Find Results] ウィンドウにリストされるオブジェクトは、[Find] ダイアログ ボックスで指定されるオブジェクト タイプとプロパティ値に基づいています。

[Find Results] ウィンドウのツールバーには、次のコマンドが含まれます。

Search 検索パターンを入力する検索フィールドが開きます。指定した検索パターンに一致するオブジェクトが [Find Results] ウィンドウに表示されます。
Schematic 選択したオブジェクトを表示する回路図が開きます。
Expand すべてのオブジェクト グループが展開されます。
Mark 選択したオブジェクトをそのオブジェクトが表示されるさまざまなウィンドウでマークします。たとえば、このコマンドは、[Device] ウィンドウでオブジェクトを簡単に見つけられるように、[Find] ウィンドウでオブジェクトをマークするために使用できます。
Show Tcl Command リストされるオブジェクトを検索するために使用する Tcl コマンドを表示します。
Rerun 結果のウィンドウを生成し直すための検索条件を再実行します。たとえば、現在のデザインへの変更を含めて結果がアップデートされます。
