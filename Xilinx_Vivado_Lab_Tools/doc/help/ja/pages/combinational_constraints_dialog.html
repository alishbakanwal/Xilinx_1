[Combinational Constraints] ダイアログ ボックス

このダイアログ ボックスでは、現在選択している組み合わせパスの入力および出力遅延を定義できます。パスの遅延は set_input_delay および set_output_delay コマンドと関連しています。

  • [Input min] : 選択したパスの最小入力遅延 (ナノ秒) を指定します。
  • [Input max] : 選択したパスの最大入力遅延を指定します。
  • [Output min] : 選択したパスの最小出力遅延を指定します。
  • [Output max] : 選択したパスの最大出力遅延を指定します。

[OK] をクリックすると、現在選択しているパスにその遅延が追加されます。
