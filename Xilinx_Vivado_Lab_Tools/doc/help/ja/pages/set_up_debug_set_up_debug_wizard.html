Set Up Debug ウィザード

Set Up Debug ウィザードを使用すると、ウィザードの指示に従ってデバッグ コアを現在のデザインの合成済みネットリストに挿入できます。

ヒント: デバッグ機能は、デバッグ コアを RTL デザインに直接インスタンシエートしても追加できます。これは、「HDL インスタンシエーション デバッグ プローブ フロー」と呼ばれます。詳細は、『Vivado Design Suite ユーザー ガイド : プログラムおよびデバッグ』 (UG908) のこのセクションを参照してください。
