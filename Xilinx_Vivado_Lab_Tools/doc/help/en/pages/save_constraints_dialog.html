Save Constraints

This dialog box lets you define a target constraints file to write new constraints into. This dialog displays only when there is no target constraints file defined in the current constraint set.

Constraints are read and applied to the design in the order files are listed. Constraints defined early in the list can be overwritten by constraints defined later in the list. Refer to this link in the Vivado Design Suite User Guide: Using Constraints (UG903) for more information on constraint precedence.
Important: Only new constraints are saved to the specified file. Existing design constraints are saved to the constraints file they originated from. Refer to this link in the Vivado® Design Suite User Guide: Using Constraints (UG903) for more information on entering and saving constraints.
  • Create a new file: This radio button specifes the creation of a new target constraints file and adds that file to the current constraint set.
    • File Type: Currently only XDC file types are supported.
    • File Name: Specify the name of the constraints file.
    • File Location: Specify the location for the new constraints file. You can create the new constraints file in the local project directory or in a location outside of the current project.
  • Select an existing file: Provides a drop down menu listing the constraint files defined in the constraint set associated with the current design.
