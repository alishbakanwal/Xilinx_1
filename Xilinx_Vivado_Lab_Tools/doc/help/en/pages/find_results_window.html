Find Results Window

The results of a query run from the Edit > Find command are returned to the Find Results window. The objects listed in the Find Results window are based on the object type and Property values specified in the Find dialog box.

The Find Results window toolbar includes the following commands:

Search Opens a search field to enter a search pattern. Objects matching the specified search pattern are displayed in the results window.
Schematic Opens a schematic displaying the selected object or objects.
Expand Expands all object groupings.
Mark Marks the selected object or objects in the various windows where the objects are displayed. For example, you can use this command to mark an object in the Find window so that you can easily locate it in the Device window.
Show Tcl Command Shows the Tcl command used to find the listed objects.
Rerun Reruns the Find query to regenerate the results window. For example, you can use this command to update the results with any changes to the current design.
