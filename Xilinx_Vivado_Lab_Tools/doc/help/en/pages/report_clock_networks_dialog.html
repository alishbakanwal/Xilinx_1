Report Clock Networks

The Report Clock Networks dialog box generates and opens a Clock Networks window in the Vivado IDE. The Clock Networks report provides a tree view of the clock trees in an opened synthesized or implemented design.

  • Results name: Specify the name to apply to the Clock Networks report in the Vivado® IDE.
  • Open in a new tab: Open the Clock Networks report in a new tab in the Vivado IDE. If this checkbox is not enabled, the new report overwrites the last report that was opened.

Click OK to generate the Clock Networks report.
