Verify Device

This dialog box lets you specify a bitstream file and a corresponding mask (.msk) file to use for comparison against the bitstream programmed into the current device. The Verify Device command uses both the bitstream and the required mask file to compare only the bits that are marked as important in the mask file.

You must generate the mask file when creating the bitstream using Generate Bitstream or the write_bitstream Tcl command. To create the mask file use the -mask_file option in the Project Settings - Bitstream dialog box before generating the bitstream, or as an option to the Tcl command. The Vivado tool uses the mask file to determine which readback data bits to skip and which ones to compare against the corresponding bits in the bitstream file.

  • Bitstream file: Specify the file name and path of the bitstream file associated with the current device.
    Important: Verification cannot be performed if the device is programmed with an encrypted bitstream.
  • Mask file: Specify the file name and path of the mask file associated with the specified bitstream file.

Click OK to read the bitstream from the current device and compare it with the specified files.
