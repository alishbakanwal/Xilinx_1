Create Block Design Hierarchy

The Create Block Design Hierarchy dialog box lets you define a new hierarchical cell to be added to the current block design.

  • Cell name: Specifies the name of the hierarchical cell to create.
  • Move selected blocks to new cell: Moves the currently selected IP into the new hierarchical cell.
    Tip: This option is only available if you selected IP in the block design prior to using the Create Hierarchy command. You can also drag and drop IP into the new hierarchical cell after it is created.

Click OK to create the new hierarchical cell and add it to the current block design. The new hierarchical cell is created without pins or interfaces.
