Add Module

The Add Module dialog box lets you add a module as a reference in the current block design. This requires that the source file defining the module was added as a design source to the current project. You can use the Add Sources wizard to add source files to the project.

Important: Supported file formats include VHDL and Verilog.

The list box displays the modules that are available to add to the block design. A search field is provided to enter a search pattern to limit the modules listed.

Select a module and click OK to add the module to the block design.
