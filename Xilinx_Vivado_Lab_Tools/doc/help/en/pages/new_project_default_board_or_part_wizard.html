Default Board or Part

The Default Board or Part page displays a table of available target parts or boards that can be used with the template design. The template designs include pin assignments and timing constraints that are targeted for a specific set of devices or boards. The results table displays several columns of information, such as Device Name, I/O Pin Count, the number of Block RAMs, and DSPs.

Select a target part or board for the design, and click Next to open the New Project Summary page.
Tip: For the Configurable Microblaze Design template, the Configure Project page is opened next.
