IP Catalog

The IP Catalog displays a list of IP cores that you can add to your design as well as interface definitions supported by the tool. The top of the IP Catalog has tabs to display either Cores or Interfaces. The interface definitions can be opened to view the pins that an interface is comprised of, but interfaces cannot be used from the IP catalog. When creating IP using the IP Packager you can map to the interfaces.

The IP Catalog is sorted into functional categories, or taxonomies, and IP repositories. You can also choose to list IP by name. The taxonomies are categories defined when an IP is packaged. The repository is the physical location of the IP in the Vivado® Design Suite installation or in a user-specified repository. You can point to user repositories using the Add Repository command from the right-click menu, or from the Project Settings - IP dialog box. See this link in the Vivado Design Suite User Guide: Designing with IP (UG896) for more information.

There is also a Search field () that lets you specify a search string for locating specific IP.

The IP Catalog toolbar includes the following commands when the Cores tab is displayed:
Tip: Some of these commands are not active until an IP core is selected.
Options Displays a list of categories to filter the IP Catalog by the output products a core generates, interfaces used by a core, licensing, IP provider, and the state of the IP.
Collapse all Collapses all IP hierarchies.
Expand all Expands all IP hierarchies.
Hide Hides IP that is incompatible with the currently selected target device. Disable this to show all IP.
Group by Groups the IP displayed in the catalog by the taxonomy, and by the IP repository.
Customize IP Opens the IP customization dialog box for the currently selected IP. You can also customize IP by double-clicking a selected IP.
License Status Indicates that the selected IP requires a license key. The licenses for many IP are included with the Vivado Design Suite. Others are separately licensed by Xilinx or by third-party IP providers.
Compatible Families Opens the Compatible Families dialog box, which lists the devices compatible with the currently selected IP, and lets you change the target part for the current project if needed.
View Information Provides access to various documents and websites related to the currently selected IP.
IP Settings Opens the Project Settings - IP dialog box to let you configure IP-related tool settings for the current project.
Scroll Scrolls to display the selected core in the IP Catalog when an object is selected in another window, such as the Sources window.

The IP Catalog appears in the Vivado® IDE workspace. For more information on workspace windows refer to this link in the Vivado Design Suite User Guide: Using the Vivado IDE (UG893).
