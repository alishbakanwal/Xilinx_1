Select Clock Domain

The Select Clock Domain dialog box is a predefined query to let you quickly locate clocks in the current design.

Search Opens a search field to enter a search pattern.
Collapse all Collapses the hierarchy of clocks.
Expand all Expands the hierarchy of clocks.
Clock Type Specifies the type of clock to list as:
  • ALL_CLOCKS: Lists all the clocks in the design.
  • GLOBAL_CLOCK: Lists the clocks connected to BUFGs on the device.
  • REGIONAL_CLOCK: Lists the clocks connected to BUFRs on the device. Regional clock networks are independent of the global clock network.
  • LOCAL_CLOCK: Lists the clock networks routed on general routing resources.
Search Hierarchically Searches for clocks through the design hierarchy.

The Select Clock Domain dialog box displays a list of clocks for you to assign one to a selected signal in the Nets to Debug page of the Set Up Debug wizard.

Select a clock from the list, and click OK to close the dialog box.
