Open Hardware Target Summary

The Open Hardware Target Summary page reports on the connection to the hardware server, and the hardware target. Look over the summary to be sure your selections are correct.

  • Click Back to return to prior pages to change selections as needed.
  • Click Finish to connect to the hw_server and target as indicated.
  • Click Cancel to close the wizard without connecting to the hw_server.

After the connection to the server has been established, and the hw_target is opened, the Hardware Manager opens the Program and Debug Environment. The Hardware window displays the current target.
