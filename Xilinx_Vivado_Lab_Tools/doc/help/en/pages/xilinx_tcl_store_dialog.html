Xilinx Tcl Store

The Xilinx® Tcl Store is an open source repository of Tcl code, or apps, designed specifically for use with the Vivado® Design Suite. An app is a group or collection of one or more Tcl scripts that is published and maintained by Xilinx or a third-party contributor. You can download and install apps shared by others, and write apps to share on the Xilinx Tcl Store.

The Xilinx Tcl Store dialog box lists the available apps that are installed or can be installed for use with the Vivado tools. The dialog box is divided into two sections: The list of available apps on the left side, and details of the currently selected app on the right side, with a description of the app and a list of specific Tcl procedures (procs) included in the app.

Features of the dialog box include:

  • All | Installed: Tabs to display either all of the available apps, or only the apps that are installed on your system.
  • () Search: Lets you enter a search string to limit the apps that are listed, or search for a specific app.
  • Install: Button to install the currently selected app. If the app is already installed, it will display as such.
    Tip: To uninstall an app look for a link on the right-side Details, beneath the description of the currently selected app.
  • Refresh: A button to update the current list of apps. This reloads the list of available apps from the online Xilinx Tcl Store. You should refresh the list prior to installing any apps to insure you are getting the latest version.
  • Support on GitHub: A link to the support page of the Xilinx Tcl Store.
  • Details: On the right side you will find the following details for the currently selected Tcl app:
    • Name: Name of the app.
    • Description: A short description of the app.
    • Revision: The app revision number.
    • Required: The minimum required release of the Vivado Design Suite for the app.
    • Company: The company name providing and maintaining the app.
    • URL: A link to the documentation, test data, and scripts for the selected app.
    • Tcl Procs: The list of Tcl procedures included in the app.
      Tip: When the Tcl app is installed into the Vivado Design Suite, the listed Tcl procs are links that open a menu of commands when clicked. You can access help text or source code for the Tcl script, or open the Customize Commands dialog box to add the procedure to the Vivado IDE.
