Package Pins Window

The Package Pins window lists the pins on the package, and displays the attributes associated with those pins, including the I/O ports assigned to each pin. The Package Pins window is a table window, which presents data about the pins in column form. To display and hide columns as well as manage column widths, right-click in the header of any column. You can also sort information in the table by clicking once or twice in the column header. See this link in the Vivado® Design Suite User Guide: Using the Vivado IDE (UG893) for more information.

The right-click menu of the Package Pins window includes commands to help you assign ports to package pins or prohibit the assignment of ports to package pins.

The Package Pins toolbar includes the following commands:

Search Opens a search field to enter a search pattern. The Package Pins table is searched for the specified value.
Collapse all Collapses the package pin hierarchy.
Tip: The Collapse and Expand commands are only available when the pins are grouped by I/O bank.
Expand all Expands the package pin hierarchy.
Group by I/O Bank Groups the package pins into their respective I/O banks. Pins are listed by name when this command is not enabled.
Scroll Scrolls the I/O Port window to display the selected object.
