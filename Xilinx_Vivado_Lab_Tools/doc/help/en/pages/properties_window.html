Properties Window

The Properties window displays general information as well as detailed information associated with the currently selected object. The Properties window changes depending on the type of object selected. For example, the information displayed for a selected CELL object is different from the information displayed for a selected NET or SITE object.

The Properties window has multiple views that vary based on the selected object and display details about the object. For example, selecting a CELL object displays the following views:

  • General: Provides general information related to the cell, such as name and object type.
  • Properties: Lists the properties currently defined on the selected cell. Some of the properties can be directly edited in the Properties window. Others are read-only and can only be modified by the tool.
  • Nets: Lists the nets connected to the current cell.
  • Cell Pins: Lists the pins defined on the current cell.

The Properties window toolbar includes the following commands when the Properties view is selected:

Search Opens a search field to enter a search pattern.
Collapse all Collapses all property hierarchies.
Expand all Expands all property hierarchies.
Show/Hide Hierarchy Groups or ungroups related properties.
Add Adds a property to the currently selected object from a list of predefined properties. Clicking this button opens the Add Properties dialog box with a list of available properties to choose from.
Remove Removes the selected property from the current object.
Help Displays a small help area at the bottom of the Properties window to define properties.
Sort Sorts the listed properties in increasing alphanumeric order.
