Create Pblocks - Format Names

The Format Names page lets you define the form that Pblock names will take as they are created by the Create Pblocks wizard.
Important: The name format applies to the current Create Pblock command, and does not affect Pblocks created by other means.
  • Prefix: Enter a text string to be applied as a prefix to the Pblock name.
  • Suffix: Specify the suffix of the Pblock name will be defined by the Cell name of the hierarchical module associated with the Pblock, or by a Numeric value automatically assigned by the tool.

The Sample region will display an example of the Pblock name formatted according to the Prefix and Suffix you selected. Click Next to move to the next page of the Summary page of the Create Pblocks wizard.
