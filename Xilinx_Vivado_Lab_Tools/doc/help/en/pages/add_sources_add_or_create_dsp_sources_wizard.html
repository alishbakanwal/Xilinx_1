Add or Create DSP Sources

The Add or Create DSP Sources page lets you add an existing System Generator for DSP design model file as a DSP module into the overall design. You can also create a new DSP module from within the Vivado® IDE and launch System Generator for DSP to complete the DSP design.

  • Add Sub-Design: Opens a file browser letting you select one or more DSP source files (.slx or .mdl). Selecting multiple files will add the files in sorted order to the file list.
    Tip: Alternatively, you can click and select Add Sub-Design.
  • Create Sub-Design: Opens the Create System Generator Source dialog box that lets you specify the DSP module name to create. Clicking Finish invokes System Generator for DSP.
    Tip: Alternatively, you can click and select Create Sub-Design.
  • Remove (): Removes selected files from the current project. The files are removed from the project but are not removed from the hard drive.
  • Move Up: When one or more files have been selected in the file list display area, the blue Up arrow button () is activated on the left, letting you move the selected file or files up in the listed order.
  • Move Down: When one or more files have been selected in the file list display area, the blue Down arrow button () is activated on the left hand menu, letting you move the selected file or files down in the listed order.
  • Copy sources into project: This checkbox lets you copy the specified DSP source files into the sources folder of the local project directory.
