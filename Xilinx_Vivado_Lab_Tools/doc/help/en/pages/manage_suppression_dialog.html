Manage Suppression

This dialog box lets you create and manage rules for suppressing messages in the Messages window. You can create suppression rules using the right-click menu in the Messages window to suppress specific messages or specific message IDs or severities. You can also define your own suppression rules based on matching search strings in messages.

Messages are suppressed in the Messages window if they match strings defined in the message suppression rules. The search function for matching strings is case-sensitive, matches partial strings without the need for wildcard characters, and performs a logical AND of multiple search strings in a single suppression rule.

Click the Add command () to define a new message suppression rule. This opens the New Suppression Rule dialog box.

Select one or more suppression rules and click the Remove command () to remove existing rules. Select and remove all rules to clear message suppression from the Messages window.

Click Close to close the dialog box.

Important: Messages are suppressed as soon as a valid suppression rule is defined.
