Create Block Design

The Create Block Design dialog box lets you define a new block design (BD) to add to the current project.

  • Design name: Specify the name of the block design to create.
  • Directory: Specify where to create the block design file, whether in the local project directory or in a location outside of the current project.
  • Specify source set: Add the new block design to the design sources fileset, to the default simulation fileset, or to a new simulation fileset.
    Tip: Generally, Xilinx recommends creating the block design in the design sources fileset. However, you can create a block design for simulation only by setting the fileset to a simulation fileset.

When you click OK, the design is opened in the IP Integrator feature of the Vivado IDE.
