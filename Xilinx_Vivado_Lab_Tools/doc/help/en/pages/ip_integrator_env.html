IP Integrator

The IP Integrator environment lets you edit block designs. Using the Vivado® IP integrator feature, you can create complex system designs by instantiating and interconnecting IP from the Vivado IP catalog, or a custom IP repository, onto a design canvas.

When a block design is open, you can activate the IP Integrator environment by clicking IP Integrator in the Flow Navigator. When a block design is not already open, you can click the Open Block Design command.

From the IP Integrator environment you can:

  • Add IP from the Vivado IP catalog, or a custom IP repository.
  • Add modules from referenced RTL files.
  • Choose board components from the Board tab if the Platform Board Flow is used.
  • Use Designer Assistance to quickly make design connections.
  • Mark nets for debug or insert debug cores into the block design.
  • Create a hardware wrapper to generate a top-level design for synthesis and implementation.
  • Package your block design as an IP for use in other projects.
