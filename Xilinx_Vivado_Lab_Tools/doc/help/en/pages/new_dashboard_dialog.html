New Dashboard

This dialog box lets you define a new dashboard within the logic analyzer feature of the Vivado Design Suite or the Vivado Lab Edition. The logic analyzer is used to interact with ILA, VIO, SysMon, and JTAG-to-AXI Master debug cores in your design, and programmed into the current device. The New Dashboard dialog lists the available debug cores, and lets you build a collection of associated windows to provide you a meaningful view of the activity related to that core.

  • Name: Specify a name for the dashboard that will be displayed in the logic analyzer.
  • List of Cores: The dialog box displays the current device, a list of the debug cores programmed into the device, and the windows associated with each core. Each window has a checkbox to let you add or remove the window to the dashboard. You can define a dashboard for a single debug core, so that you can see the settings, status, and activity related to that single core. You can also create custom dashboards, for example with multiple waveform windows, to see the activity on multiple cores in one dashboard.
    Tip: Adding too many windows can make the dashboard cluttered, with only a little space available for each window. You can add and remove windows in an open Dashboard window, using the Dashboard Options feature.

Click OK to open the dashboard with the selected windows.
