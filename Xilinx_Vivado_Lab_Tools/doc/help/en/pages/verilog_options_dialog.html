Verilog Options

This dialog box lets you define the following fields:.

  • Verilog Include Files Search Paths: This field lets you define and lists additional search paths for Verilog `include statements. This field has a toolbar menu with the following commands:
    Add Opens a file browser to specify a new included search path.
    Remove Removes the selected path.
    Move up Moves the selected path up in the displayed list.
    Move down Moves the selected path down in the displayed list.
  • Defines: Specify Name/Value pairs for Verilog `define statements. This field has a toolbar menu with the following commands:
    Add Opens the Add Value dialog box to define a new Name/Value pair.
    Remove Removes the selected Name/Value pair.
    Edit Opens the Edit Value dialog box to edit the selected Name/Value pair.
  • Uppercase all identifiers: Change all identifiers in the design to uppercase. This is applied to the in-memory design, and does not affect the HDL source file.

Click OK to add the options to the Simulation settings, and close the dialog box.
