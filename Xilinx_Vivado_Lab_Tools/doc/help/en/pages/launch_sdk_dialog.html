Launch SDK

Launch the Xilinx® Software Development Kit (SDK) to design the software for a top-level design that includes a block design from IP Integrator with an embedded processor like MicroBlaze, Zynq®-7000 All Programmable SoC, or Zynq UltraScale+™ MPSoC. This dialog box lets you specify the location of files needed by SDK.

  • Exported location: Specify the location of the hardware specification file as defined in the Export Hardware dialog box.
    Tip: This is typically written to the local project directory structure, but may have been written to a different location.
  • Workspace: Specify the SDK project workspace to open. The tool will create the specified directory if it does not already exist.
