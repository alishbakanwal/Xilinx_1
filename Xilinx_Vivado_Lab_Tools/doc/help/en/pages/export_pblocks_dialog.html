Export Pblocks

This dialog box lets you export an EDIF netlist for the elements of the design assigned to a selected Pblock. This lets you use the netlist for logic assigned to a Pblock as a design source in another design. An EDIF netlist will also be extracted for the top-level design.

  • Directory name: Specify the directory location for the output files exported from the selected Pblocks. The top.edn file will be written to the specified directory, as well as a directory and netlist for each selected Pblock.
  • File types to generate: Specify the output file formats to output from the selected Pblocks.
    Important: Currently only the EDIF netlist format is available, and it must be selected. These files can be added as design source files to netlist projects, but are not intended to be read into a design using update_design.
  • Add (): Opens the Select Pblock dialog to let you select one or more Pblocks from the design. A separate netlist will be exported for each Pblock selected.
  • Remove (): Removes selected Pblocks from the list of Pblocks to export.

Click OK to export the netlist from the selected Pblocks.
