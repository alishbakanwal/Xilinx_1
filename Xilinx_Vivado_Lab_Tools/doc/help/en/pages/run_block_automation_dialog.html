Run Block Automation

The Run Block Automation link is available in the IP Integrator design canvas banner for some IP such as the MicroBlaze and Zynq processors, as well as some hierarchical IP. This designer assistance feature configures and puts together the basic design connectivity necessary for the IP in question.

The Run Block Automation dialog box presents a tree view of the blocks available to connect. You can enable or disable the checkbox next to a block to enable or disable the designer assistance feature. Selecting a block in the tree view displays the connection options that you can configure for the selected block. The dialog box also displays a brief description of the automation process that will be run for that block. The available connection options depend on the IP block selected.

Click OK to run the block automation with the selected options.
