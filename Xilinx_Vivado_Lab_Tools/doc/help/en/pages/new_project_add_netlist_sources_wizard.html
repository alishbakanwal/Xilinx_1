Add Netlist Sources

The Add Netlist Sources page lets you add post-synthesis netlists to the current project. The Vivado® Design Suite can import EDIF, NGC, structural SystemVerilog, or structural Verilog format netlists as well as Vivado Design Suite design checkpoint (DCP) files. Design files are read into the design and compiled in the order listed. Refer to this link in the Vivado Design Suite User Guide: Using the Vivado IDE (UG893) for more information on compile order and the Hierarchy Update command.

Important: When working with EDIF and NGC files, the top cell name must match the name of the file it is contained in.
  • Add Files: Opens a file browser letting you select one or more netlist source files for including in the project. Selecting multiple files will add the files in the specified order to the file list.
    Tip: Alternatively, you can click and select Add Files.
  • Add Directories: Opens a file browser letting you select one or more directories to search for design source files. Specified directories will be searched and any netlists sources found will be added to the current project.
    Tip: Alternatively, you can click and select Add Directories.
  • Remove (): Removes selected files from the current project. The files are removed from the project but are not removed from the hard drive.
  • Move Up: When one or more files have been selected in the file list display area, the Up button () is activated letting you move the selected file or files up in the listed order.
  • Move Down: When one or more files have been selected in the file list display area, the Down button () is activated letting you move the selected file or files down in the listed order.
  • Copy sources into project: This checkbox lets you copy the specified design source files into the sources folder of the local project directory. Refer to this link in Vivado® Design Suite User Guide: System-Level Design Entry (UG895) for more information on using local and remote sources.
  • Add sources from subdirectories: This checkbox adds source files from the subdirectories under the directories specified using the Add Directories option.
