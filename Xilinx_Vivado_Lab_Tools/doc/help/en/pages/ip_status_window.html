IP Status Window

The IP Status window displays the results of the Tools > Report > Report IP Status command. This report lists the various IP used in the current project and displays the status for the IP in the current release of the Vivado® Design Suite. Selecting an IP in the report also selects that IP in other open windows, such as the Sources window, or in the IP integrator design canvas if the block design is open.

The IP Status window is a table window, which presents data about each IP in column form. Each IP listed has a checkbox that you can enable to select the IP for upgrading if needed. You can click the checkbox in the column header to select or deselect all IP in the report. The IP Status, Recommendation, and Change Log columns provide links to additional information for each IP in the report. After reviewing the IP Status report, select the IP to upgrade, and click the Upgrade Selected command.

You can also sort information in the table by clicking once or twice in the column header. To display and hide columns as well as manage column widths, right-click in the header of any column. See this link in the Vivado Design Suite User Guide: Using the Vivado IDE (UG893) for more information on working with data tables. Depending on the IP in the project there are also checkboxes along the top of the report for turning on or off the display of up-to-date IP, and IP with minor, major, or revision upgrades available.

The IP Status window toolbar includes the following commands:

Search Opens a search field to enter a search pattern. IP names or values matching the search pattern are displayed.
Collapse all Collapses any hierarchy in the IP Status report.
Expand all Expands all hierarchy in the IP Status report.
Rerun report Regenerates the IP Status report after making any changes to the current project.
Tip: Use this command when the IP Status window displays a note indicating that the report is out-of-date and recommends rerunning the report when needed.
