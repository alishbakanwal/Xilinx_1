Analog Settings

This dialog box lets you establish the waveform characteristics of an analog signal in the Waveform window. In the right-click menu of the timeline, you can define the Waveform Style of a signal as either digital, or analog. The Analog Settings dialog box lets you define the characteristics of the waveform for that signal.

  • Row Height: Specify the number of pixels to use for the analog signal waveform in the Waveform window.
  • Y Range: Define the range of values on the Y-axis for the analog signal waveform.
    • Auto: Have the Waveform window automatically define the Y value range based on the signal values encountered during simulation.
    • Fixed: Specify a minimum and maximum value for the analog signal waveform. The waveform will be drawn within the range of values specified.
      • Min: Specify the minimum value at the bottom of the waveform.
      • Max: Specify the maximum value at the top of the waveform.
  • Interpolation style: Specify the method of interpolating the signal waveform when the data is sparse. The tool can perform a Linear extension of the current waveform curve, or Hold the last value until the next value.
  • Off scale: Specify the method of handling the signal when its value exceeds the specified Y Range value. The tool can Hide the excess waveform, or Clip the signal at the Min or Max value to stay within the specified range, or Overlap the signal with neighboring signal waveforms above and below as needed.
  • Horizontal Line: Draw a horizontal line through the analog waveform at the Y Value specified as a center-line for the waveform.
    • Y Value: Specifies the value on the analog waveform where the horizontal line will be drawn if enabled.
