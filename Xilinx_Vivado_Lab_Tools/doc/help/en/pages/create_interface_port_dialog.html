Create Interface Port

The Create Interface Port dialog box lets you define a new external interface port to add to the current block design.

  • Interface name: Specifies he name of the interface port to create.
  • VLNV: Lists the VLNV (Vendor:Library:Name:Version) of currently defined interfaces available for use with the current block design.
  • Mode: Defines the interface as either a Master, Slave, or Monitor interface.
  • Connect to selected interface: Connects the new interface port to the selected ports or pins that are compatible.
    Tip: This checkbox is enabled when interface pins in the design are selected when you run the Create Interface Port command.

Click OK to create the new interface port and optionally, connect it.
