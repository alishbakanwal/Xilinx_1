Methodology Results

The Methodology Results window displays the results of the Report Methodology command. The window lists the violations of the methodology and timing rule checks performed, arranged by the associated rule name. Each violation shows the assigned severity and details of the violation, such as a list of related design objects. Selecting a violation displays details in the Violation Properties window and selects related design objects in other windows, such as the Netlist or Device windows.

The Methodology Results window is a table window, which presents data about design rule violations in column form. To display and hide columns as well as manage column widths, right-click in the header of any column. You can sort information in the table by clicking once or twice in the column header. See this link in the Vivado® Design Suite User Guide: Using the Vivado IDE (UG893) for more information.

The Methodology Results toolbar includes the following commands:

Search Opens a search field to enter a search pattern. Methodology violations matching the search pattern are displayed.
Collapse all Enabled when the Methodology violations are grouped by rule. Collapses the violation hierarchy.
Expand all Enabled when the Methodology violations are grouped by rule. Expands the violation hierarchy.
Group by Rule Groups the Methodology violations by rule hierarchy. Violations are simply listed by name when this command is not enabled.
