Create and Package New IP

The Create and Package IP wizard lets you define a new IP core from the current project, a specified directory, or a block design in the current project. This wizard also lets you create an AXI4 peripheral.

  • Create peripheral, package IP, or package a block design
  • Package your current project
  • New IP creation
  • Packaging your Block Design
  • Package a specified directory
  • Edit in IP packager project name
  • Peripheral details
  • Add interfaces
  • Create peripheral
