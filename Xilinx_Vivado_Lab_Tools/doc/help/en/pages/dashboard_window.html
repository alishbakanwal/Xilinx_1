Dashboard

A Dashboard window acts as a collection of windows within the logic analyzer feature of the Vivado Design Suite or the Vivado Lab Edition. The logic analyzer is used to interact with ILA, VIO, SysMon, and JTAG-to-AXI Master debug cores in your design, and programmed into the current device.

The Dashboard window lets you build a collection of associated windows to provide you a meaningful view of the activity related to that core. The Hardware ILA window is an example of a Dashboard windows in the logic analyzer. It includes all of the windows associated with a particular ILA debug core programmed into the current device. However, you can also create custom dashboards using the New Dashboard dialog box, to create a dashboard with multiple waveform windows for example, to see the activity on multiple cores in one dashboard.

The Dashboard window also includes a Dashboard Options feature that can be accessed from a slim button hidden along the left side of the dashboard. Opening the Dashboard Options displays a list with the current device, the debug cores programmed into the device, and the windows associated with each core. The list has checkboxes to let you add or remove a window to the dashboard.

The Dashboard window appears in the Vivado® IDE workspace. For more information on the features of workspace windows refer to this link in the Vivado Design Suite User Guide: Using the Vivado IDE (UG893).
