Program eFUSE Registers Summary

The Program eFUSE Registers Summary page reports the changes to be written to the one-time writable eFUSE registers.

  • Click Back to return to prior pages to change selections as needed.
  • Click Finish to write to the eFUSE registers as indicated. A final warning dialog box is displayed prior to the actual writing of the eFUSE bits. This is your FINAL warning.
  • Click Cancel to close the wizard without writing any eFUSE registers.
