Run Connection Automation

The Run Connection Automation dialog box automatically connects selected pins and interfaces between other IP present in the design, or to external I/O ports in the design. In case a targeted board platform is used, this feature can also help connect to the components present on the board. The tree view of the dialog box lists the available ports and interfaces to connect. You can enable or disable the checkbox next to a port or interface to enable or disable the designer assistance feature.

Selecting a port or interface in the tree view displays the connection options for the selected object. The dialog box also displays a brief description of the automation process that will be run for that object. The available options depend on the port or interface selected.

Click OK to run the block automation with the selected options.
