Report Noise

This dialog box lets you perform analysis of the switching noise levels associated with the current design. Noise analysis provides estimates of the disruption that simultaneously switching outputs can cause on other output ports in the I/O bank. Too much switching noise results in unreliable I/O.

  • Results name: Specify the name to apply to the report in Noise Results window in the Vivado® IDE.
  • Export to file: Write the results of the Noise report into the specified file. You can specify the file format to export as either CSV, HTML, or text.
  • Phase: Consider the clock phase when analyzing signals in different clock domains for a more accurate result.
  • Open in a new tab: Open the Noise report in a new tab of the Noise Results window in the Vivado IDE. If this checkbox is not enabled, the new report overwrites the last report that was opened.
