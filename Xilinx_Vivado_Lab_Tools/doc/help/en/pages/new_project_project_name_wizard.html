Project Name

  • Project Name: Specifies the name of the new project to create.
  • Project Location: Specifies the disk or network location to create the current project. This field also offers a file browser to let you navigate to specify a project location.
  • Create project subdirectory: Creates a subdirectory with the same name as the project in the specified project location to store the project files and folders. If you do not enable this checkbox, the project files and folders are written directly into the specified project location.

Click Next to define the Project Type.
