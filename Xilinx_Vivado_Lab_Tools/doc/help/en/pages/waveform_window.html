Waveform Window

In the Vivado simulator, an HDL object is an entity that can hold a value, such as a wire, signal, or register. When you launch the Vivado simulator, the Waveform window displays with top-level HDL objects from the test bench. You can add other HDL objects from the Scope or Objects windows and run the simulation.

Important: The Waveform window does not necessarily display all of the captured simulation data for all HDL objects; it can display just the objects of interest to you at that time. After simulation, you can add or remove signals in the Waveform window as long as they have been logged, or captured during simulation. For more information on logging signals, refer to this link in the Vivado Design Suite User Guide: Logic Simulation (UG900), or to the log_wave Tcl command.

The Waveform window is split into two regions, with the object names and values listed on the left, and the simulation timeline and waveforms displayed on the right. The right-click menus differ between the two regions. On the object side commands are available to configure the display of object names and values, group objects, and add dividers to help make the simulation results more understandable. With a signal object selected, the Report Drivers command in the right-click menu prints the name and value of the driving signal, as well as the current value of the selected object. On the waveform side the right-click menu commands are available to manage the timeline, add markers, and move up and down the timeline to different transition events. Selecting an object on the left, provides focus for some of the waveform commands on the right.

The Waveform window toolbar has the following commands for navigating the simulation timeline through the displayed waveforms:

Go to Cursor Used when the waveform display is zoomed in, this command centers the display on the current cursor location.
Go to Time 0 Move the waveform cursor to the start of the simulation timeline.
Go to Last Time Move the waveform cursor to the end of the simulation timeline.
Previous Transition With a signal selected on the objects side of the Waveform window, this command moves the cursor from its current location to the point of the prior transition on the selected signal.
Next Transition With a signal selected on the objects side of the Waveform window, this command moves the cursor from its current location to the point of the next transition on the selected signal.
Add Marker Insert a marker at the current cursor location.
Previous Marker With one or more markers added to the timeline, move the cursor from its current location to the point of the prior marker.
Next Marker With one or more markers added to the timeline, move the cursor from its current location to the point of the next marker.
Swap Cursors With multiple cursors displayed on the timeline, switch between the cursors.
Note: You can add a second cursor to the timeline by holding the Ctrl key, selecting the cursor, and clicking and dragging the mouse. See this link in the Vivado Design Suite User Guide: Logic Simulation (UG900) for details.
Snap to Transition When moving the cursor, snap it to the nearest transition point in the timeline. Disabling this option allows the free movement of the cursor along the timeline.
Floating Ruler With multiple cursors displayed on the timeline, this command displays or hides the floating ruler at the bottom of the waveform display. The ruler can be used to measure time intervals between events.

You can configure what is displayed in the waveform window, and pan and zoom to display the level of detail needed to see specific areas of a waveform, or time ranges of the simulation. The toolbar menu has the following commands for controlling what is displayed in the Waveform window.

Options Display selectable options to configure what is displayed in the Waveform window, such as setting signal radix, and truncating signal names. For more information refer to this link in the Vivado Design Suite User Guide: Logic Simulation (UG900).
Save Save the Waveform window to a waveform configuration file.
Zoom Fit View the complete simulation runtime in the Waveform window.
Zoom In Display a smaller period of the simulation runtime, with greater detail of the signal waveforms and simulation events.
Zoom Out Display more of the simulation runtime.
The Waveform window also supports the use of mouse strokes to zoom the display. Click and hold the left mouse button, and draw a line or box to zoom into or out of the device. You can use the scroll bars shown on the right side or bottom of the Waveform window to scroll the display and see a new area.
Tip: Analog waveforms use an extended mouse stroke pattern as discussed at this link in the Vivado Design Suite User Guide: Logic Simulation (UG900).

The Waveform window appears in the Vivado® IDE workspace. For more information on workspace windows refer to this link in the Vivado Design Suite User Guide: Using the Vivado IDE (UG893).
