Block Design Signals Window

The Signals window is split into two tabs: Clocks and Resets, which can be accessed from the bottom of the window. The window displays a hierarchical tree view of either clocks or resets, letting you quickly view and make connections to signals. When you select objects in the Signals window, the same object is selected in the Design Canvas window.

Connected signals are sorted from driver pins to load pins, and unconnected signals are listed in a separate folder. You can select an unconnected signal and use the right-click menu to access the Make Connection command, or by selecting and dragging unconnected clock loads onto the appropriate driver.

You can use the Group by Clock Domain command () to group the clock connections under a clock domain. This lets you select and view the whole clock net rather than just the individual driver and load pins.
