Export ILA Data

This dialog box lets you export the captured data from a selected ILA debug core. This uploads the captured data from the ILA debug core and writes it to an ILA archive file containing the waveform database file and configuration file, a waveform CSV and VCD file, and the debug probes file.

  • ILA Core: Select from a list of ILA Debug cores programmed into the current device.
  • Format: Specify the file format of the data file to write. The available formats are the native ILA archive, CSV, and VCD.
  • File: Specify the path and file name for the data file to write.
