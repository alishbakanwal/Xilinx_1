Create Interface Definition

An interface definition lets you group functional signals into a common interface to connect between IP in an IP integrator block diagram. This dialog box lets you define a new blank interface or use an existing interface definition to create a new interface.

  • Create a new interface definition: Creates new blank interface definition.
  • Extend from: Creates a new interface using an existing interface definition as a starting point. Click the Browse button () to open the Interface Definition Chooser dialog box to select the starting interface.
  • Identification: The Vendor:Library:Name:Version (VLNV) settings combine to create the unique identifier for the new interface definition.
  • Location: Defines the IP repository to write the new interface definition into. You must add the specified repository to the Vivado® IP Catalog using the IP Project Settings dialog box to enable proper use of the interface definition by the Vivado tools.

Click OK to open the Interface Definition window to edit and complete the definition of the interface.
