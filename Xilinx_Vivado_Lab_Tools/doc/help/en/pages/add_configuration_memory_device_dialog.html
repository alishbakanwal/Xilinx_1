Add Configuration Memory Device

The process whereby the design specific data is loaded or programmed into a Xilinx device is called configuration. Xilinx devices can be configured from an external flash memory device, or they can be configured by an external smart source, such as a microprocessor, DSP processor, or board tester. Two configuration datapaths include the serial peripheral interface (SPI) that is used to minimize the device pin requirements for configuration, and a parallel byte peripheral interface (BPI) used for higher performance configuration. The Add Configuration Memory Device dialog box lets you select a configuration memory device to store the bitstream of the current design, and for configuring and booting the hardware device.

The Vivado device programmer can indirectly program flash-based configuration memory devices via JTAG, using a special configuration memory file (cfgmem) created with the Write Configuration Memory File command in the Vivado Hardware Manager. The configuration memory device can be programmed with the bitstream and other data from the cfgmem file using the Program Configuration Memory Device command.

  • Filter: Filter the listed configuration memory devices according to the following criteria.
    • Manufacturer: Choose from one of the supported manufacturers of configuration memory devices.
    • Density: Select the memory size for the device.
    • Type: Filter according to byte peripheral interface (BPI) or serial peripheral interface (SPI) type devices, or show all types.
    • Width: Select the programming data width for the device.
    Tip: The Reset Filters command removes the filters and restores the default settings.
  • Search: Lets you further filter the devices displayed in the results table by pattern matching the specified search string.

The Select Configuration Memory Part table lists the available devices based on the filters applied, or the search string pattern. The resulting table displays several columns of information, such as Name, Part, Manufacturer, and Type. To sort the results table, click in a column header. If you click once, the column is sorted in ascending order. If you click twice, the column is sorted in descending order. To sort multiple columns, press and hold the Ctrl key while clicking in the column headers.

Select a configuration memory device and click OK. The configuration memory device is added to the current device, and displayed in the Hardware window.
