Clock Interaction Window

The Clock Interaction report analyzes timing paths that cross from one clock domain (the source clock) into another clock domain (the destination clock). The Clock Interaction window displays a grid matrix of all the clocks in the design and provides a visual representation of their interaction. Use the Hide unused clocks toolbar button () to reduce the size of the matrix. Place the cursor over a cell in the matrix to get a quick summary of the clock interaction. The colors displayed in the matrix indicate the interaction between the source and destination clocks. Use the Show legend command () to show or hide the matrix legend. Refer to this link in the Vivado Design Suite User Guide: Design Analysis and Closure Techniques (UG906) for more information on matrix colors.

The window also displays a table view of the clock interactions, which presents timing data for the source to destination clock interactions. To display and hide columns as well as manage column widths,right-click the header of any column. You can also sort information in the tabl by clicking once or twice in the column header. See this link in the Vivado Design Suite User Guide: Using the Vivado IDE (UG893) for more information. From the right-click menu, you can report timing for selected timing paths, apply common design constraints to selected paths, or export the table to a spreadsheet.

The Clock Interaction window appears in the Vivado® IDE workspace . The workspace displays graphical windows, such as the Device, Package, or Schematic windows, and other windows that require more display area, such as the Text Editor and Project Summary. Open workspace windows are tabbed in the workspace. Click a tab to bring the window to the front of the workspace.

The workspace can also be divided into multiple horizontal or vertical spaces, allowing you to view multiple workspace windows at the same time. Right-click in the tab of a workspace window and select New Horizontal Group or New Vertical Group to divide the workspace, or select Tile Horizontally or Tile Vertically to tile all the open workspace windows. For more information refer to this link in the Vivado Design Suite User Guide: Using the Vivado IDE (UG893).
