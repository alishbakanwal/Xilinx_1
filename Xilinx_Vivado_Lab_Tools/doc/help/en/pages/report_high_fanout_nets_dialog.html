Report High Fanout Nets

This dialog box lets you report the high fanout nets in the current design. It is important to look at the list of high fanout signals post synthesis as well as post physical optimization. It is often difficult to meet timing on nets that have many thousands of endpoints. You should drive high fanout nets with a fabric register (FD*), which is easier to replicate and relocate during physical optimization.

  • Results name: Specify the name to apply to the report in High Fanout Nets window in the Vivado IDE.
  • Export to file: Write the results of the High Fanout Nets report into the specified file. You can specify the file format to export as either text or XML.
  • Open in a new tab: Open the High Fanout Nets report in a new tab of the High Fanout Nets window in the Vivado IDE. If this checkbox is not enabled, the new report overwrites the last report that was opened.
