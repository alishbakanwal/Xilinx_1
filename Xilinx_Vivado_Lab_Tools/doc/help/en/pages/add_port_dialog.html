Add Port

The Add Port dialog box lets you define a new port for an interface definition. Specify the Name, Description, and Default Value for the port. The Default Value should be specified as a positive integer.

Tip: Although you can modify the port using many of the fields in the Interface Definition window, you cannot modify the Name field after the port is created.

For both Master and Slave modes for the interface, you must define the Width, Presence, and Direction of the port. Presence specifies the port as required, optional, or illegal for the interface in master or slave mode.

Finally, you can specify the type of signal carried by the port. The Type can be defined as address and data, or clock, or reset.

Click OK to add the new port to the interface definition.
