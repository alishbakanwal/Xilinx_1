Create Peripheral

The Create Peripheral page presents a summary of the actions that are taken when you click Finish and offers the following tasks to perform next:

  • Add IP to the repository: Creates the AXI4 peripheral IP and adds the IP location as a User Repository to the Vivado IP Catalog.
  • Edit IP: Creates the AXI4 peripheral IP, adds the IP to the Vivado® IP catalog, and opens the IP in an Edit IP project to let you edit additional details of the packaged IP. The name of the Edit IP project includes the name of the newly created peripheral IP and is located in the specified IP location.
  • Verify peripheral IP using AXI4 BFM Simulation interface: Creates the AXI4 peripheral IP and adds it to the Vivado IP catalog. The Vivado tools create a Vivado IP integrator block design using the peripheral IP and launch the Vivado simulator using an AXI4 BFM simulation interface.
    Important: Licensing is required to use AXI4 BFM simulation. Refer to this link in the Vivado Design Suite User Guide: Logic Simulation (UG900).
  • Verify peripheral IP using JTAG interface: Creates the AXI4 peripheral IP and adds it to the Vivado IP catalog. The Vivado tools create a Vivado IP integrator block design, so you can debug the peripheral IP in hardware using the JTAG-to-AXI Master debug core. Refer to this link in the Vivado Design Suite User Guide: Programming and Debugging (UG908) for more information on using the JTAG-to-AXI debug core.

Click Finish to create the peripheral IP and continue with the selected option.
