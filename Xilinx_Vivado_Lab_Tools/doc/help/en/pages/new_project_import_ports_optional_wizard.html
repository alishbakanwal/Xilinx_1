Import Ports (Optional)

The Import Ports page lets you import a comma separated value (CSV) file or XDC file, containing port definitions for use in the I/O planning project.

  • Import CSV: Set this radio button, and specify a CSV file. You can use File > Export I/O Ports command from the main menu to create a CSV.
  • Import XDC: Set this radio button and select an XDC file. The XDC file uses Tcl commands such as create_port to define the port list for the design.
  • Do not import I/O ports at this time: This radio button bypasses importing an I/O port list. You can skip this now but will need to define or import ports later.
