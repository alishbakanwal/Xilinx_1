Create HDL Wrapper

The Create HDL Wrapper dialog box lets you define a new HDL wrapper for the current block design. The wrapper defines the block design as the top-level design, so you can synthesize, implement, and generate a bitstream for the block design.

  • Copy generated wrapper to allow user edits: Creates a copy of the HDL wrapper and adds it to the project source files so you can edit it as needed.
    Important: You must manually update the wrapper whenever the external interface of the block design changes. The wrapper is not updated automatically by the Vivado® tools.
  • Let Vivado manage wrapper and auto-update: Instructs the Vivado tools to manage the new wrapper and update it as needed when the block design changes.

Click OK to create the HDL wrapper and add it to the top-level design.
