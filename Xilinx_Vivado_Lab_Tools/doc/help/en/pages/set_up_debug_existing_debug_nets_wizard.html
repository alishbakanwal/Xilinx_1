Existing Debug Nets

This page of the Set Up Debug wizard specifies the actions to take when existing debug nets are found in the design.

Tip: This page only opens when existing debug nets are found in the current design.

The available actions include:

  • Continue debugging ## nets connected to existing debug core: This preserves the existing ILA cores and net assignments, and lets you add new nets to debug.
  • Only debug new nets: This eliminates the existing ILA cores and net assignments and lets you assign new nets to debug.
  • Disconnect all nets and remove debug cores: This eliminates the existing ILA core and net assignments and does not let you assign new nets to debug.
