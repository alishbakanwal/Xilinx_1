Package Window

The Package window displays a graphical representation of the external package for the target device. This includes graphical representations of different pin types, such as configuration pins, clock-capable (CC) pins, VCC and GND pins, and GT pins, so that you can quickly differentiate between the available pins. You can also enable the display of differential pin pairs, to see related pins, by clicking the Show Differential IO Pairs toolbar button ().

To interactively assign I/O ports to package pins, you can select ports in the I/O Ports window and drag and drop them onto a pin in the Package window. For controlling placement of objects the toolbar has the following commands:

Place Ports Defines how you will place selected ports: in an I/O Bank, in an area, or sequentially. Refer to this link in the Vivado Design Suite User Guide: I/O and Clock Planning (UG899) for more information on placing I/O ports.
Autocheck I/O Placement Automatically runs I/O placement design rule checks during interactive placement. Enabling these DRC checks prevents you from violating design rules for the proper placement of ports.
Tip: The tool will prevent you from violating design rules with this enabled. However, if there are multiple errors, you may need to turn this off to correct the port assignments.

By default, the Package window shows a view looking down at the top of the package, and the pins are arrayed accordingly. However, if you prefer, you can click the Show Bottom View toolbar button () to flip the package over.

You can pan and zoom to display the level of detail needed to see specific package pins. The toolbar menu has the following commands for controlling what is displayed.

Previous Restore the previous view of the Package window, including zoom-level and position.
Next Return from the previous view.
Tip: This command is only available after using the Previous command.
Zoom In Display a closer view of the package pins.
Zoom Out Display more area of the package.
Zoom Fit View the whole package.
Select Area Draw a rectangle to select objects in the Package window.
Fit Selection Zoom into and center the Package window around the currently selected objects.
Autofit Selection Automatically pan and zoom the Package window to center an object that has been selected in another window.
The Package window also supports the use of mouse strokes to zoom the display. Click and hold the left mouse button, and draw a line or box to zoom into or out of the package. You can use the scroll bars shown on the right side or bottom of the Package window to scroll the display and see a different area.
Tip: The scroll bars are only visible when the display area is zoomed into the package.

The Package window appears in the Vivado® IDE workspace. For more information on workspace windows refer to this link in the Vivado Design Suite User Guide: Using the Vivado IDE (UG893).
