Design Analysis Results

The Design Analysis Results window displays the results of the Report Design Analysis command. The window is divided into two sections. The left side has a table of contents, including General Information related to the design and the report command, and Setup and Hold Path Characteristics for reported timing paths.

The Design Analysis Results are displayed in table form on the right. The results are displayed for the timing paths, the complexity analysis, and the congestion analysis performed by the report, depending on which options you selected. To display or hide columns in the table, as well as manage column widths, right-click in the header of any column. You can also sort information in the table by clicking once or twice in the column header. See this link in the Vivado® Design Suite User Guide: Using the Vivado IDE (UG893) for more information on working with table windows.

The Design Analysis Results window includes toolbars for the Tree View on the left side, and the Results Table on the right:

  • The left side Tree View toolbar includes the following commands:
    Search Opens a search field to enter a search pattern.
    Collapse all Collapses the left side hierarchy of the Design Analysis Results window.
    Expand all Expands the left side hierarchy of the Design Analysis Results window.
    Previous Restore the previous view of the Design Analysis Results window.
    Next Return from the previous view.
    Tip: This command is only available after using the Previous command.
    Rerun Rerun the Report Design Analysis command.
    Show only sections with content Enable or disable this option to show or hide the sections of the Design Analysis report that do not have any content or results.
  • The Results Table toolbar includes the following commands:
    Search Opens a search field to enter a search pattern. The contents displayed in the Results Table are filtered to match the search pattern.
    Schematic Opens a Schematic window containing the currently selected objects.
    Mark Selected Objects Marks selected objects of the Results Table in other windows such as the Device and Netlist windows.
    Tip: Timing path startpoints are marked in green, the endpoints are marked in red, and any through points are marked in the selected color.
    Show Percentage Shows values in the Results Table as percentages rather than as values.
    Select Clock Paths when Timing Paths are selected Selects the source and destination clock paths along with the datapath.
    Hide Unused Toggle to hide or display empty or unused columns in the Results Table.
