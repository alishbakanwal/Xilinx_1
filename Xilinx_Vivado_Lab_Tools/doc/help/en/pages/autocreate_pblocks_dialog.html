Auto-create Pblocks

This dialog box lets the Vivado tool automatically create Pblocks in order to create a floorplan for hierarchical modules in the current design. The tool will analyze the hierarchy of the design and select the cells with the greatest number of leaf cells to assign to a Pblock.

  • Pblock to partition: Create Pblocks for the top-level (ROOT partition) of the design, or further partition an existing Pblock into nested blocks.
  • Maximum number of Pblocks to generate: Specify the number of Pblocks to generate from the current design. The default is 10 Pblocks.
  • Process cells with leaf cell count of at least: Specify the minimum number of leaf cells a hierarchical cell must have in order to be considered for assignment to a Pblock.

The Preview region will list the hierarchical cell instances that will be assigned to Pblocks. Click OK to automatically create Pblocks and assign the elements of the hierarchical cells to those blocks.
