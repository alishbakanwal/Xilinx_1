Invalid Top Module

This dialog box is displayed when the specified Top Module for a new Reconfigurable Module (RM) cannot be found in the files defining the module. It lets you specify how to proceed in this case:

  • Specify a new top module: Enter a text string to name the new top module, or use the browse button () to display a list of candidates.
  • Ignore and continue with invalid top module: Leave the problem to be resolved at a later time.
  • Automatically pick new top module: Let the tool find an appropriate top module based on the available source files.

Click OK after you have made your selection.
