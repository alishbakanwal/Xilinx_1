Package a Specified Directory

  • Directory: Specifies the location of the source files for the IP.
    Important: The Vivado Design Suite uses inference rules to package a directory as an IP. The specified directory must adhere to these rules.
  • Package as a library core: Defines the packaged IP as a library core. A library core cannot be customized from the Vivado IP Catalog on its own but can be incorporated as a hierarchical module when developing other IP.
