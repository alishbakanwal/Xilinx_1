XADC Plot

The XADC Plot window displays a graph of the selected sensors of the System Monitor (SYSMON) core on the current hardware device. The SYSMON Analog-to-Digital Converter (ADC) measures die temperature and voltages via on-chip temperature and supply sensors. The ADC provides a high-precision analog interface for a range of applications.

The toolbar for the XADC Plot window includes the following commands:

Add selected sensors Displays a list of sensors on the SYSMON to let you select and add to the plot graph.
Tip: The Y-axis of the plot will change to reflect the scale of the various sensors being displayed.
Remove selected sensors Removes selected sensors from the XADC Plot window.
Refresh System Monitor Refreshes the properties of the SYSMON debug core with values from the physical System Monitor on the current device.
Set JTAG Scan Rate Determines the frequency that the XADC Plot window will be updated from the physical System Monitor on the current device.
