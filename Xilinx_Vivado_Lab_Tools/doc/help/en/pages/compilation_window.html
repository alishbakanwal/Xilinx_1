Log Window

The Log window shows the compilation and execution logs for Vivado® synthesis, implementation, and simulation displayed in separate Synthesis, Implementation, and Simulation tabs. In the case of out-of-context design modules, IP, or block designs, the synthesis logs for each separate OOC objects is provided. The log messages are displayed for each of these processes in a separate tab at the bottom of the Log window. Click on a tab to view the specific log of interest.

The Log window is updated in real-time as the process is running. You can watch the process log scrolling up the window as it runs. Click the Pause output toolbar button () to stop the scrolling and read the log data. The scrolling pauses, but the process continues, and the log is captured while paused. Click the Pause output toolbar button again to resume the scrolling of the Log window.

Search through the Log window using the Show Find toolbar button (), which opens a search box at the bottom of the Log window. The Find function lists the previously used search terms and lets you define features of the search, such as Match Case and Whole Words. You can use the Find Previous and Find Next buttons to scroll through the search results.

Select portions of the data in the Log window, and click the Copy toolbar button () to copy the data to a text editor or another location.
