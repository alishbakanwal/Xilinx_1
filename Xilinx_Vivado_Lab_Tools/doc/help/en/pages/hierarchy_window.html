Hierarchy Window

The Hierarchy window displays a hierarchical view of the currently opened design, or a selected portion of the design. The hierarchical modules are displayed with relative sizes reflecting the amount of device resources consumed. To open the Hierarchy window, select a design object in a window such as the Netlist window and select the Show Hierarchy command from the right-click menu.

The Hierarchy window toolbar features the following commands:

Options Display selectable options to control what is displayed in the Hierarchy window.
Show tree view Toggle to display the hiearchy as a tree view from the top-level down, or as a block design.
Go up one level Move up one level of the hierarchy.
Tip: This command is only available when you have used the Push into selected cell command to move down into the hierarchy from the initial display.
Push into selected cell Move down into the hierarchy of the selected cell.
Zoom In Display a closer view of the hierarchy.
Zoom Out Display more area of the hierarchy.
Zoom Fit View the whole hierarchy.
Select Area Draw a rectangle to select objects in the Hierarchy window.
Fit Selection Zoom into and center the Hierarchy window around the currently selected objects.
Autofit Selection Automatically pan and zoom the Hierarchy window to center an object that has been selected in another window.
The Hierarchy window also supports the use of mouse strokes to zoom the display. Click and hold the left mouse button, and draw a line or box to zoom into or out of the hierarchy. You can use the scroll bars shown on the right side or bottom of the Hierarchy window to scroll the display and see a new area.
Tip: The scroll bars are only visible when the display area is zoomed into the hierarchy.

The Hierarchy window appears in the Vivado® IDE workspace. For more information on workspace windows refer to this link in the Vivado Design Suite User Guide: Using the Vivado IDE (UG893).
