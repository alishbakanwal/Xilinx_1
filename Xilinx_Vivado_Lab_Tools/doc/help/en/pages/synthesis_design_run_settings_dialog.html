Synthesis Design Run Settings

The Synthesis Design Run Settings dialog box lets you define the strategy and options to use for the selected synthesis run. The default options for the selected strategy are displayed in a table, and you can modify options or values to customize the strategy.

  • Strategy: Specifies the predefined Vivado® Design Suite strategies, or lets you create your own strategy by modifying the options for the selected strategy.
    Tip: When you change the options for a selected strategy, as indicated by an asterisk (*) in the strategy name, you can click the Save Strategy As button () to save those options as a new user-defined strategy.
  • Description: Provides a brief summary of the goal of the selected strategy. You can define the description when saving user-defined strategies.
  • Options: Shows a table view of the options available in the selected synthesis strategy. Select an option in the table to display a description of the option. You can use the More Options field at the bottom of the Options table to add any additional command options for the synth_design command.
