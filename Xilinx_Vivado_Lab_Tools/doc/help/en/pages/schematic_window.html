Schematic Window

The Schematic window displays a schematic view of the netlist for the currently opened design, or a selected portion of the design. To open the Schematic window, select a design object in any window, such as the Netlist window, and click the Schematic toolbar button ().

For interactively exploring the schematic hierarchy, or adding and removing items to the schematic, the toolbar features the following commands:

Expand Expand the selected hierarchical objects in the Schematic window.
Collapse Collapse the selected hierarchical objects in the Schematic window.
Remove Remove the selected objects from the Schematic window.
Add Add objects that are selected in other windows, such as the Netlist window, to the schematic to create a custom schematic of selected logic

You can zoom into the Schematic window to see portions of the design. The toolbar menu has the following commands for controlling what is displayed.

Previous Restore the previous view of the Schematic window, including zoom-level and position.
Next Return from the previous view.
Tip: This command is only available after using the Previous command.
Zoom In Display a closer view of the schematic.
Zoom Out Display more area of the schematic.
Zoom Fit View the whole schematic.
Select Area Draw a rectangle to select objects in the Schematic window.
Fit Selection Zoom into and center the Schematic window around the currently selected objects.
Autofit Selection Automatically pan and zoom the Schematic window to center an object that has been selected in another window.
The Schematic window also supports the use of mouse strokes to zoom the display. Click and hold the left mouse button, and draw a line or box to zoom into or out of the schematic. You can use the scroll bars shown on the right side or bottom of the Schematic window to scroll the display and see a new area.
Tip: The scroll bars are only visible when the display area is zoomed into the schematic.

The Schematic window appears in the Vivado® IDE workspace. For more information on workspace windows refer to this link in the Vivado Design Suite User Guide: Using the Vivado IDE (UG893).
