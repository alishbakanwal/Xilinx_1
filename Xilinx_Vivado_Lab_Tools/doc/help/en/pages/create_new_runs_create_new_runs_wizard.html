Create New Runs

The Create New Runs wizard guides you through the process of creating new synthesis and implementation runs. The first step is to specify which types of design runs to create:

  • Synthesis: Create a new synthesis run without an implementation run.
  • Implementation: Create a new implementation run.
  • Both: Create both a synthesis and implementation run.

When you select the Create Synthesis Runs or the Create Implementation Runs commands from the Flow Navigator context menu, the first page of the Create New Runs wizard informs you that you are creating synthesis or implementation runs. It does not offer a choice.

Tip: For netlist-based projects where no synthesis run is required, the first page of the Create New Runs wizard is skipped, and the Configure Implementation Runs page is opened.

Click Next to open the Configure Synthesis Runs page or the Configure Implementation Runs page.
