Add Existing IP

The Add Existing IP page lets you specify an existing customized IP file to add to the project. The existing IP can be in a format native to the Vivado® Design Suite (XCIX or XCI) or can be a legacy IP format supported by the Core Generator™ tool (XCO). You can add IP directly from the Vivado IP Catalog as you are developing a design.

  • Add Files: Opens a file browser letting you select one or more existing IP files (.xcix, .xci, .xco). Selecting multiple files will add them in the specified order to the page.
    Tip: Alternatively, you can click and select Add Files.
  • Add Directories: Opens a file browser letting you select one or more directories to search for existing IP files (.xcix, .xci, .xco). Specified directories will be searched and any IP files found will be added to the current project.
    Tip: Alternatively, you can click and select Add Directories.
  • Remove (): Removes selected files from the current project. The files are removed from the project but are not removed from the hard drive.
  • Move Up: When one or more files have been selected in the file list display area, the blue Up arrow button () is activated on the left hand menu, letting you move the selected file or files up in the listed order. Design files are read into the design and compiled in the order listed. Refer to this link in the Vivado Design Suite User Guide: Using the Vivado IDE (UG893) for more information on compile order and the Hierarchy Update command.
  • Move Down: When one or more files have been selected in the file list display area, the blue Down arrow button () is activated on the left hand menu, letting you move the selected file or files down in the listed order.
  • Copy sources into project: This checkbox lets you copy the specified IP files under the sources folder of the local project directory.
