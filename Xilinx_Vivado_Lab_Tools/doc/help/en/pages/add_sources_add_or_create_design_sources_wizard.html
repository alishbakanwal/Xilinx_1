Add or Create Design Sources

The Add or Create Design Sources page lets you add an existing RTL file or MEM file to the current project or create a new, empty file.

  • Add Files: Opens a file browser letting you select one or more design source files for including in the project. Selecting multiple files will add the files in the specified order to the page.
    Tip: Alternatively, you can click and select Add Files.
  • Add Directories: Opens a file browser letting you select one or more directories to search for design source files. Specified directories will be searched and any design source files found will be added to the current project.
    Tip: Alternatively, you can click and select Add Directories.
  • Create File: Opens the Create Source File dialog box in which you can specify the file type, file name, and file location to write the file to. An empty source file will be created and added to the project. When the Add Sources wizard is finished, a module editor will open for the newly created source files. See the Define Module dialog box for more information, or refer to this link in the Vivado® Design Suite User Guide: System-Level Design Entry (UG895).
    Tip: Alternatively, you can click and select Create Files.
  • Remove (): Removes selected files from the current project. The files are removed from the project but are not removed from the hard drive.
  • Move Up: When one or more files have been selected in the file list display area, the blue Up arrow button () is activated, letting you move the selected file or files up in the listed order. Design files are read into the design and compiled in the order listed. Refer to this link in the Vivado Design Suite User Guide: Using the Vivado IDE (UG893) for more information on compile order and the Hierarchy Update command.
  • Move Down: When one or more files have been selected in the file list display area, the blue Down arrow button () is activated on the left, letting you move the selected file or files down in the listed order.
  • Scan and add RTL include files into project: This checkbox scans the added RTL files and adds any referenced Verilog 'include files into the local project directory structure as well.
  • Copy sources into project: This checkbox lets you copy the specified design source files under the sources folder of the local project directory. Refer to this link in Vivado® Design Suite User Guide: System-Level Design Entry (UG895) for more information on using local and remote sources.
  • Add sources from subdirectories: This checkbox adds source files from the subdirectories under the directories specified using the Add Directories option.
