Configure Project

The Configure Project page lets you configure the Microblaze processor and the IP that are added to the example block design in the Vivado IP Integrator.

  • Microblaze: Configure elements of the processor.
    • Local Memory: Specify the amount of memory dedicated to the processor.
    • DDR4: Add a DDR4 Memory Interface IP to the design.
      Tip: The DDR4 switch affects the data and instruction caches and the DMA access mode for the Ethernet controller.
    • Cache: Specify sizes for both the data and instruction caches. This option is disabled when DDR4 is disabled.
    • Ethernet: Add the AXI Ethernet IP with FIFO or DMA to the block design. The default mode is FIFO. When DDR4 is enabled, the DMA mode can be used.
  • General Purpose IOs: Add IP for the specified I/O interfaces to the example block design. You can also configure the UART IP.
