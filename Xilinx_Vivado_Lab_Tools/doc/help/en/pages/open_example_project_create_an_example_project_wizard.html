Create an Example Project Wizard

The Create Example Project wizard helps you create example projects for RTL designs with IP from the IP catalog, post-synthesis netlist designs, and Zynq and Microblaze embedded processor designs using the Vivado® IP Integrator. The flow used in creating an example project includes the following steps:

  • Specify the project template
  • Define the project name and location
  • Define the default part or board for the design
  • Optionally configure the design
  • Summary page
