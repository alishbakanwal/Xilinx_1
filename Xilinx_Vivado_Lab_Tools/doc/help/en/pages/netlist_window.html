Netlist Window

The Netlist window displays the netlist for the currently open elaborated, synthesized, or implemented design. A single project can support many different synthesis or implementation results, targeting different parts or boards. Each of these variations can result in a different netlist for the project.

The Netlist window shows a hierarchical tree view of the netlist, which lists the nets and leaf cells, or primitive logic, at each level of the design hierarchy. Objects in the Netlist window are displayed with icons representing the state of netlist logic, such as primitive or leaf cells, hierarchical cells, and hierarchical cells assigned to Pblocks. Refer to this link in Vivado® Design Suite User Guide: Using the Vivado IDE (UG893) for more information on understanding these icons.

Selecting an object in the Netlist window results in a related object being selected or highlighted in other windows, such as the Device window or I/O Ports window. You can use the right-mouse menu to access multiple commands for exploring the design, such as Go To Source to view the RTL source of a specific cell in the design, or the Show Hierarchy command to view a graphical representation of the netlist, or the Highlight or Mark commands to track specific design objects in other windows of the Vivado IDE.

In the Netlist window, use the Schematic toolbar button () to open a schematic of the design containing the selected netlist objects. If no object is selected, the schematic is generated for the top-level of the design.

Objects selected in other windows are also selected in the Netlist window. Enable the Scroll toolbar button () in the Netlist window to automatically scroll to display the selected item so you do not need to search for it.

The hierarchical tree automatically expands to display selected objects or can be manually expanded to explore the netlist. Use the Collapse toolbar button () to collapse the expanded netlist tree.
