Project Summary

The Project Summary displays project and design information, such as the target part and board, the state of synthesis and implementation, and timing summary and resource utilization. Some of the summary information is only estimated or is not available until after synthesis and implementation are run. As synthesis and implementation complete, DRC violations, timing values, utilization percentages, and power estimates are added to the summary. The Project Summary also provides hyperlinks to detailed information for the design, such as links to the Messages, Log, and Reports windows as well as the Project Settings dialog box.

In a project that has multiple synthesis and implementation runs defined, the data displayed in the Project Summary is dependent on the current design and the active run. The active synthesis and implementation runs are displayed in the Project Summary. You can change the active runs in the Design Runs window or by using the current_run Tcl command in the Tcl Console.

The Project Summary appears in the Vivado® IDE workspace. For more information on workspace windows refer to this link in the Vivado Design Suite User Guide: Using the Vivado IDE (UG893).
