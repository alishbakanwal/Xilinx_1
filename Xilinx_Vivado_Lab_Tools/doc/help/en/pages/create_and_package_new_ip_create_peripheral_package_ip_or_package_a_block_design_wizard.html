Create Peripheral, Package IP, or Package a Block Design

  • Package your current project: Package the currently opened project as an IP core to add to the Vivado® IP Catalog.
  • Package a block design from the current project: Package a block design from the current project as an IP core.
    Tip: This option is only available when your current project contains one or more block designs.
    • Select a block design: Select a block design from a list of the block designs in the current project.
  • Package a specified directory: Specify a directory of files organized for IP packaging. The Vivado tool uses inference rules when reading the directory contents to assist in packaging the IP.
  • Create a new AXI4 peripheral: Create a new AXI4 peripheral that includes HDL, drivers, a test application, and a BFM example template.
    Important: You must acquire a BFM license to use BFM models.
