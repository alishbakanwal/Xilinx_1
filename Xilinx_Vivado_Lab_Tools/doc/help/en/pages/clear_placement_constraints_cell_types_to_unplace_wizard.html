Cell Types to Unplace

This page of the Clear Placement wizard displays when you choose to clear the placement of cells in the design, or clear both ports and cells. The page displays a list of leaf cell types, and provides the count for each of the cell types, as well as a total cell count the Clear Placement command will affect.
Note: The list of cells is determined by the whether you had any cells selected before running the Clear Placement command, and what choice you made on the Unplace Cells page.
  • Leaf Cell Types to Unplace: Lists the different types of primitive cells found in the design, or currently selected, and lets you enable or disable clear placement.
  • Select All: Select all of the displayed leaf cell types.
  • Select None: Deselect all of the displayed leaf cell types.
  • Default: This restores the default settings for the page.
