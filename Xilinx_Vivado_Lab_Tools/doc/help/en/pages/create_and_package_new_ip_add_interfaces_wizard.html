Add Interfaces

The Add Interfaces page lets you add and configure slave and master interface pins, as well as an interrupt pin, to the AXI4 peripheral you define.

  • Enable Interrupt Support: Enable this checkbox to add an interrupt pin to the IP. The generated IP supports edge or level interrupts.
  • Add Interface: Above the middle display window, where the added interfaces are listed, click the green plus sign () to add a new interface pin to your peripheral IP.
  • Remove Interface: Click the red minus sign () to delete an interface pin from your peripheral IP.
  • Name: Specify a name for the interface that is currently selected in the middle display window.
  • Interface Type: Specify the interface type for the AXI interface. The supported interface types are AXI4, AXI4-Lite, and AXI4-Stream. See AXI4 Details for more information.
  • Interface Mode: Specify a Slave or Master interface mode for the selected interface.
  • Data Width (Bits): Defines the read and write data width of the AXI4 interface. The data width, expressed in bits, varies depending upon the AXI4 interface type selected. Select the appropriate data width from the available values.
  • Memory Size (Bytes): The memory size (MEMORY_SIZE), expressed in bytes, is only supported for AXI4-full slave interfaces. Select the appropriate memory size from the available values.
  • Number of Registers: The number of registers (NUM_REG) is only supported for AXI4-Lite slave interfaces.

When you finish adding and configuring interface pins to the AXI4 peripheral IP, you can click Next to open the Create Peripheral page.
