Add Sources Wizard

The Add Sources wizard guides you through the process of adding the following file types to the current project:

  • Add or create constraints
  • Add or create design sources
  • Add or create simulation sources
  • Add or create DSP sources
  • Add existing block designs
  • Add existing IP
