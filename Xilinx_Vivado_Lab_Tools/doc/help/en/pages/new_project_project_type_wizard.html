Project Type

  • RTL Project: Creates a new RTL project, which supports the use of hierarchical VHDL, Verilog, and SystemVerilog design sources as well as EDIF netlists for previously synthesized design modules. Click Next to Add Sources.
    • Do not specify sources at this time: Creates a new empty project and lets you specify design sources later. Clicking Next with this checkbox enabled will let you skip ahead to specify the Default Part.
  • Post-synthesis Project: Creates a post-synthesis netlist project using EDIF netlists generated by Vivado® synthesis or other synthesis tools. The post-synthesis project can be used to place and route and to generate a bitstream for a previously generated netlist design. Click Next to Add Netlist Sources.
    • Do not specify sources at this time: Creates a new empty project and lets you specify design sources later. Clicking Next with this checkbox enabled will let you skip ahead to specify the Default Part.
  • I/O Planning Project: Creates a new project to do early definition of ports and assignment of package pin locations. For more information refer to the Vivado Design Suite User Guide: I/O and Clock Planning (UG899). Click Next to Import Ports.
  • Imported Project: Creates a new project and imports RTL project data from the ISE® Design Suite Project Navigator, from XST, or from Synopsys Synplify to migrate an existing project into the Vivado Design Suite. The project source files and compilation order are imported, but implementation results and project settings are not. Click Next to Import Project.
  • Example Project: Create a new example project using one of the available design templates. Click Next to Select Project Template.
