Export I/O Ports

This dialog box lets you export the I/O port assignments in the current design to one or more output file formats.

  • CSV: Output a comma separated value (CSV) file for use in third-party design and analysis tools. See this link in Vivado® Design Suite User Guide: I/O and Clock Planning (UG899) for more information.
  • XDC: Write the constraints defining characteristics of the I/O ports, such as I/O standard, direction, drive, skew, and package pin assignment.
  • Verilog: Write a Verilog stub file for the top-level of the design based on I/O assignments.
  • VHDL: Write a VHDL stub file for the top-level of the design based on I/O assignments.
