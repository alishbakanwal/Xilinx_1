Fixed Placement

The choices displayed on this page depend on whether there are any fixed cells or ports in your selections in the Clear Placement wizard up to this point.
Tip: Fixed cells are associated with user-assigned placements, while unfixed cells are associated with tool assigned placements. See this link in Vivado® Design Suite User Guide: Using Constraints (UG903) for more information.
  • Fixed Cells : Is only displayed if any of the cells that will be unplaced have fixed locations.
    • Keep <#> fixed cells: Do not clear the placement of any of the fixed cells that the Clear Placement command may apply to, where <#> indicates the number of fixed cells.
    • Unplace all <#> placed cells: Clear the placement of all cells that the Clear Placement command may apply to, where <#> indicates the number of affected cells.
  • Fixed Ports : Is only displayed if any of the ports that will be unplaced have fixed locations.
    • Do not unplace any fixed ports: Do not clear the placement of any of fixed ports that the Clear Placement command may apply to.
    • Keep <#> fixed ports: Do not clear the placement of the selected fixed ports, where <#> is the number of selected fixed ports.
    • Unplace all <#> placed ports: Clear the placement of all ports that the Clear Placement command may apply to, where <#> indicates the number of affected ports.
