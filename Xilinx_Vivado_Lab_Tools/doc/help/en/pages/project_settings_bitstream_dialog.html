Project Settings - Bitstream

The Project Settings dialog box allows you to specify values for various settings used throughout the design flow. The settings are grouped according to the steps in the design flow, such as synthesis, simulation, and implementation.

The Bitstream settings let you define and adjust options that affect bitstream generation.

  • Configure additional bitstream settings: Opens the Edit Device Properties dialog box to let you configure additional properties of the bitstream, such as Enable Bitstream Compression and Enable Bitstream Encryption. See this link in the Vivado® Design Suite User Guide: Programming and Debugging (UG908) for more information on device configuration settings.
    Tip: This selection is only available when an implemented design is opened. If the implemented design is not open, there is a note that states Additional bitstream settings will be available once you open an implemented design.
  • Write Bitstream Options: Shows a table view of the options available when writing the bitstream. Select an option in the table to display a description of the option.
