Debug Window

The Debug window displays the debug cores and ports and the nets marked for debugging in the current design. To access this information, click one of the following tabs at the bottom of the window.
  • Debug Cores: Displays a hierarchical tree view of the ILA debug cores added to the current design, lists the probe ports assigned to the ILA cores, and lists the clocks and nets assigned to those probe ports.
  • Debug Nets: Lists the nets in the design that are marked for debug with the MARK_DEBUG property, shows nets that are assigned to an ILA core, and shows nets that are unassigned.

The toolbar has the following commands:

Search Opens a search field to enter a search pattern. The Debug window is searched for the specified value.
Collapse All Collapses the Clock Resources window cells.
Expand All Expands the Clock Resources window cells.
Set Up Debug Launch the Set Up Debug wizard to guide you through the netlist insertion process of selecting nets to mark for debugging and adding debug cores and ports to the design
Create Debug Core Add a new ILA debug core to the design.
Create Debug Port Define probe ports on the selected debug core.
Assign to Debug Port Assign nets to the probe ports to make signals accessible during debug.
