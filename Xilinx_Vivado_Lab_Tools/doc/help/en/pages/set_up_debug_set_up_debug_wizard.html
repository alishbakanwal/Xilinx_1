Set Up Debug

The Set Up Debug wizard guides you through the process of inserting debug cores into the post-synthesized netlist of the current design.

Tip: You can also add debug capabilities to your design by instantiating debug cores directly into your RTL design. This is called the HDL Instantiation Debug Probing flow. Refer to this link in the Vivado Design Suite User Guide: Programming and Debugging (UG908) for more information.
