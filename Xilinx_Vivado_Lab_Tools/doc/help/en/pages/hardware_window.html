Hardware

The Hardware window displays a tree view of the current hardware server, hardware target, and hardware device open in the Hardware Manager of the Vivado tool, and displays the status of the hardware. Selecting the device, target, or hardware server in the Hardware window displays the properties of the selected object in the Properties window.

From the right-click menu of the Hardware window, you can close or refresh the selected hardware server, or target. With the current device selected, the right-click menu lets you program the bitstream for your design into the device, and select from and manage the system monitor and debug cores implemented in the design.

The toolbar includes the following commands:

Search Opens a search field to enter a string search pattern.
Collapse all Collapses the current device hierarchy to hide the system monitor and debug cores.
Expand all Expands the current device hierarchy to show the system monitor and debug cores.
Scroll Enable automatic scrolling to the selected object in the Hardware window when the object is selected in another window.
Auto Connect Auto-connect to a hardware server and target running on the local host.
Tip: This command is only available when the current target is closed.
Run Trigger Arm the selected ILA core to detect trigger events that are defined by the core's basic or advanced trigger settings.
Run Immediate Arm the selected ILA core to trigger immediately regardless of the trigger settings.
Stop Trigger Stop the trigger on the selected ILA core.
